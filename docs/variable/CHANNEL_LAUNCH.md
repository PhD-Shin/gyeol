## 3. 세계관 고정 문장 (Canonical Statements)

세계관을 직접 설명하지 않되, 필요시 사용할 수 있는 문장들:

1. **이곳은 이야기를 설명하지 않는다. 결만 남긴다.**
   (This place does not explain stories. It only leaves 결.)

2. **우리는 옳은 길을 보여주지 않는다. 각자의 결을 지나갈 뿐이다.**
   (We do not show the right path. We only pass through each one's 결.)

3. **이유는 없다. 방향만 있다.**
   (There is no reason. Only direction.)

## 6. 썸네일 규칙

### 기본 원칙
- 한 장으로 세계관이 전달되어야 함
- 텍스트 최소화 (제목만 또는 없음)
- 스타일 가이드 색상 팔레트 준수

- 실루엣 또는 상징 요소
- 여백 확보
- 저채도 색상

- 클릭베이트 표현 ❌
- 과도한 텍스트 ❌
- 밝은 색상/네온 ❌
- 얼굴 클로즈업 ❌

## 10. 커뮤니티 운영 원칙

### 응답 스타일
- 설명하지 않는다
- 짧게 답한다
- 세계관 암시만 (직접 설명 ❌)

### 답변 예시
```

A: 👀

Q: 다음 영상 언제 나와요?
A: 흐름을 따라가고 있습니다.
```

### 금지 행동
- 세계관 직접 설명 ❌
- 캐릭터/상징 의미 해설 ❌
- 논쟁 참여 ❌
