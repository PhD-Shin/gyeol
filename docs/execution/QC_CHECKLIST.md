# QC_CHECKLIST — 결 품질 검사표

> **이 문서는 실행(execution) 문서입니다.**
> 모든 트랙/영상은 업로드 전 이 체크리스트를 통과해야 합니다.

---

## 사용 방법

1. 콘텐츠 완성 후 각 항목 확인
2. 모든 항목 통과 시 업로드 승인
3. 실패 항목 있으면 수정 후 재검사
4. 결과를 트랙 문서에 기록
