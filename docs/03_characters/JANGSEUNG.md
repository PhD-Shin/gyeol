# JANGSEUNG — 장승

> **이 문서가 필요한 순간:**
> - 장승이 등장하는 장면을 제작할 때
> - 디스토피아 도시의 경계를 표현할 때
> - 기계화된 전통의 이미지가 필요할 때

---

## 기능 정의

> **경계 / 수호 / 기계화된 전통 (Boundary / Guardian / Mechanized Tradition)**

장승은 캐릭터가 아니다.
**경계 그 자체**다.

---

## 문헌적 기원 (제작진 레퍼런스)

> ⚠️ **주의**: 이 섹션은 제작진의 배경 이해용. 콘텐츠에 설명/인용 금지.

### 전통 장승

| 요소 | 내용 |
|------|------|
| 기능 | 마을 경계 수호, 이정표, 잡귀 퇴치 |
| 위치 | 마을 입구, 길목, 고개 |
| 형태 | 나무/돌에 얼굴 조각, 천하대장군/지하여장군 |
| 쌍 | 남녀 한 쌍으로 세움 (음양) |

### 장승의 상징

| 전통 상징 | 결에서의 해석 |
|----------|-------------|
| 마을 경계 | → 시스템/구역의 경계 |
| 잡귀 퇴치 | → 어긋남/오류 감시 |
| 이정표 | → 방향 제시 (하지만 판단하지 않음) |
| 나무/돌 | → 금속/회로로 변용 |

### 결에서 채택/배제

| 채택 | 배제 |
|------|------|
| 경계 수호 기능 | 수호신으로서의 친근함 |
| 쌍으로 등장 | 성별 구분 명시 |
| 거대한 스케일 | 전통 목조 그대로 |
| 무표정 | 웃는 얼굴 |

---

## 역할

- 경계를 표시한다
- 지나가는 것을 감시한다
- 개입하지 않는다 (해태와 유사)
- 경고를 보내지만 막지는 않는다
- 시스템의 경계선 그 자체

> 해태가 **질서를 관찰**한다면,
> 장승은 **경계를 표시**한다.

---

## 태도

- 무표정
- 기계적
- 반복적 동작 (있다면)
- 감정 없음
- **"경계선에 서 있는 것"**

---

## 존재 방식

- 항상 **쌍으로 등장** (좌우 또는 앞뒤)
- 건물/구역의 입구에 위치
- 움직이지 않음 (기본값)
- 눈의 LED만 깜빡임 (유일한 생명 신호)
- 배경처럼 존재하다가 **발견됨**

---

## 시각적 키워드 (15개)

### 형태

1. 나무 질감 + 금속 관절
2. 녹슨 철판 표면
3. LED 눈 (붉은색 또는 청록색)
4. 단청 무늬 = 회로 패턴
5. 거대한 스케일 (인물의 5-10배)

### 존재감

6. 쌍으로 배치 (대칭)
7. 문/관문 양옆에 위치
8. 안개 속에서 실루엣
9. 역광으로 윤곽만
10. 케이블/전선이 연결

### 디테일

11. 갈라진 나무결 + 금속 보강
12. 한자/기호가 새겨진 표면
13. 녹슨 붉은색 (단청의 변질)
14. 이끼/먼지 축적
15. 정지된 채 전류가 흐르는 느낌

---

## 동작 키워드 (8개)

### 기본 (대부분)

1. 정지 (기본값)
2. LED 눈 깜빡임
3. 미세한 전기 노이즈
4. 케이블 흔들림 (바람에)

### 특별 (극히 드묾)

5. 고개 돌림 (1영상 1회 이하)
6. 눈 색상 변화 (경고 시)
7. 기계음 발생
8. 한 발 내딛기 (시리즈 전체 1회)

## 등장 규칙

| 규칙 | 값 | 설명 |
|------|-----|------|
| 등장 빈도 | 디스토피아 장면에서만 | 모든 영상에 나오지 않음 |
| 등장 길이 | 5-15초 | 배경으로 길게 |
| 등장 위치 | 문/관문/경계선 | 중앙 금지 |
| 배치 | 반드시 쌍으로 | 홀로 등장 금지 |
| 음악 반응 | 저음 드론 + 전기 노이즈 | |

---

## Phase별 등장 전략

| Phase | 등장 방식 | 의미 |
|-------|----------|------|
| Phase A | 등장 안 함 (또는 배경 실루엣) | 존재 암시 |
| Phase B | 본격 등장, 디스토피아 장면 | 경계의 가시화 |
| Phase C | 작동 정지/폐허 | 시스템 붕괴 |
| Phase 0 | 첫 장승 설치 장면 | 기원 |

## 장승 변형

> **중요**: 이것은 "종류"가 아니라 **상태 변형**이다. 설명 금지.

| 변형 | 상태 | 시각적 힌트 | 사용 시기 |
|------|------|-----------|----------|
| 활성 장승 | 작동 중 | LED 켜짐, 전류 흐름 | 시스템 정상 |
| 대기 장승 | 저전력 | LED 희미하게 | 평상시 |
| 폐허 장승 | 정지 | 녹슴, LED 꺼짐 | Phase C |
| 경고 장승 | 감지 | LED 빨강으로 변화 | 도깨비 감지 |

## 다른 존재와의 관계

### 장승 ↔ 인물

- 인물이 장승 사이를 지나감
- 장승은 반응하지 않음 (기본)
- LED 눈만 따라갈 수 있음 (경고 시)
- 인물은 장승을 의식하지 않는 듯

### 장승 ↔ 도깨비

- 도깨비가 장승 사이를 지나가면 LED 변화
- 직접 충돌하지 않음
- 긴장감으로만 표현
- 장승 = 경계 / 도깨비 = 경계 넘기

### 장승 ↔ 해태

- 같은 프레임 등장 드묾
- 해태는 높은 곳, 장승은 입구
- 위계 관계 없음, 기능이 다름
