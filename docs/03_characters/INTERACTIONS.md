# INTERACTIONS — 상호작용 규칙

> **이 문서가 필요한 순간:**
> - 여러 존재가 함께 등장하는 장면을 제작할 때
> - 존재들 간의 관계를 설정할 때
> - 금지된 상호작용을 확인할 때

---

## 핵심 원칙

> **결의 존재들은 직접 대화하지 않는다.**
> **관계는 시선, 위치, 리듬으로만 표현된다.**

---

## 상호작용 매트릭스

### 기본 존재

| | 도깨비 | 해태 | 탈 | 인물 |
|---|--------|------|-----|------|
| **도깨비** | - | 긴장 | 지나감 | 흔듦 |
| **해태** | 긴장 | - | 내려봄 | 지켜봄 |
| **탈** | 반응 | 무시 | 병치 | 경계 모호 |
| **인물** | 무인식 | 무인식 | 섞임 | 스쳐감 |

### 확장 존재

| | 장승 | 무당 | 불가사리 |
|---|------|------|----------|
| **도깨비** | 통과 | 감지당함 | 동반 혼란 |
| **해태** | 공존 | 경계 | 대치 |
| **탈** | 지나감 | 매개 | 도망 |
| **인물** | 무시/통과 | 의뢰 | 공포 |
| **장승** | - | 허용 | 저지 |
| **무당** | 호출 | - | 봉인 시도 |
| **불가사리** | 흡수 | 파괴 | - |

---

## 1. 도깨비 ↔ 인물

### 관계 정의
- 도깨비가 인물의 **결을 흔든다**
- 직접 접촉하지 않음
- 인물은 도깨비를 **인식하지 못하는 듯**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 스쳐지나감 | 도깨비가 인물 옆을 지남 |
| 배경 등장 | 인물 뒤에 도깨비 실루엣 |
| 리듬 깨짐 | 도깨비 등장 시 인물 동작 어긋남 |
| 시선 | 도깨비가 인물을 봄 (역은 드묾) |

### 금지 표현
- 직접 접촉 ❌
- 대화 ❌
- 인물이 도깨비와 싸움 ❌
- 도깨비가 인물을 구함/해침 (명확히) ❌

---

## 2. 해태 ↔ 인물

### 관계 정의
- 해태는 **지켜볼 뿐**
- 인물은 해태를 **의식하지 못함**
- 같은 프레임에 있어도 **상호작용 없음**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 배경 관찰 | 해태가 멀리서 지켜봄 |
| 높은 곳에서 | 해태가 위에서 내려봄 |
| 인물 무반응 | 인물이 해태를 발견해도 무반응 |
| 분리된 공간 | 같은 프레임, 다른 레이어 |

### 금지 표현
- 해태가 인물에게 개입 ❌
- 인물이 해태와 대화 ❌
- 해태가 인물을 구함 ❌
- 해태가 인물을 판단 (명시적으로) ❌

---

## 3. 탈 ↔ 인물

### 관계 정의
- 탈을 쓴 것인지 원래 얼굴인지 **모호**
- 탈과 인물의 **경계 흐림**
- **시선 교차 가능**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 경계 모호 | 인물이 탈이 되는 순간 |
| 시선 교차 | 인물과 탈이 서로 봄 |
| 군중 속 | 인물이 탈 쓴 군중 사이에 |
| 전환 | 인물 → 탈 전환 (역도 가능) |

### 금지 표현
- 탈을 벗는 장면 ❌ (핵심 금지)
- 인물이 탈과 대화 ❌
- 탈의 정체 밝힘 ❌

### 인물 ↔ 탈 전환 연출 가이드

전환은 경계의 핵심 표현:

| 전환 방향 | 의미 | 연출 |
|----------|------|------|
| 인물 → 탈 | 사회화, 페르소나 착용 | 회전/컷 전환 |
| 탈 → 인물 | 본성 드러남 (드묾) | 그림자/포커스 |
| 경계 모호 | 구분 불가 | 지속적 교차 편집 |

**핵심 원칙:**
- 전환 순간은 항상 모호
- "언제 바뀌었는지" 알 수 없어야 성공
- 전환 과정 (쓰기/벗기) 절대 금지
- 컷으로 건너뛰거나, 이미 바뀌어 있음

---

## 4. 도깨비 ↔ 해태

### 관계 정의
- 직접 충돌하지 않음
- 같은 프레임 등장 **드묾**
- **긴장감으로만 표현**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 긴장 | 같은 장면에서 긴장감 |
| 분리 등장 | 교차 편집으로 대비 |
| 원거리 | 멀리서 서로 인식 |
| 암시 | 한쪽이 다른 쪽을 암시 |

### 금지 표현
- 직접 싸움 ❌
- 대화 ❌
- 한쪽이 이김 ❌
- 협력 ❌

> **도깨비 = 개입 / 해태 = 관찰**
> 이 대립은 충돌이 아니라 **긴장**으로만 존재

---

## 5. 도깨비 ↔ 탈

### 관계 정의
- 도깨비가 탈 쓴 군중 사이를 **지나감**
- 탈 하나가 **반응할 수 있음**
- 군중의 리듬을 **깨뜨림**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 통과 | 도깨비가 군중 사이로 |
| 반응 | 한 탈만 고개 돌림 |
| 리듬 깨짐 | 군중 움직임 어긋남 |
| 혼란 | 탈들 사이 미세한 혼란 |

### 금지 표현
- 도깨비와 탈의 대화 ❌
- 모든 탈이 반응 ❌
- 도깨비가 탈을 벗김 ❌

---

## 6. 해태 ↔ 탈

### 관계 정의
- 해태가 탈 쓴 군중을 **내려다봄**
- 계급/위계의 시각화
- 탈은 해태를 **보지 않음**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 위계 | 해태가 위에서 군중 내려봄 |
| 무시 | 탈들은 해태 무시 |
| 질서 | 해태 아래 탈들이 정렬 |
| 분리 | 같은 프레임, 다른 층위 |

### 금지 표현
- 탈이 해태를 공격 ❌
- 해태가 탈과 대화 ❌
- 해태가 탈에게 명령 ❌

---

## 7. 탈 ↔ 탈

### 관계 정의
- 병치 (Juxtaposition)
- 개별성 없는 군중
- **반복되는 표면**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 병치 | 여러 탈 나란히 |
| 군중 | 끝없이 반복 |
| 한 명 다름 | 한 탈만 다른 방향 |
| 동시 움직임 | 전부 같이 고개 돌림 |

---

## 8. 장승 ↔ 인물

### 관계 정의
- 장승은 **문 (門)** — 인물이 지나가는 경계
- 장승은 인물을 **심판하지 않음**
- 인물은 장승을 **배경처럼 통과**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 통과 | 인물이 장승 사이를 지나감 |
| 무시 | 인물이 장승을 보지 않음 |
| 문 역할 | 장승이 공간을 나눔 |
| LED 반응 | 인물 통과 시 눈 깜빡임 (미묘) |

### 금지 표현
- 장승이 인물을 막음 ❌
- 인물이 장승과 대화 ❌
- 장승이 인물을 판단 (명시적) ❌
- 장승 하나만 등장 ❌ (항상 쌍으로)

> **핵심**: 장승은 문이지, 문지기가 아니다

---

## 9. 장승 ↔ 도깨비

### 관계 정의
- 도깨비는 장승 사이를 **마음대로 통과**
- 장승은 도깨비를 **막지 못함**
- 도깨비가 지나갈 때 장승의 **LED가 어지러움**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 무력한 문 | 도깨비가 유유히 통과 |
| 글리치 반응 | 장승 LED가 깜빡/왜곡 |
| 경계 무시 | 도깨비에게 경계는 없다 |
| 사이로 | 장승 사이가 아닌 옆/위로도 |

### 금지 표현
- 장승이 도깨비를 막음 ❌
- 장승과 도깨비의 대립 ❌
- 장승이 도깨비에게 반응 (의식적) ❌

---

## 10. 장승 ↔ 해태

### 관계 정의
- 둘 다 **수호자** — 같은 편
- 같은 프레임 등장 **가능** (드묾)
- 해태가 장승 위에 앉음 = **위계의 시각화**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 공존 | 장승 근처에 해태 |
| 위계 | 해태가 장승 머리/위에 |
| 수호 | 함께 어떤 공간을 지킴 |
| 고요 | 둘 다 움직이지 않음 |

### 금지 표현
- 해태가 장승에게 명령 ❌
- 장승이 해태에게 복종 (명시적) ❌
- 둘이 대화 ❌

---

## 11. 장승 ↔ 불가사리

### 관계 정의
- 장승이 **유일하게 저항하는 존재**
- 불가사리가 장승을 **먹으려 함**
- 장승은 **경계를 지키려 하지만 실패**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 저항 | 장승이 버티는 자세 |
| 부식 | 불가사리가 장승을 침식 |
| 마지막 문 | 장승이 최후의 경계선 |
| 붕괴 | 장승이 쓰러지며 경계 무너짐 |

### 금지 표현
- 장승이 불가사리를 이김 ❌
- 장승이 불가사리를 막음 (성공) ❌

> **핵심**: 장승의 유일한 패배 장면 = 불가사리

---

## 12. 무당 ↔ 인물

### 관계 정의
- 무당은 **경계 사이를 오가는 자**
- 인물은 무당에게 **의뢰함**
- 결에서 유일하게 **의식적 접촉 가능**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 의뢰 | 인물이 무당을 찾아옴 |
| 굿판 | 인물이 굿판에 참여 |
| 거리 | 무당 작업 중 인물은 거리 둠 |
| 전달 | 무당이 인물에게 무언가 건넴 (말 제외) |

### 금지 표현
- 무당과 인물의 일상 대화 ❌
- 무당이 인물에게 설명 ❌
- 인물이 무당의 능력 의심 ❌
- 무당이 인물 문제를 완전히 해결 ❌

> **핵심**: 무당은 연결만 한다. 해결하지 않는다.

---

## 13. 무당 ↔ 도깨비

### 관계 정의
- 무당은 도깨비를 **감지할 수 있는 유일한 인간**
- 무당이 도깨비를 **호출하거나 달램**
- 도깨비는 무당을 **피하거나 따름** (상황에 따라)

### 허용 표현
| 표현 | 설명 |
|------|------|
| 감지 | 무당이 도깨비 기척을 느낌 |
| 호출 | 굿판에서 도깨비 등장 |
| 시선 | 무당만 도깨비를 봄 |
| 달램 | 무당이 도깨비에게 제물 |

### 금지 표현
- 무당과 도깨비의 대화 ❌
- 무당이 도깨비를 완전히 제어 ❌
- 도깨비가 무당에게 복종 ❌

---

## 14. 무당 ↔ 해태

### 관계 정의
- 긴장 관계 — 무당은 **경계를 넘고**, 해태는 **경계를 지킴**
- 서로 **인식하지만 개입 안 함**
- 같은 프레임 등장 = **대립 암시**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 긴장 | 같은 프레임에서 거리 유지 |
| 시선 교차 | 서로를 인식 |
| 대비 | 무당=움직임, 해태=정지 |
| 경계 | 해태가 무당을 지켜봄 (경계) |

### 금지 표현
- 해태가 무당을 막음 ❌
- 무당이 해태에게 도전 ❌
- 둘의 직접 충돌 ❌

---

## 15. 무당 ↔ 불가사리

### 관계 정의
- 무당은 불가사리를 **봉인하려 시도**
- 불가사리는 **봉인을 뚫음** (결국)
- 무당의 **유일한 실패**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 봉인 시도 | 무당이 굿을 통해 막으려 함 |
| 저항 | 불가사리가 봉인에 저항 |
| 실패 | 봉인이 깨지는 순간 |
| 공포 | 무당조차 두려워함 |

### 금지 표현
- 무당이 불가사리를 완전히 봉인 ❌
- 무당이 불가사리를 제거 ❌

> **핵심**: 불가사리 앞에서 무당의 한계가 드러난다

---

## 16. 불가사리 ↔ 모든 존재

### 관계 정의
- 불가사리는 **2-3회만 등장**
- 모든 것을 **삼키는** 존재
- **대응 불가** — 피하거나 당하거나

### 불가사리 등장 시 반응

| 존재 | 반응 | 표현 |
|------|------|------|
| 인물 | 공포/도망 | 뒷걸음, 넘어짐 |
| 도깨비 | 혼란/동반 | 불가사리와 함께 혼돈 |
| 해태 | 대치/후퇴 | 유일하게 맞섬, 그러나 막지 않음 |
| 탈 | 도망/흩어짐 | 군중 해체 |
| 장승 | 저항/붕괴 | 막으려다 먹힘 |
| 무당 | 봉인 시도/실패 | 굿 → 깨짐 |

### 불가사리 금지 규칙
- 누구도 불가사리를 이기지 않음 ❌
- 불가사리가 대화함 ❌
- 불가사리가 자주 등장 ❌ (2-3회 제한)
- 불가사리가 개별 인물을 표적으로 삼음 ❌

> **핵심**: 불가사리는 재앙이다. 개인이 아니라 세계에 닥친다.

---

## 동시 등장 가이드

### 추천 조합

| 조합 | 효과 |
|------|------|
| 도깨비 + 인물 | 개입, 흔들림 |
| 해태 + 인물 | 관찰, 긴장 |
| 탈 + 인물 | 경계 모호, 사회 |
| 탈 + 해태 | 위계, 질서 |
| 도깨비 + 탈 | 혼란, 리듬 깨짐 |
| 장승 + 인물 | 통과, 경계 |
| 장승 + 해태 | 수호, 위계 |
| 무당 + 도깨비 | 호출, 굿판 |
| 무당 + 인물 | 의뢰, 연결 |

### 피해야 할 조합

| 조합 | 이유 |
|------|------|
| 도깨비 + 해태 (같은 프레임) | 긴장이 충돌로 변함 |
| 4존재 전부 | 너무 복잡, 의미 과잉 |
| 불가사리 + 일상 장면 | 불가사리는 클라이맥스만 |
| 무당 + 해태 (직접 대면) | 대립이 명시화됨 |
| 장승 단독 | 항상 쌍으로 |

## 3존재 동시 등장 구도

### 도깨비 + 탈 + 인물

```
[구도]
      탈(배경, 군중)
         ↑
도깨비 ←  인물
(프레임   (중앙)
 가장자리)
```

| 위치 | 존재 | 역할 |
|------|------|------|
| 전경 가장자리 | 도깨비 | 개입, 스쳐지나감 |
| 중경 중앙 | 인물 | 흔들리는 대상 |
| 후경 | 탈 군중 | 관찰/반응 |

### 해태 + 탈 + 인물

```
[구도]
      해태(높은 곳, 후경)
           ↓
      탈(중경, 정렬)
           ↓
      인물(전경)
```

| 위치 | 존재 | 역할 |
|------|------|------|
| 높은 곳/후경 | 해태 | 내려다봄 |
| 중경 | 탈 군중 | 위계 아래 |
| 전경 | 인물 | 시스템 내 존재 |

### 장승 + 인물 + 도깨비

```
[구도]
 장승         장승
  ||           ||
  ||    인물   ||  ← 도깨비(가장자리)
  ||    ●     ||
```

| 위치 | 존재 | 역할 |
|------|------|------|
| 양쪽 | 장승 (쌍) | 문, 경계 |
| 중앙 | 인물 | 통과하는 자 |
| 가장자리 | 도깨비 | 경계 무시하며 관찰 |

### 무당 + 도깨비 + 인물 (굿판)

```
[구도]
      도깨비들 (배경, 희미)
           ↓
      무당 (중앙, 작두 위)
           ↓
      인물 (전경, 거리 두고 관찰)
```

| 위치 | 존재 | 역할 |
|------|------|------|
| 후경 | 도깨비들 | 호출되어 등장 |
| 중앙 | 무당 | 의례 진행 |
| 전경 | 인물 | 의뢰인, 거리 유지 |

### 불가사리 + 장승 + 해태 (클라이맥스)

```
[구도]
          해태 (높은 곳, 관망)
              ↑
장승 ←← 불가사리 →→ 장승
(저항)     (중앙)     (붕괴)
```

| 위치 | 존재 | 역할 |
|------|------|------|
| 상단 | 해태 | 막지 않음, 관망 |
| 양쪽 | 장승 | 저항하다 무너짐 |
| 중앙 | 불가사리 | 삼킴, 확장 |

> **이 구도는 전체 시리즈에서 1회만 사용**

## 등장 시간 순서

### 원칙
- **해태는 항상 이미 있다** (발견되기만 함)
- 도깨비는 **마지막에 등장** (개입)
- 탈은 **인물과 함께** 또는 **인물보다 먼저**
- 인물은 **첫 번째 또는 두 번째**

### 권장 순서

| 조합 | 등장 순서 |
|------|----------|
| 도깨비 + 인물 | 인물 → 도깨비 |
| 해태 + 인물 | 해태(발견) → 인물 또는 동시 |
| 탈 + 인물 | 탈 → 인물 또는 동시 |
| 도깨비 + 탈 | 탈 → 도깨비 |
| 해태 + 탈 | 해태(발견) → 탈 또는 동시 |

### 퇴장 순서

| 조합 | 퇴장 순서 |
|------|----------|
| 도깨비 + 인물 | 도깨비 먼저 (개입 후 사라짐) |
| 해태 + 인물 | 인물 먼저 (해태는 남음) |
| 탈 + 인물 | 동시 또는 탈 먼저 (페이드) |

> **핵심**: 도깨비는 "나타나고 사라진다", 해태는 "이미 있고 남는다"

## Phase별 상호작용 전략

| Phase | 주 조합 | 상호작용 강도 | 특징 |
|-------|--------|-------------|------|
| Phase A | 인물 단독, 탈 배경 | 약함 | 존재 암시, 거리감 |
| Phase B | 도깨비+인물, 탈 군중 | 중간 | 개입 시작, 긴장 |
| Phase C | 탈+인물, 해태 관찰 | 강함 | 경계 흐림, 판단 |
| Phase 0 | 존재들 각각 분리 | 최소 | 기원, 분리된 상태 |

### Phase별 동시 등장 허용
| Phase | 최대 존재 수 | 권장 조합 |
|-------|-----------|----------|
| Phase A | 2 | 인물+탈, 인물+도깨비 |
| Phase B | 3 | 도깨비+탈+인물 |
| Phase C | 3 | 해태+탈+인물 |
| Phase 0 | 1-2 | 단독 또는 인물+1 |

## 상호작용 실패 예시

### 잘못된 상호작용

| 실패 | 문제 | 왜 실패인가 |
|------|------|-----------|
| 도깨비가 인물을 밀침 | 직접 접촉 | 접촉 금지 위반 |
| 해태가 인물에게 끄덕임 | 의사소통 | 개입 금지 위반 |
| 탈이 말함 | 대화 | 결은 설명 안 함 |
| 인물이 도깨비와 눈 맞춤 | 인식 | 무인식 규칙 위반 |
| 4존재 전부 등장 | 과밀 | 복잡, 의미 과잉 |

### 수정 방법

| 실패 | 수정 |
|------|------|
| 직접 접촉 | 스쳐지나감으로 대체 |
| 의사소통 | 관찰로만 축소 |
| 대화 | 장면 삭제 |
| 인식 | 인물 시선 다른 곳으로 |
| 과밀 | 존재 수 줄이기 |

## 동시 등장 편집 기법

### 같은 프레임 내

| 기법 | 설명 | 사용 |
|------|------|------|
| 레이어 분리 | 포커스로 층위 구분 | 해태+인물 |
| 거리 분리 | 전경/중경/후경 분리 | 3존재 동시 |
| 조명 분리 | 빛 받는 영역 다름 | 긴장 표현 |
| 색온도 분리 | 존재별 색온도 차이 | 대립 표현 |

### 교차 편집

| 기법 | 설명 | 사용 |
|------|------|------|
| 숏-리버스 숏 | A → B → A | 대면 (인물 2명) |
| 평행 편집 | A1 → B1 → A2 → B2 | 긴장 (도깨비-해태) |
| 점프 컷 | 같은 대상, 시간 점프 | 탈 군중 |
| 매치 컷 | 형태 연결 | 인물→탈 전환 |

### 전환 타이밍

| 조합 | 전환 | 속도 |
|------|------|------|
| 도깨비+인물 | 컷 | 빠름 (0.5-1초) |
| 해태+인물 | 디졸브/롱테이크 | 느림 (3-5초) |
| 탈+인물 | 컷/매치컷 | 중간 (1-2초) |
| 도깨비+탈 | 컷 | 빠름 (0.5초) |

### 편집 금지

- 빠른 휩 팬 ❌
- 스플릿 스크린 ❌
- 화려한 전환 효과 ❌
- 슬로우 모션 + 빠른 모션 혼합 ❌
