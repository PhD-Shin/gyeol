# INTERACTIONS — 상호작용 규칙

> **이 문서가 필요한 순간:**
> - 여러 존재가 함께 등장하는 장면을 제작할 때
> - 존재들 간의 관계를 설정할 때
> - 금지된 상호작용을 확인할 때

---

## 핵심 원칙

> **결의 존재들은 직접 대화하지 않는다.**
> **관계는 시선, 위치, 리듬으로만 표현된다.**

---

## 상호작용 매트릭스

| | 도깨비 | 해태 | 탈 | 인물 |
|---|--------|------|-----|------|
| **도깨비** | - | 긴장 | 지나감 | 흔듦 |
| **해태** | 긴장 | - | 내려봄 | 지켜봄 |
| **탈** | 반응 | 무시 | 병치 | 경계 모호 |
| **인물** | 무인식 | 무인식 | 섞임 | 스쳐감 |

---

## 1. 도깨비 ↔ 인물

### 관계 정의
- 도깨비가 인물의 **결을 흔든다**
- 직접 접촉하지 않음
- 인물은 도깨비를 **인식하지 못하는 듯**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 스쳐지나감 | 도깨비가 인물 옆을 지남 |
| 배경 등장 | 인물 뒤에 도깨비 실루엣 |
| 리듬 깨짐 | 도깨비 등장 시 인물 동작 어긋남 |
| 시선 | 도깨비가 인물을 봄 (역은 드묾) |

### 금지 표현
- 직접 접촉 ❌
- 대화 ❌
- 인물이 도깨비와 싸움 ❌
- 도깨비가 인물을 구함/해침 (명확히) ❌

---

## 2. 해태 ↔ 인물

### 관계 정의
- 해태는 **지켜볼 뿐**
- 인물은 해태를 **의식하지 못함**
- 같은 프레임에 있어도 **상호작용 없음**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 배경 관찰 | 해태가 멀리서 지켜봄 |
| 높은 곳에서 | 해태가 위에서 내려봄 |
| 인물 무반응 | 인물이 해태를 발견해도 무반응 |
| 분리된 공간 | 같은 프레임, 다른 레이어 |

### 금지 표현
- 해태가 인물에게 개입 ❌
- 인물이 해태와 대화 ❌
- 해태가 인물을 구함 ❌
- 해태가 인물을 판단 (명시적으로) ❌

---

## 3. 탈 ↔ 인물

### 관계 정의
- 탈을 쓴 것인지 원래 얼굴인지 **모호**
- 탈과 인물의 **경계 흐림**
- **시선 교차 가능**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 경계 모호 | 인물이 탈이 되는 순간 |
| 시선 교차 | 인물과 탈이 서로 봄 |
| 군중 속 | 인물이 탈 쓴 군중 사이에 |
| 전환 | 인물 → 탈 전환 (역도 가능) |

### 금지 표현
- 탈을 벗는 장면 ❌ (핵심 금지)
- 인물이 탈과 대화 ❌
- 탈의 정체 밝힘 ❌

---

## 4. 도깨비 ↔ 해태

### 관계 정의
- 직접 충돌하지 않음
- 같은 프레임 등장 **드묾**
- **긴장감으로만 표현**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 긴장 | 같은 장면에서 긴장감 |
| 분리 등장 | 교차 편집으로 대비 |
| 원거리 | 멀리서 서로 인식 |
| 암시 | 한쪽이 다른 쪽을 암시 |

### 금지 표현
- 직접 싸움 ❌
- 대화 ❌
- 한쪽이 이김 ❌
- 협력 ❌

> **도깨비 = 개입 / 해태 = 관찰**
> 이 대립은 충돌이 아니라 **긴장**으로만 존재

---

## 5. 도깨비 ↔ 탈

### 관계 정의
- 도깨비가 탈 쓴 군중 사이를 **지나감**
- 탈 하나가 **반응할 수 있음**
- 군중의 리듬을 **깨뜨림**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 통과 | 도깨비가 군중 사이로 |
| 반응 | 한 탈만 고개 돌림 |
| 리듬 깨짐 | 군중 움직임 어긋남 |
| 혼란 | 탈들 사이 미세한 혼란 |

### 금지 표현
- 도깨비와 탈의 대화 ❌
- 모든 탈이 반응 ❌
- 도깨비가 탈을 벗김 ❌

---

## 6. 해태 ↔ 탈

### 관계 정의
- 해태가 탈 쓴 군중을 **내려다봄**
- 계급/위계의 시각화
- 탈은 해태를 **보지 않음**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 위계 | 해태가 위에서 군중 내려봄 |
| 무시 | 탈들은 해태 무시 |
| 질서 | 해태 아래 탈들이 정렬 |
| 분리 | 같은 프레임, 다른 층위 |

### 금지 표현
- 탈이 해태를 공격 ❌
- 해태가 탈과 대화 ❌
- 해태가 탈에게 명령 ❌

---

## 7. 탈 ↔ 탈

### 관계 정의
- 병치 (Juxtaposition)
- 개별성 없는 군중
- **반복되는 표면**

### 허용 표현
| 표현 | 설명 |
|------|------|
| 병치 | 여러 탈 나란히 |
| 군중 | 끝없이 반복 |
| 한 명 다름 | 한 탈만 다른 방향 |
| 동시 움직임 | 전부 같이 고개 돌림 |

---

## 동시 등장 가이드

### 추천 조합

| 조합 | 효과 |
|------|------|
| 도깨비 + 인물 | 개입, 흔들림 |
| 해태 + 인물 | 관찰, 긴장 |
| 탈 + 인물 | 경계 모호, 사회 |
| 탈 + 해태 | 위계, 질서 |
| 도깨비 + 탈 | 혼란, 리듬 깨짐 |

### 피해야 할 조합

| 조합 | 이유 |
|------|------|
| 도깨비 + 해태 (같은 프레임) | 긴장이 충돌로 변함 |
| 4존재 전부 | 너무 복잡, 의미 과잉 |

## 체크리스트

여러 존재 등장 장면:

- [ ] 직접 대화가 없는가?
- [ ] 직접 접촉이 없는가? (허용된 경우 제외)
- [ ] 상호작용이 시선/위치/리듬으로만 표현되는가?
- [ ] 금지 조합이 아닌가?
- [ ] 각 존재의 등장 규칙을 지키는가?
