# SHAMAN — 무당

> **이 문서가 필요한 순간:**
> - 무당이 등장하는 장면을 제작할 때
> - 굿판/의례 장면을 구성할 때
> - 한의 폭발/흥의 정점을 표현할 때

---

## 기능 정의

> **매개자 / 의례 / 경계 넘기 (Mediator / Ritual / Boundary Crossing)**

무당은 캐릭터가 아니다.
**경계를 넘는 통로**다.

---

## 문헌적 기원 (제작진 레퍼런스)

> ⚠️ **주의**: 이 섹션은 제작진의 배경 이해용. 콘텐츠에 설명/인용 금지.

### 무당의 기능

| 요소 | 내용 |
|------|------|
| 역할 | 신과 인간 사이 매개, 굿을 통한 소통 |
| 입무 | 신병(神病)을 거쳐 무당이 됨 |
| 의례 | 굿판 — 춤, 노래, 작두타기 |
| 복장 | 무복, 다양한 색상의 띠, 방울/부채/칼 |

### 작두타기의 의미

| 요소 | 상징 |
|------|------|
| 맨발로 작두 위에 | 경계 위에 서 있음 |
| 피가 나지 않음 | 신이 내린 상태 |
| 트랜스 상태 | 인간과 신의 경계 모호 |
| 군중의 환호 | 한의 집단 해소 |

### 결에서 채택/배제

| 채택 | 배제 |
|------|------|
| 매개자 기능 | 종교적 의미 설명 |
| 트랜스/경계 모호 | 점술/예언 |
| 작두타기 이미지 | 굿거리 전체 재현 |
| 무복의 형태 | 민속 의상 그대로 |

---

## 역할

- 경계를 넘는다
- 한을 흥으로 전환한다
- 인간과 시스템 사이를 매개한다
- 트랜스 상태에서 무언가를 전달한다
- 작두 위에서 시스템을 과부하시킨다

> 도깨비가 **개입**하고 해태가 **관찰**한다면,
> 무당은 **매개**한다.

---

## 태도

- 트랜스 상태
- 인간인지 아닌지 모호
- 광기 어린 집중
- 감정 과잉처럼 보이지만 **표현이 아닌 전달**
- **"몸이 통로가 된 것"**

---

## 존재 방식

- 인물이면서 **인물을 넘어선 존재**
- 사이보그 요소 (네온 띠, 회로, 기계 부품)
- 얼굴이 보일 수도, 탈일 수도 있음
- 항상 **의례 중**에만 등장
- 일상 장면에서는 등장 금지

---

## 시각적 키워드 (15개)

### 형태

1. 무복 (흰색 베이스 + 색 띠)
2. 네온 띠 (전통 띠의 변용)
3. 회로 문양이 새겨진 부채
4. 금속 작두 (LED/회로 포함)
5. 맨발 (또는 발광 발자국)

### 존재감

6. 트랜스 표정 (눈 감거나 흰자위)
7. 머리카락 날림
8. 회전 중 잔상
9. 군중(탈) 가운데 위치
10. 작두 위 균형

### 사이보그 요소

11. 팔목/목에 회로 문양
12. 눈에서 빛 (선택적)
13. 방울 대신 금속 파티클
14. 무복에 반사 소재
15. 땀/액체가 빛나는 효과

---

## 동작 키워드 (12개)

### 의례 동작

1. 작두 타기 (핵심)
2. 회전 (빠르게)
3. 팔 벌림 (날개처럼)
4. 부채 휘두름

### 트랜스 동작

5. 떨림 (전신)
6. 고개 젖힘
7. 눈 감고 움직임
8. 갑작스런 정지

### 전환 동작

9. 무릎 꿇음
10. 쓰러짐 (트랜스 후)
11. 일어섬 (천천히)
12. 군중 속으로 사라짐

## 무당의 상태

> **중요**: 이것은 "종류"가 아니라 **의례 단계**다.

| 상태 | 단계 | 시각적 표현 | 음악 |
|------|------|-----------|------|
| 준비 | 시작 전 | 무릎 꿇음, 눈 감음 | 저음 드론 |
| 상승 | 트랜스 진입 | 떨림, 회전 시작 | 리듬 빌드업 |
| 정점 | 작두 위 | 작두타기, 눈 흰자위 | 타악기 폭발 |
| 하강 | 트랜스 종료 | 천천히 내려옴, 쓰러짐 | 페이드 아웃 |

## 작두 장면 연출 규칙

### 작두의 시각

| 요소 | 설명 |
|------|------|
| 재질 | 금속 + 회로/LED |
| 크기 | 과장됨 (실제보다 큼) |
| 날 | 빛남 (위험함 강조) |
| 받침 | 금속 프레임 |

### 작두타기 시퀀스

```
1. 작두 클로즈업 (날 강조)
2. 무당 맨발 클로즈업
3. 발이 작두 위에 올라감 (과정)
4. 롱샷 — 작두 위 무당
5. 군중(탈)의 반응 컷
6. 무당 얼굴 (트랜스)
7. 작두에서 내려옴 (또는 컷)
```

### 금지 연출

- 피가 나는 장면 ❌
- 실패/떨어지는 장면 ❌
- 작두 위에서 오래 서 있기 ❌ (3-5초 내)
- 안전하다는 암시 ❌

---

## 금지 연출

| 금지 | 이유 |
|------|------|
| 점술/예언 장면 ❌ | 종교적 설명 금지 |
| 신 내림 설명 ❌ | 민속 소개 금지 |
| 굿거리 전체 재현 ❌ | 축약/상징화 필요 |
| 일상 장면에서 등장 ❌ | 의례 중에만 |
| 무당 개인사 ❌ | 캐릭터화 금지 |
| 구원/치유 장면 ❌ | 결은 판단 안 함 |

## 다른 존재와의 관계

### 무당 ↔ 탈 (군중)

- 탈 쓴 군중이 무당을 둘러싸고 있음
- 군중은 반응하지만 표정 없음
- 무당의 움직임에 동기화될 수 있음
- 한의 집단 해소를 상징

### 무당 ↔ 도깨비

- 도깨비가 의례 중에 스침
- 무당이 도깨비를 **인식**할 수 있음 (유일)
- 긴장 또는 연결
- 개입과 매개의 만남

### 무당 ↔ 인물

- 인물이 무당이 될 수 있음 (전환)
- 경계 모호: 인물인지 무당인지
- 탈을 쓰면 더 모호
