# BULGASARI — 불가사리

> **이 문서가 필요한 순간:**
> - 불가사리가 등장하는 장면을 제작할 때
> - 시스템 붕괴/해체 장면을 구성할 때
> - 통제 불가능한 힘을 표현할 때

---

## 기능 정의

> **탐식 / 붕괴 / 통제 불가 (Devouring / Collapse / Uncontrollable)**

불가사리는 캐릭터가 아니다.
**멈출 수 없는 힘**이다.

---

## 문헌적 기원 (제작진 레퍼런스)

> ⚠️ **주의**: 이 섹션은 제작진의 배경 이해용. 콘텐츠에 설명/인용 금지.

### 불가사리 설화

| 요소 | 내용 |
|------|------|
| 기원 | 밥풀/꿈으로 만들어진 작은 존재 |
| 특성 | 쇠(철)를 먹고 자람, 커질수록 더 먹음 |
| 의도 | 처음엔 도움 (쇠를 먹어 적을 물리침) |
| 결말 | 통제 불가 → 결국 스스로를 삼킴 |

### 불가사리의 역설

> **"도움을 주려고 만들었지만, 멈출 수 없게 되었다."**

| 단계 | 상황 |
|------|------|
| 창조 | 작고 귀여움, 도움이 됨 |
| 성장 | 커지며 유용해짐 |
| 폭주 | 멈추지 않음, 모든 쇠를 먹음 |
| 자멸 | 먹을 게 없어지면 자신을 삼킴 |

### 결에서 채택/배제

| 채택 | 배제 |
|------|------|
| 쇠를 먹는 기능 | 귀여운/작은 형태 |
| 통제 불가능 | 선한 의도/영웅 |
| 기하급수적 성장 | 자멸 장면 |
| 기계/산업 파괴 | 민속 설화 재현 |

---

## 역할

- 쇠(기계/시스템)를 먹는다
- 커질수록 더 많이 먹는다
- 멈추지 않는다
- 판단하지 않는다 — 배고픔만 있다
- 시스템의 붕괴를 가져온다

> 도깨비가 **개입**하고 장승이 **경계**한다면,
> 불가사리는 **삼킨다**.

---

## 태도

- 배고픔
- 판단 없음
- 멈추지 않음
- 기계적 탐식
- **"그냥 먹는 것"**

---

## 존재 방식

- 처음엔 **보이지 않음** (존재 암시만)
- 사라진 금속/기계 부품으로 존재 추론
- 등장하면 **거대함**
- 항상 **먹는 중** 또는 **직후**
- 만족하지 않음

---

## 시각적 키워드 (15개)

### 형태

1. 금속 파편으로 덮인 몸
2. 곰/소/용의 혼합체
3. 빛나는 눈 (탐욕)
4. 입 = 용광로/블랙홀 느낌
5. 불확정적 크기 (계속 변함)

### 존재감

6. 금속 파편이 빨려 들어감
7. 주변 기계가 해체됨
8. 어둠 속에서 눈만 빛남
9. 거대한 실루엣
10. 연기/증기와 함께

### 디테일

11. 몸에 녹슨 금속 조각들
12. 전선/케이블이 몸에 감김
13. 먹다 만 기계 부품
14. 이빨 = 톱니바퀴
15. 타액 = 녹슨 액체

---

## 동작 키워드 (10개)

### 탐식 동작

1. 씹기 (금속 으깨는 소리)
2. 삼키기
3. 빨아들이기 (자기장처럼)
4. 핥기 (금속 표면)

### 이동 동작

5. 느린 전진 (거대함)
6. 돌진 (먹이 발견 시)
7. 몸 부풀기 (먹은 후)
8. 웅크림 (다음 사냥 전)

### 등장/퇴장

9. 어둠에서 나타남
10. 안개/연기 속으로 사라짐

## 등장 규칙

| 규칙 | 값 | 설명 |
|------|-----|------|
| 등장 빈도 | 시리즈 전체 2-3회 | 극히 희소 |

| 음악 반응 | 저음 럼블, 금속음 | |

## Phase별 등장 전략

| Phase | 등장 방식 | 의미 |
|-------|----------|------|
| Phase A | 등장 안 함 | - |
| Phase B | 흔적만 (사라진 금속) | 존재 암시 |
| Phase C | 본격 등장 | 시스템 붕괴 |
| Phase 0 | 등장 안 함 (또는 최초 탄생) | 기원 가능 |

## 불가사리 크기 단계

> **중요**: 크기는 Phase/장면에 따라 달라진다.

| 단계 | 크기 | 시각적 표현 | 사용 |
|------|------|-----------|------|
| 암시 | 보이지 않음 | 사라진 금속만 | Phase B |
| 소형 | 인물 크기 | 실루엣, 눈만 | 첫 등장 |
| 중형 | 건물 크기 | 부분만 보임 | Phase C 초반 |
| 대형 | 도시 스케일 | 실루엣만 | Phase C 클라이맥스 |

### 크기 표현 규칙

- 전체를 보여주지 않음 (부분만)
- 커질수록 더 모호하게
- 대형은 **형태가 아니라 현상**처럼

## 다른 존재와의 관계

### 불가사리 ↔ 장승

- 장승(금속)을 먹을 수 있음
- 경계 시스템의 붕괴
- 같은 프레임에서 긴장감
- 장승의 LED가 경고로 변함

### 불가사리 ↔ 도깨비

- 도깨비는 불가사리를 피함 (또는 무시)
- 불가사리는 도깨비에 관심 없음 (쇠가 아님)
- 둘 다 시스템을 해체하지만 방식이 다름
- 개입 vs 탐식

### 불가사리 ↔ 해태

- 해태는 관찰할 뿐
- 판단하지 않음 (결의 규칙)
- 불가사리도 해태에 관심 없음 (돌/청동)
- 두 존재가 만나면 **긴장만**

### 불가사리 ↔ 인물

- 인물은 불가사리를 피함
- 직접 상호작용 없음
- 인물의 반응으로만 불가사리 존재 암시 가능
