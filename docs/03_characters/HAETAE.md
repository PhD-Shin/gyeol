# HAETAE — 해태

> **이 문서가 필요한 순간:**
> - 해태가 등장하는 장면을 제작할 때
> - 등장 규칙/금지 연출을 확인할 때
> - 해태의 역할을 이해해야 할 때

---

## 기능 정의

> **경계 / 질서 / 관찰자 (Boundary / Order / Observer)**

해태는 캐릭터가 아니다.
**시스템적 질서의 표상**이다.

---

## 문헌적 기원 (제작진 레퍼런스)

> ⚠️ **주의**: 이 섹션은 제작진의 배경 이해용. 콘텐츠에 설명/인용 금지.

### 최초 기록

| 문헌 | 시기 | 내용 |
|------|------|------|
| 설문해자(說文解字) | 중국 한대 | "해치(獬豸) — 소와 비슷하고 뿔이 하나" |
| 해동잡록 | 조선 | 사헌부 관복에 해태 문양 사용 기록 |
| 광해군일기 | 조선 | 경복궁 해태상 설치 기록 |

### 존재론적 기원

> **"옳고 그름을 가리며, 거짓을 행하는 자를 뿔로 받는다"**

- 법(法)과 정의(正義)의 상징
- 불을 먹는 존재 → 화재 방지 (경복궁)
- 사시사철 지켜보지만 개입하지 않음

### 경복궁 해태상의 의미

| 요소 | 의미 |
|------|------|
| 위치 | 광화문 앞 — 궁궐(권력)의 경계 |
| 방향 | 관악산(화산) — 화기(火氣) 제어 |
| 자세 | 앉아 있음 — 지켜봄, 움직이지 않음 |
| 뿔 | 단일 — 옳고 그름의 판단 (양시비비가 아님) |

### 사헌부(司憲府) 연계

- 조선시대 사헌부 관원 흉배에 해태 문양
- 사헌부 = 관리 감찰, 비위 규탄
- 해태 = **"지켜보고, 판단하지만, 스스로 행하지 않는다"**

### 전통적 특성 (결에서 변용)

| 전통 특성 | 결에서의 해석 |
|----------|-------------|
| 법과 정의 | → 시스템, 질서 |
| 거짓 판단 | → 관찰, 암묵적 판단 |
| 화재 방지 | → 도깨비(火)와의 대척점 |
| 움직이지 않음 | → 정적 존재, 조각상 |
| 뿔로 처벌 | → (사용 안 함) 개입 금지 |

### 결에서 채택/배제

| 채택 | 배제 |
|------|------|
| 관찰자/심판자 | 적극적 처벌 |
| 경계 지킴 | 수호신 |
| 정적/부동 | 전투/액션 |
| 사자-용 혼합형 | 친근한 이미지 |
| 법/질서 상징 | 선한 도움 |

---

## 역할

- 지켜본다
- 판단한다 (하지만 드러내지 않음)
- 개입하지 않는다 (거의)
- 경계를 지킨다
- 법과 정당성을 상징한다

> 도깨비가 개입하는 힘이라면,
> 해태는 **관찰하는 질서**다.

---

## 태도

- 정적
- 무거운 존재감
- 시스템적 질서의 표상
- 감정 없음
- **"그냥 거기 있는 것"**

---

## 존재 방식

- 움직이지 않는다
- 항상 **"이미 거기 있었다"**
- 배경의 일부처럼 존재
- 갑자기 나타나지 않음
- 카메라가 발견할 뿐

---

## 시각적 키워드 (15개)

### 형태
1. 사자 + 용의 혼합 형태
2. 비늘과 갈기
3. 돌/청동 질감
4. 대칭적 자세
5. 거대한 스케일

### 존재감
6. 정면 응시 (또는 45도)
7. 움직이지 않는 눈
8. 연기/안개 위에 위치
9. 건축물과 일체화
10. 빛을 받지 않는 실루엣

### 배치
11. 높은 곳에 위치
12. 프레임 끝에서 지켜봄
13. 배경처럼 존재
14. 문/관문 옆에 위치
15. 하늘/산과 함께

---

## 동작 키워드 (12개)

### 기본 (대부분)
1. 정지 (기본값)
2. 미세한 호흡 (유일한 움직임)
3. 응시 유지
4. 시선 이동 (극히 느리게)

### 특별 (극히 드묾)
5. 고개 돌림 (한 영상에 1회 이하)
6. 일어섬 (클라이맥스에서만)
7. 걸음 (극히 드묾, 거의 금지 수준)
8. 울음소리 (음향으로만)

### 등장/퇴장
9. 이미 있었던 듯 나타남
10. 페이드로 사라짐
11. 안개 속으로 녹음
12. 카메라가 돌아오면 없음

## 등장 규칙

| 규칙 | 값 | 설명 |
|------|-----|------|
| 등장 빈도 | 1-2회/영상 | 희소해야 무거움 유지 |
| 등장 길이 | 5-15초 | 길게 유지, 압도적 존재감 |
| 등장 위치 | 배경/프레임 끝/높은 곳 | 중앙 전면 금지 |
| 카메라 반응 | 등장 시 카메라 멈춤 | 경외감 표현 |
| 음악 반응 | 음악 최소화 또는 저음 드론 | 고요함 강조 |

## Phase별 등장 전략

| Phase | 등장 방식 | 의미 |
|-------|----------|------|
| Phase A | 멀리서 지켜봄 | 질서의 존재 암시 |

| Phase C | 미세한 반응 (고개 돌림) | 판단의 순간 |
| Phase 0 | 기원 | 질서의 시작 |

## 해태 존재 형태

> **중요**: 조각상과 살아있는 해태는 **같은 존재**다. 구분하지 않는다.

| 형태 | 특징 | 사용 상황 |
|------|------|----------|
| 조각상 | 완전히 정지, 돌/청동 질감 | 대부분의 장면 |
| 미동 | 호흡/갈기 미세 움직임 | 긴장 고조 시 |
| 시선 이동 | 눈만 움직임 | Phase C 클라이맥스 |
| 기동 | 고개 돌림 (극히 드묾) | Phase C 최종 트랙만 |

### 형태 전환 원칙
- 조각상 → 미동 → 시선 → 기동 순서로 활성화
- Phase A-B: 조각상만
- Phase C: 미동, 시선 허용
- 기동은 전체 시리즈에서 1-2회만

### "이미 있었다" 원칙
- 해태가 "나타남" ❌
- 해태가 "발견됨" ⭕
- 조각상이든 살아있든 **원래 거기 있었다**

## 다른 존재와의 관계

### 해태 ↔ 인물
- 해태는 지켜볼 뿐
- 인물은 해태를 의식하지 못함
- 같은 프레임에 있어도 상호작용 없음
- 인물이 해태를 발견해도 무반응

### 해태 ↔ 도깨비
- 직접 충돌하지 않음
- 같은 프레임 등장 드묾
- 긴장감으로만 표현
- 질서 vs 개입의 대립

### 해태 ↔ 탈
- 탈 쓴 군중을 내려다봄
- 계급/위계의 시각화
- 탈은 해태를 보지 않음

### 오방색 연계 (제작진 레퍼런스)

| 청(靑) `#2c3e50` | 동/봄/목 | 남색 기조, 냉정함 |

## Phase C 미세 반응 상세

해태의 미세 반응은 Phase C에서만 허용:

| 반응 | 조건 | 표현 방법 |
|------|------|----------|
| 시선 이동 | 인물의 결정적 행동 시 | 3초에 걸쳐 천천히 |
| 고개 돌림 | 클라이맥스에서 1회만 | 5초 이상 소요 |
| 눈 깜빡임 | 극히 드묾 | 한 영상에 1회 이하 |
| 호흡 변화 | 긴장 고조 시 | 갈기/비늘 미세 움직임 |

### 금지되는 반응
- 빠른 고개 돌림 ❌
- 일어섬 (Phase C에서도) ❌
- 걸음 ❌
- 울음소리 (음향으로만 허용) ❌
- 인물을 향해 다가감 ❌

### 반응 타이밍
- 인물 행동 → 2-3초 딜레이 → 해태 반응
- 즉각 반응 금지 (인과관계 모호하게)
- 반응인지 우연인지 알 수 없어야 함

## 등장 예고 신호

해태는 예고 없이 등장하지 않음. **이미 있었다**가 원칙:

| 발견 방법 | 설명 |
|----------|------|
| 카메라 이동 | 패닝/틸트로 해태 발견 |
| 포커스 변화 | 배경에서 해태로 포커스 이동 |
| 안개 걷힘 | 안개가 걷히며 해태 드러남 |
| 빛 변화 | 역광 → 실루엣 인식 |

→ "나타남"이 아니라 "발견됨"

## 조각상→미동 전환 기법

### 전환 단계

| 단계 | 상태 | 표현 | 지속 시간 |
|------|------|------|----------|
| 1 | 완전 정지 | 텍스처만 | 기본 상태 |
| 2 | 미세 진동 | 0.5% 흔들림 | 2-3초 |
| 3 | 호흡 | 갈기/비늘 미세 움직임 | 5-10초 |
| 4 | 시선 | 눈만 천천히 이동 | 3-5초 |
| 5 | 고개 | 고개 돌림 (극히 드묾) | 5초 이상 |

### 전환 연출 기법

| 기법 | 설명 | 효과 |
|------|------|------|
| 포커스 변화 | 배경→해태로 포커스 이동 | 발견 느낌 |
| 미세 애니메이션 | 1-2% 움직임 루프 | 호흡 암시 |
| 조명 변화 | 빛이 해태로 이동 | 활성화 암시 |
| 사운드 추가 | 저음 드론 증가 | 존재감 강화 |

### 금지 전환 기법

- 갑작스런 움직임 ❌
- 눈 깜빡임 (빠르게) ❌
- 일어서기 (대부분 금지) ❌
- 모핑/변신 ❌
