# STYLE_GUIDE_AUDIO — 결 오디오 스타일 가이드

> **이 문서는 불변(immutable) 문서입니다.**
> 음악은 감정을 설명하지 않습니다. 결을 만들 뿐입니다.

---

## 1. 핵심 원칙

### 음악의 역할
- 감정을 설명하지 않는다
- 감정을 몰고 가는 주체가 아니다
- **움직임을 흔드는 진동**이다

### 핵심 공식

> **전통 소리 = 재료 (Material)**
> **K-pop = 형식 (Form)**
> **AI 시대 = 상태 (State, shown not said)**

---

## 2. 전통 악기 사용 원칙

### 절대 규칙
**전통 악기를 "전통처럼" 쓰면 실패한다.**

### 피해야 할 것 ❌
- 전통 국악 멜로디 패턴
- 민속 분위기
- 판소리식 감정 과잉
- 악기 "소개"처럼 들리는 사용

### 성공 공식 ⭕
> **전통 악기 = 텍스처**
> **K-pop = 구조**
> **AI 시대 = 리듬과 반복**

---

## 3. 악기별 역할 정의

### 가야금 (Gayageum)

**역할**: 결의 선 (Line of 결)

| 항목 | 규칙 |
|------|------|
| 멜로디 | ❌ 금지 |
| 사용법 | 반복 리프, 신스 패드처럼 깔기 |
| 느낌 | 차갑고 세련됨 |
| 배치 | Intro, Break, Outro |

### 장구 (Janggu)

**역할**: 인간의 호흡 (Human Breath)

| 항목 | 규칙 |
|------|------|
| 전통 장단 | ❌ 금지 |
| 사용법 | 트랩/드릴 비트 위에 레이어 |
| 느낌 | 살아 있음, 불완전함 |
| 배치 | Drop 섹션, 리듬 강조 |

### 태평소 / 피리 (Taepyeongso / Piri)

**역할**: 경고 / 신호 (Warning / Signal)

| 항목 | 규칙 |
|------|------|
| 긴 연주 | ❌ 금지 |
| 사용법 | 짧은 찢어지는 샘플 |
| 느낌 | 불안, 각성 |
| 배치 | 전환 직전, 클라이맥스 진입 |

### 사용량 규칙
- 이 세 악기면 충분하다
- **더 많이 쓰면 = "국악"이 된다**
- 전통 악기는 전체 사운드의 **20-30%** 이하

---

## 4. "고유 K-pop" 구조

### K-pop의 본질
K-pop은 "한국 음악"이 아니라:
> **장르 혼합 + 강한 시각성 + 퍼포먼스 중심 구조**

### 곡 구조 템플릿

```
[Intro] - 전통 악기 솔로 (10-15초)
    ↓
[Verse 1] - 미니멀, 공간감
    ↓
[Pre-Chorus] - 빌드업
    ↓
[Drop 1] - 트랩/하이퍼팝, 장구 레이어
    ↓
[Break] - 무음에 가까운 공간 (5-10초)
    ↓
[Drop 2] - 보컬 찢기, 최고조
    ↓
[Outro] - 전통 악기만 남김
```

### 이것은:
- **고유 K-pop** ⭕
- "국악 퓨전" ❌

---

## 5. AI 시대 간접 표현

### 피해야 할 것 ❌
- AI 관련 가사
- 로봇 이미지
- 디지털 텍스트 오버레이
- "미래"를 직접 언급

### 결 방식 ⭕
> **인간과 시스템의 리듬 차이**를 보여준다

### 음악에서의 표현

| 요소 | AI/시스템 | 인간 |
|------|----------|------|
| 비트 | 완벽한 반복 | 미세한 어긋남 |
| 퀀타이즈 | 100% 정확 | 90-95% |
| 악기 | 신스, 드럼머신 | 장구, 숨소리 |

### 메시지 (말하지 않음)
> AI는 완벽하지만,
> 우리는 결을 만든다.

## 6. 보컬 규칙

### 보컬의 정체성
- 캐릭터가 아니다
- 내레이터가 아니다
- **"세계 상태의 목소리"**다

### 보컬 스타일

| 스타일 | 사용 비율 | 설명 |
|--------|----------|------|
| 속삭임 | 30% | 공간감, 친밀감 |
| 샘플/파편 | 25% | 단어 조각, 반복 |
| 클린 보컬 | 25% | 후렴, 훅 |
| 찢어지는 보컬 | 15% | 클라이맥스 |
| 코러스/레이어 | 5% | 배경 |

### 가사 원칙
- 가사보다 **발음과 호흡**이 먼저
- 의미보다 **리듬**이 먼저
- 명확한 후렴 ❌
- 반복되는 훅 ⭕

### 가사 분량
- 전체 곡에서 가사 있는 부분: **40-50%**
- 나머지는 인스트루멘탈 또는 보컬 샘플

---

## 7. BPM & 키 가이드

### BPM 범위

| 분위기 | BPM | 사용 |
|--------|-----|------|
| 긴장 | 85-95 | Phase A 초반 |
| 기본 | 95-110 | 대부분의 트랙 |
| 고조 | 110-125 | 클라이맥스 트랙 |

### 키 가이드
- **마이너 키** 기본 (70%)
- 메이저 키는 "밝음"이 아닌 **"기이함"**으로 사용
- 전조: 한 곡에 최대 1회

### 일관성 규칙
- 같은 Phase 내 트랙들은 **연관된 키** 사용
- BPM 변화는 ±10 범위 내

---

## 8. Suno 사용 규칙

### 기본 제약
- 일관된 BPM과 키 범위
- 통제된 프롬프트 언어
- **변형보다 반복** 우선

### 프롬프트 구조 예시

```
[Style]
dark k-pop, trap beat, minimal vocal,
korean traditional instrument texture

[Instruments]
808 bass, hi-hats, gayageum riff,
janggu percussion layer

[Mood]
cold, dry, mysterious, not sad

[Avoid]
cheerful, epic orchestral, folk music feel
```

### 선택 기준
1. 결의 톤과 맞는가?
2. 전통 악기가 "소개"처럼 들리지 않는가?
3. 감정 과잉이 없는가?
4. 반복 구조가 있는가?

### 실패 규칙
- "거의 맞는데" 싶으면 → **버린다**

---

## 9. 믹싱 가이드

### 공간감
- 리버브: 중간~넓음
- 딜레이: 리듬에 맞춰 사용
- 전체적으로 **공간이 느껴지는** 믹스

### 주파수 배치

| 주파수 | 요소 |
|--------|------|
| Sub (20-60Hz) | 808, 베이스 |
| Low (60-250Hz) | 킥, 장구 저음 |
| Mid (250-2kHz) | 가야금, 보컬 |
| High (2k-8kHz) | 하이햇, 태평소 |
| Air (8k+) | 공기감, 노이즈 |

### 다이나믹스
- 과도한 컴프레션 ❌
- 숨 쉴 공간 필요
- 클라이맥스와 Break의 볼륨 차이 확보

---

## 10. 체크리스트

트랙 완성 전 확인:

- [ ] 전통 악기가 "소개"처럼 들리지 않는가?
- [ ] 가사가 감정을 직접 설명하지 않는가?
- [ ] 완벽한 비트 + 인간적 어긋남이 공존하는가?
- [ ] BPM이 지정 범위 내인가?
- [ ] 곡 구조가 템플릿을 따르는가?
- [ ] 밝은 느낌이 아닌가? (밝더라도 "기이함"인가?)
- [ ] 반복 구조가 있는가?
