## Suno 프롬프트

### Version 1
```
[Style]
dark k-pop, trap beat, minimal vocal,
atmospheric, mysterious, cool not sad

[Instruments]
808 bass, hi-hats, gayageum riff loop,
janggu percussion layer, ambient pads

[Mood]
cold, dry, mysterious, tense,
trembling, unstable, observing

[Structure]
intro with gayageum, build to trap drop,
silent break, intense second drop,
outro with only gayageum

[Avoid]
cheerful, epic orchestral, folk music feel,
emotional excess, clear chorus melody
```

### Version 2 (백업)
```
[Style]
minimal dark pop, korean traditional texture,
trap influenced, whisper vocals

[Instruments]
synth bass, percussion, gayageum as texture,
janggu rhythm layer

[Mood]
cool, detached, mysterious, tense

[Avoid]
traditional korean folk sound,
melodic gayageum, emotional vocals
```

### Version 3 (백업)
```
[Style]
experimental k-pop, dark ambient trap,
sparse arrangement, tension building

[Instruments]
deep bass, glitchy percussion,
gayageum drone, breath sounds

[Mood]
unsettling, watching, trembling

[Avoid]
upbeat, hopeful, clear structure
```
