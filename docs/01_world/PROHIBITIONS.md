# PROHIBITIONS — 절대 금지 규칙

> **이 문서가 필요한 순간:**
> - 콘텐츠 제작 전 금지 사항 확인
> - QC 체크 시 위반 여부 검증
> - "이거 해도 돼?"라는 질문에 답할 때

---

## 핵심 원칙

> **결은 설명하지 않는다.**
> **금지 규칙을 어기면 세계관이 무너진다.**

---

## 1. 표현 금지

| 금지 항목 | 이유 | 대안 |
|----------|------|------|
| 내레이션 ❌ | 설명은 결의 적 | 시각으로만 보여줌 |
| 세계관 설명 텍스트 ❌ | 팬이 해석해야 함 | 상징과 행동으로 암시 |
| 감정 과잉 ❌ | 쿨하게 | 절제된 표현, 침묵 활용 |
| 교훈 제시 ❌ | 정답 주지 않음 | 열린 결말, 질문만 남김 |
| 민속/설화 소개 ❌ | 전통 "소개"는 금지 | 현대적 재해석으로 사용 |

---

## 2. 해석 금지

| 금지 항목 | 이유 | 대안 |
|----------|------|------|
| 전통의 "소개"처럼 보이는 연출 ❌ | 결은 박물관이 아님 | 전통 요소를 텍스처로만 사용 |
| 캐릭터의 상징 과잉 소비 ❌ | 한 존재에 너무 많은 의미 부여 금지 | 한 트랙에 1-2개 테마만 |
| 선악 구도 ❌ | 결에 옳고 그름 없음 | 모든 선택을 동등하게 |
| 희망/구원 메시지 ❌ | 결은 구원하지 않음 | 열린 상태로 끝냄 |

---

## 3. Phase A 특별 금지 (팬덤 확장기)

> **Phase A에서 직구는 금물이다.**

| 금지 항목 | 대안 |
|----------|------|
| 지명/국가명 직접 언급 ❌ | "이름 없는 섬"으로 상징화 |
| 역사적 주장 직구 ❌ | 경계/지움/기록 테마로 간접 표현 |
| 정치적 논쟁 유발 표현 ❌ | 상징으로 보여주고 해석은 팬에게 |

## 4. 비주얼 금지

| 금지 항목 | 이유 | 대안 |
|----------|------|------|
| 순수 흰색 (#ffffff) ❌ | 너무 깨끗함 | 오프화이트, 아이보리 (#F5F5DC) |
| 순수 검정 (#000000) ❌ | 너무 단순함 | 먹색, 숯검정 (#1A1A1A) |
| 채도 높은 네온 ❌ | 결의 톤 파괴 | 채도 낮춘 색상 (S: 40-60%) |
| 파스텔 톤 ❌ | 결의 톤 파괴 | 탁한 색, 그레이 섞인 색 |
| 밝은 대낮 ❌ | 분위기 파괴 | 황혼, 새벽, 안개 낀 낮 |

## 5. 오디오 금지

| 금지 항목 | 이유 | 대안 |
|----------|------|------|
| 전통 국악 멜로디 패턴 ❌ | "국악" 느낌 금지 | 전통 음계를 현대적으로 변형 |
| 민속 분위기 ❌ | 전통 소개 금지 | 앰비언트, 일렉트로닉 질감 |
| 판소리식 감정 과잉 ❌ | 쿨하게 | 절제된 보컬, 속삭임 |
| 악기 "소개"처럼 들리는 사용 ❌ | 악기는 텍스처 | 악기음을 가공/레이어링 |
| 밝은 느낌 (희망적) ❌ | 밝더라도 "기이함"이어야 | 밝음 + 불협화음/텐션 |

## 6. 캐릭터 금지

### 도깨비
- 친근한 존재로 표현 ❌
- 직접 대화/설명 ❌
- 전통 도깨비 이미지 그대로 ❌
- 코믹 릴리프 ❌

### 해태
- 적극적 행동 ❌
- 인물과 대화 ❌
- "수호신"으로 설명 ❌
- 누군가를 구함 ❌

### 탈
- "전통 연희"로 소개 ❌
- 탈 쓴 인물이 탈을 벗음 ❌
- 탈 각각에 이름/역할 부여 ❌
- "재미있는 요소"로 사용 ❌

### 인물 (Human)
- 이름/신원 부여 ❌
- 얼굴 클로즈업 ❌
- 대사/독백 ❌
- 감정 표현 클로즈업 ❌
- 영웅적 행동 ❌

## 8. 빠른 체크리스트

콘텐츠 완성 전 확인:

### 표현
- [ ] 설명 텍스트 없음
- [ ] 교훈 없음
- [ ] 감정 과잉 없음

### 세계관
- [ ] 선악 구도 없음
- [ ] 희망/구원 없음
- [ ] 전통 "소개" 없음

### Phase A
- [ ] 지명/국가명 없음
- [ ] 역사 직구 없음
- [ ] 정치 논쟁 없음

### 비주얼
- [ ] 순수 흰/검 없음
- [ ] 네온/파스텔 없음
- [ ] 밝은 대낮 없음

### 캐릭터
- [ ] 금지 연출 없음
- [ ] 등장 규칙 준수

## 9. 금지 위반 수정 예시

위반을 발견했을 때 어떻게 수정하는가:

| 위반 | 문제 | 수정 방법 |
|------|------|----------|
| 내레이션 있음 | "옛날에..." 설명 | 시각으로만 표현, 텍스트 삭제 |
| 감정 과잉 | 울며 쓰러지는 장면 | 실루엣으로 축소, 무표정 유지 |
| 순수 흰색 | #ffffff 사용 | #f5f0e6 (종이색)으로 교체 |
| 밝은 대낮 | 정오 태양 | 안개 추가 또는 황혼으로 변경 |
| 탈을 벗음 | 정체 노출 | 벗는 장면 삭제, 컷 전환 |
| 도깨비 대화 | 말풍선/자막 | 삭제, 행동만으로 표현 |
| 영웅 서사 | "세계를 구함" | 열린 결말로 변경 |
| 선악 구도 | 악당 설정 | 모든 존재 동등하게 |

## 금지 위반 심각도 등급

### 등급 분류

| 등급 | 설명 | 예시 | 조치 |
|------|------|------|------|
| **Critical (치명)** | 세계관 파괴 | 내레이션, 선악 구도, 희망 메시지 | 즉시 수정 필수 |
| **Major (중대)** | 톤 파괴 | 감정 과잉, 밝은 대낮, 네온색 | 출시 전 수정 |
| **Minor (경미)** | 스타일 이탈 | 약간 밝음, 약간 설명적 | 가능하면 수정 |

### 등급별 상세

**Critical (즉시 수정):**
- 세계관 설명 텍스트
- 캐릭터 대사/독백
- 선악 판단
- 희망/구원 메시지
- 탈 벗기

**Major (출시 전 수정):**
- 순수 흰색/검정
- 밝은 대낮
- 감정 과잉 표현
- 전통 "소개" 느낌

**Minor (권장 수정):**
- 약간 밝은 톤
- 암시성 부족
- 색상 채도 높음

## 예외 허용 규칙

### 원칙

> **예외는 없다.**
>
> 하지만 다음은 **오해의 여지가 있는 것**이다.

### 예외가 아닌 것 (오해 방지)

| 상황 | 오해 | 진실 |
|------|------|------|
| 화면 일부만 밝음 | 밝음 금지 위반? | 단일 광원이면 OK |
| 탈 쓴 인물 등 보임 | 얼굴 클로즈업? | 탈 클로즈업은 OK |
| 빨간색 강조 | 네온? | 탁한 빨강 (#c23a3a)은 OK |
| 가야금 멜로디 | 국악? | 앰비언트 텍스처면 OK |

### 실제 예외 (극히 드묾)

| 상황 | 조건 | 승인 필요 |
|------|------|----------|
| Phase 0 특별 연출 | 기원 표현 목적 | 사전 승인 |
| 티저용 짧은 설명 | 5초 이하 | 사전 승인 |
| 메타 콘텐츠 (메이킹) | 본편 아님 | 자동 승인 |

## 위반 감지 방법

### 자동화 가능 항목

| 항목 | 감지 방법 | 도구 |
|------|----------|------|
| 순수 흰/검 | 색상 분석 | Photoshop/스크립트 |
| 채도 과다 | HSL 분석 | 색상 히스토그램 |
| 밝기 과다 | 휘도 측정 | 히스토그램 |
| 자막/텍스트 | OCR | 영상 분석 도구 |

### 수동 확인 필수 항목

| 항목 | 확인 방법 |
|------|----------|
| 선악 구도 | 서사 검토 |
| 감정 과잉 | 주관적 판단 |
| 전통 소개 느낌 | 톤 검토 |
| 희망 메시지 | 결말 검토 |

### 위반 감지 체크포인트

```
□ 자막/텍스트 스캔 (자동)
□ 색상 팔레트 분석 (자동)
□ 밝기/조명 분석 (반자동)
□ 서사 검토 (수동)
□ 톤 검토 (수동)
□ 캐릭터 규칙 (수동)
```

## 금지 사항 학습 가이드

### 신규 참여자 체크리스트

```
1단계: 문서 읽기
□ PROHIBITIONS.md 전체 읽기
□ 체크리스트 출력/저장

2단계: 예시 분석
□ 위반 예시 5개 분석
□ 수정 예시 5개 분석

3단계: 실습
□ 기존 콘텐츠에서 위반 찾기 연습
□ 발견한 위반 보고 작성

4단계: 검증
□ 첫 제작물 QC 체크
□ 피드백 반영
```

### 반복 학습 항목

| 항목 | 빈도 | 방법 |
|------|------|------|
| 표현 금지 | 매 제작 | 체크리스트 |
| 비주얼 금지 | 매 제작 | 색상 검증 도구 |
| 캐릭터 금지 | 월 1회 | 문서 재검토 |
| Phase 금지 | Phase 전환 시 | 문서 재검토 |
