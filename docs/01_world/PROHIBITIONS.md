# PROHIBITIONS — 절대 금지 규칙

> **이 문서가 필요한 순간:**
> - 콘텐츠 제작 전 금지 사항 확인
> - QC 체크 시 위반 여부 검증
> - "이거 해도 돼?"라는 질문에 답할 때

---

## 핵심 원칙

> **결은 설명하지 않는다.**
> **금지 규칙을 어기면 세계관이 무너진다.**

---

## 1. 표현 금지

| 금지 항목 | 이유 |
|----------|------|
| 내레이션 ❌ | 설명은 결의 적 |
| 세계관 설명 텍스트 ❌ | 팬이 해석해야 함 |
| 감정 과잉 ❌ | 쿨하게 |
| 교훈 제시 ❌ | 정답 주지 않음 |
| 민속/설화 소개 ❌ | 전통 "소개"는 금지 |

---

## 2. 해석 금지

| 금지 항목 | 이유 |
|----------|------|
| 전통의 "소개"처럼 보이는 연출 ❌ | 결은 박물관이 아님 |
| 캐릭터의 상징 과잉 소비 ❌ | 한 존재에 너무 많은 의미 부여 금지 |
| 선악 구도 ❌ | 결에 옳고 그름 없음 |
| 희망/구원 메시지 ❌ | 결은 구원하지 않음 |

---

## 3. Phase 1 특별 금지 (팬덤 확장기)

> **Phase 1에서 직구는 금물이다.**

| 금지 항목 | 대안 |
|----------|------|
| 지명/국가명 직접 언급 ❌ | "이름 없는 섬"으로 상징화 |
| 역사적 주장 직구 ❌ | 경계/지움/기록 테마로 간접 표현 |
| 정치적 논쟁 유발 표현 ❌ | 상징으로 보여주고 해석은 팬에게 |

**Phase 2-3에서 점진적으로 확장 가능**
→ [[04_strategy/PHASE_MAP]]

## 4. 비주얼 금지

| 금지 항목 | 이유 |
|----------|------|
| 순수 흰색 (#ffffff) ❌ | 너무 깨끗함 |
| 순수 검정 (#000000) ❌ | 너무 단순함 |
| 채도 높은 네온 ❌ | 결의 톤 파괴 |
| 파스텔 톤 ❌ | 결의 톤 파괴 |
| 밝은 대낮 ❌ | 분위기 파괴 |

→ [[02_style/VISUAL_COLOR]]

## 5. 오디오 금지

| 금지 항목 | 이유 |
|----------|------|
| 전통 국악 멜로디 패턴 ❌ | "국악" 느낌 금지 |
| 민속 분위기 ❌ | 전통 소개 금지 |
| 판소리식 감정 과잉 ❌ | 쿨하게 |
| 악기 "소개"처럼 들리는 사용 ❌ | 악기는 텍스처 |
| 밝은 느낌 (희망적) ❌ | 밝더라도 "기이함"이어야 |

→ [[02_style/AUDIO_CORE]]

---

## 6. 캐릭터 금지

### 도깨비
- 친근한 존재로 표현 ❌
- 직접 대화/설명 ❌
- 전통 도깨비 이미지 그대로 ❌
- 코믹 릴리프 ❌

### 해태
- 적극적 행동 ❌
- 인물과 대화 ❌
- "수호신"으로 설명 ❌
- 누군가를 구함 ❌

### 탈
- "전통 연희"로 소개 ❌
- 탈 쓴 인물이 탈을 벗음 ❌
- 탈 각각에 이름/역할 부여 ❌
- "재미있는 요소"로 사용 ❌

→ [[03_characters/]]

## 8. 빠른 체크리스트

콘텐츠 완성 전 확인:

### 표현
- [ ] 설명 텍스트 없음
- [ ] 교훈 없음
- [ ] 감정 과잉 없음

### 세계관
- [ ] 선악 구도 없음
- [ ] 희망/구원 없음
- [ ] 전통 "소개" 없음

### Phase 1
- [ ] 지명/국가명 없음
- [ ] 역사 직구 없음
- [ ] 정치 논쟁 없음

### 비주얼
- [ ] 순수 흰/검 없음
- [ ] 네온/파스텔 없음
- [ ] 밝은 대낮 없음

### 캐릭터
- [ ] 금지 연출 없음
- [ ] 등장 규칙 준수
