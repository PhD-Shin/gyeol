# THEMES — 메타 테마 (Implicit Themes)

> **이 문서가 필요한 순간:**
> - 콘텐츠가 어떤 테마를 담아야 하는지 확인할 때
> - 상징을 선택할 때
> - "이 장면이 뭘 의미하지?" 자문할 때

---

## 핵심 원칙

> **직접 언급하지 않는다. 형상으로만 보여준다.**

이 테마들은 명시적으로 말해지지 않지만,
전체 프로젝트에 스며 있어야 한다.

---

## 메타 테마 목록

| 테마 | 표현 방식 | 상징 예시 |
|------|----------|----------|
| **Border (경계)** | 지워지는 선, 모호한 영역 | 먹으로 번지는 경계선, 안개 |
| **Name (이름)** | 이름 없는 존재들 | 탈, 실루엣 인물 |
| **Record (기록)** | 흔적, 자국, 남겨진 것 | 지도, 도장, 흔적 |
| **Erasure (지움)** | 사라지는 형상, 먹히는 기억 | 페이드, 번짐, 삭제 |
| **Ritual (의례)** | 반복되는 동작, 형식 | 행렬, 굿, 춤 |
| **System vs Deviation** | 완벽한 반복 vs 인간적 어긋남 | 비트 vs 숨소리 |

---

## 테마별 상세

### Border (경계)

경계는 항상 모호하다.
- 안과 밖이 나뉘지 않는다
- 선은 그어지자마자 번진다
- 어디까지가 이쪽인지 알 수 없다

**표현 방법:**
- 먹 번짐 전환
- 안개로 끊기는 공간
- 프레임 가장자리의 존재

### Name (이름)

이름은 주어지지 않는다.
- 캐릭터에게 이름이 없다
- 장소에 이름이 없다
- 존재는 기능으로만 정의된다

**표현 방법:**
- 탈로 가려진 얼굴
- 실루엣 인물
- 익명의 군중

### Record (기록)

기록은 흔적으로만 남는다.
- 완전한 기록은 없다
- 남는 것은 자국뿐이다
- 누가 남겼는지 알 수 없다

**표현 방법:**
- 지도 위의 자국
- 찍힌 도장
- 남겨진 발자국

### Erasure (지움)

모든 것은 지워진다.
- 형상이 사라진다
- 기억이 먹힌다
- 슬퍼하지 않는다

**표현 방법:**
- 페이드 아웃
- 먹이 번지며 삼키는 장면
- 존재가 사라지는 순간

### Ritual (의례)

의례는 의미 없이 반복된다.
- 왜 하는지 설명하지 않는다
- 그냥 그렇게 해왔다
- 멈추면 세계가 어긋난다

**표현 방법:**
- 행렬
- 반복되는 춤
- 굿판의 리듬

### System vs Deviation

시스템은 완벽하고, 인간은 어긋난다.
- 완벽한 반복 = 시스템/AI
- 미세한 어긋남 = 인간/결
- 이 차이가 의미를 만든다

**표현 방법:**
- 비트 정확 vs 숨소리 불규칙
- 기계적 패턴 vs 손의 떨림
- 완벽한 대칭 vs 미세한 비대칭
