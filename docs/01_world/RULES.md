# RULES — 세계 작동 규칙 (Five Operating Rules)

> **이 문서가 필요한 순간:**
> - 세계가 어떻게 작동하는지 확인할 때
> - 새 콘텐츠가 세계관에 맞는지 검증할 때
> - 장면/서사의 논리를 점검할 때

---

## 규칙 계층

```
[1. 시간] ← 가장 근본적
    ↓
[2. 움직임] ← 시간 위에서 발생
    ↓
[3. 반복] ← 움직임의 패턴
    ↓
[4. 개입] ← 반복을 깨는 힘
    ↓
[5. 사라짐] ← 모든 것의 종착점
```

---

## 1. 시간 (Time)

- 직선이 아니다
- 과거·현재·미래가 구분되지 않는다
- 장면은 항상 **움직임의 중간**에서 시작된다

> 관객은 이유를 모른 채
> 결과가 아니라 **과정의 결**만을 목격한다.

---

## 2. 움직임 (Movement)

- 이유 없이 시작된다
- 완결되지 않는다
- 과정만 보여주고 결과는 보여주지 않는다

> 이미 시작된 싸움
> 이미 선택된 방향
> 이미 돌아갈 수 없는 지점

---

## 3. 반복 (Repetition)

- 시스템은 완벽하게 반복된다
- 인간은 미세하게 어긋난다
- 이 차이가 결을 만든다

> 완벽한 비트 vs 숨소리의 떨림
> 기계의 패턴 vs 손의 흔들림

---

## 4. 개입 (Intervention)

- 외부의 힘이 흐름을 흔든다
- 그것이 좋은지 나쁜지는 판단하지 않는다
- 개입은 변화일 뿐이다

> 도깨비는 개입한다.
> 해태는 지켜본다.
> 어느 쪽이 옳은지 결은 말하지 않는다.

---

## 5. 사라짐 (Disappearance)

- 모든 것은 결국 사라진다
- 남는 것은 흔적뿐이다
- 사라짐을 슬퍼하지 않는다

> 지워지는 이름
> 먹히는 기억
> 그것도 하나의 결이다.

---

## 규칙과 존재의 연결

| 규칙 | 주요 존재 | 표현 |
|------|----------|------|
| 시간 | 모든 존재 | 장면의 시작점 |
| 움직임 | 인물 | 미완결 동작 |
| 반복 | 탈 (군중) | 일렬 배치, 동시 움직임 |
| 개입 | 도깨비 | 리듬 파괴, 등장 |
| 사라짐 | 해태 | 관찰 후 페이드 |

---

## 규칙 충돌 시 우선순위

1. **사라짐 > 개입**: 개입도 결국 사라진다
2. **시간 > 움직임**: 시간 없이 움직임 없다
3. **반복 > 개입**: 개입은 반복 속에서만 의미가 있다

---

## Phase별 규칙 적용

| 규칙 | Phase A | Phase B | Phase C | Phase 0 |
|------|---------|---------|---------|---------|
| 시간 | 중간에서 시작 | 더 모호 | 축적된 느낌 | 태초 |
| 움직임 | 미완결 | 흐려짐 | 흔적으로 | 첫 움직임 |
| 반복 | 명확 | 어긋남 증가 | 기록됨 | 첫 반복 |
| 개입 | 암시 | 활발 | 감소 | 없음 |
| 사라짐 | 시작 | 진행 | 완료 | 이전 |

---

## 규칙별 표현 예시

### 시간 (Time)

| 시각 | 청각 |
|------|------|
| 장면 중간에서 시작하는 컷 | 음악 중간에서 시작 |
| 과거/현재 구분 없는 편집 | 리버브로 시간 모호화 |
| 시계/달력 없음 | 템포 변화로 시간 왜곡 |

### 움직임 (Movement)

| 시각 | 청각 |
|------|------|
| 미완결 동작 (휘두름 → 완료 안 보여줌) | 멜로디 해결 안 됨 |
| 쓰러짐 과정만 | 빌드업 후 드롭 없이 끝 |
| 도착점 안 보여줌 | 곡 끝이 열려 있음 |

### 반복 (Repetition)

| 시각 | 청각 |
|------|------|
| 탈 군중 일렬 배치 | 루프 패턴 |
| 단청 패턴 반복 | 퀀타이즈 100% 구간 |
| 기계적 동시 움직임 | 정확한 비트 |

### 개입 (Intervention)

| 시각 | 청각 |
|------|------|
| 도깨비 등장 | 리듬 브레이크 |
| 하나만 다른 방향 | 불협화음 삽입 |
| 카메라 흔들림 | 비트 어긋남 |

### 사라짐 (Disappearance)

| 시각 | 청각 |
|------|------|
| 페이드 아웃 | 사운드 페이드 |
| 먹 번짐으로 삼킴 | 리버브 증가 |
| 안개 속으로 | 에코로 사라짐 |

---

## 규칙 적용 강도

각 규칙은 Phase와 장면에 따라 적용 강도가 다르다:

| 강도 | 설명 | 예시 |
|------|------|------|
| 100% | 규칙 완전 적용 | 시간 규칙 모든 장면 |
| 80% | 강하게 적용 | 반복 규칙 의례 장면 |
| 50% | 중간 적용 | 개입 규칙 일반 장면 |
| 20% | 암시만 | 사라짐 규칙 초반 |

### Phase별 규칙 강도

| 규칙 | Phase A | Phase B | Phase C | Phase 0 |
|------|---------|---------|---------|---------|
| 시간 | 100% | 100% | 100% | 100% |
| 움직임 | 80% | 80% | 60% | 100% |
| 반복 | 80% | 60% | 40% | 100% |
| 개입 | 20% | 80% | 50% | 0% |
| 사라짐 | 20% | 50% | 100% | 0% |

## 규칙 위반 시 (안티패턴)

규칙을 위반하면 "결"이 아니라 "일반 MV"가 된다:

### 시간 위반 예시

| 위반 | 보이는 현상 | 문제 |
|------|-----------|------|
| 직선 시간 | 시작→중간→끝 구조 | 서사적, 설명적 |
| 과거 회상 명시 | "3년 전..." 자막 | 설명하려 함 |
| 결말 보여줌 | 해피/새드 엔딩 | 판단하려 함 |

### 움직임 위반 예시

| 위반 | 보이는 현상 | 문제 |
|------|-----------|------|
| 동작 완결 | 칼 휘두름 → 베임 | 결과 보여줌 |
| 목적 달성 | 문 열고 들어감 | 완결됨 |
| 감정 해소 | 울다 → 웃음 | 카타르시스 강요 |

### 반복 위반 예시

| 위반 | 보이는 현상 | 문제 |
|------|-----------|------|
| 반복 없음 | 매 컷이 다름 | 패턴 없음 |
| 어긋남 없음 | 완벽한 동일 반복 | 인간 부재 |
| 과한 어긋남 | 모두 다른 방향 | 혼란이 결이 아님 |

### 개입 위반 예시

| 위반 | 보이는 현상 | 문제 |
|------|-----------|------|
| 개입 판단 | 도깨비=악당 | 선악 규정 |
| 개입 설명 | "왜 개입했나" 보여줌 | 동기 부여 |
| 개입 결과 | "개입 덕분에 구원" | 인과관계 명시 |

### 사라짐 위반 예시

| 위반 | 보이는 현상 | 문제 |
|------|-----------|------|
| 사라짐 슬픔 | 눈물, 애도 장면 | 감정 강요 |
| 사라짐 저항 | "사라지지 마!" | 교훈적 |
| 사라짐 설명 | "이래서 사라졌다" | 설명 |

---

## 규칙 체크리스트

콘텐츠 검증 시 확인:

- [ ] 시간이 직선으로 흐르지 않는가?
- [ ] 움직임이 완결되지 않는가?
- [ ] 시스템과 인간의 반복 차이가 있는가?
- [ ] 개입을 판단하지 않는가?
- [ ] 사라짐을 슬퍼하지 않는가?
- [ ] 위의 안티패턴에 해당하지 않는가?

## 규칙 동시 적용 시나리오

### 두 규칙이 만날 때

| 조합 | 상황 | 해결 |
|------|------|------|
| 시간 + 움직임 | 과거 회상 중 동작 | 중간에서 시작, 중간에서 끝 |
| 반복 + 개입 | 패턴 중 도깨비 등장 | 반복 유지하다 하나만 어긋남 |
| 개입 + 사라짐 | 도깨비 등장 후 사라짐 | 개입 순간 후 페이드 |
| 움직임 + 사라짐 | 움직이다 사라짐 | 완결 없이 페이드 |
| 반복 + 사라짐 | 반복되다 사라짐 | 반복 횟수 줄며 페이드 |

### 세 규칙 조합 (고급)

| 조합 | 예시 장면 |
|------|----------|
| 시간+반복+개입 | 언제인지 모를 때, 탈 군중 반복 중 도깨비 암시 |
| 움직임+개입+사라짐 | 걷는 중 도깨비 스침 후 안개 속으로 |
| 반복+개입+사라짐 | 의례 반복 중 어긋남 발생 후 서서히 사라짐 |

## 규칙 구현 체크포인트

### 시간 (Time)

```
□ 시작 지점: 이미 시작된 상태인가?
□ 끝 지점: 완결되지 않았는가?
□ 시간 표시: 시계/달력/자막 없는가?
□ 시제: 과거/현재/미래 구분 불가능한가?
```

### 움직임 (Movement)

```
□ 동작 시작: 이미 진행 중인가?
□ 동작 끝: 완료 안 보여주는가?
□ 결과: 결과 장면 없는가?
□ 목적: 목적 달성 장면 없는가?
```

### 반복 (Repetition)

```
□ 패턴: 반복 패턴 있는가?
□ 어긋남: 미세한 어긋남 있는가?
□ 비율: 시스템 90% / 어긋남 10%?
□ 인간: 인간 요소에서 어긋남 발생하는가?
```

### 개입 (Intervention)

```
□ 개입자: 도깨비/외부 힘 등장하는가?
□ 판단: 좋고 나쁨 판단 안 하는가?
□ 원인: 개입 이유 설명 안 하는가?
□ 결과: 개입 결과 판단 안 하는가?
```

### 사라짐 (Disappearance)

```
□ 사라짐: 무언가 사라지는가?
□ 흔적: 흔적만 남는가?
□ 감정: 슬픔/저항 없는가?
□ 방식: 페이드/먹번짐/안개 등인가?
```

## 규칙 위반 복구 방법

### 시간 위반 복구

| 위반 | 복구 방법 |
|------|----------|
| 시작점이 명확함 | 컷 트리밍으로 중간에서 시작 |
| 끝점이 완결됨 | 컷 트리밍 또는 페이드 아웃 |
| 시간 표시 있음 | 해당 요소 제거/블러 |

### 움직임 위반 복구

| 위반 | 복구 방법 |
|------|----------|
| 동작 완결됨 | 완결 직전에서 컷 |
| 결과 보여줌 | 결과 장면 삭제 |
| 목적 달성 | 오픈 엔딩으로 변경 |

### 반복 위반 복구

| 위반 | 복구 방법 |
|------|----------|
| 반복 없음 | 패턴 요소 추가 |
| 어긋남 없음 | 하나 다르게 편집 |
| 어긋남 과다 | 어긋남 축소, 패턴 강화 |

### 개입 위반 복구

| 위반 | 복구 방법 |
|------|----------|
| 개입 판단 | 판단 요소 제거 |
| 개입 설명 | 설명 텍스트/장면 삭제 |
| 개입 결과 판단 | 결과 판단 장면 삭제 |

### 사라짐 위반 복구

| 위반 | 복구 방법 |
|------|----------|
| 슬픔 표현 | 감정 장면 축소/삭제 |
| 저항 표현 | 저항 장면 삭제 |
| 사라짐 설명 | 설명 텍스트 삭제 |

## 규칙별 음악 구현

### 시간 (Time)

| 요소 | 구현 |
|------|------|
| BPM | 트랙 중간 BPM으로 시작 |
| 인트로 | 페이드 인 없이 바로 시작 |
| 아웃트로 | 해결 없이 열린 상태로 끝 |
| 템포 | 느려지다 중간에서 끝 |

### 움직임 (Movement)

| 요소 | 구현 |
|------|------|
| 멜로디 | 해결음 없이 서스펜션 |
| 코드 진행 | 완결 코드 (I) 회피 |
| 리듬 | 4마디 루프 중간에서 컷 |
| 빌드업 | 클라이맥스 직전 컷 |

### 반복 (Repetition)

| 요소 | 구현 |
|------|------|
| 퀀타이즈 | 시스템 = 100%, 인간 = 90% |
| 루프 | 4-8마디 반복 |
| 어긋남 | 16분음표 1개 지연/앞당김 |
| 휴머나이즈 | 벨로시티 ±10% |

### 개입 (Intervention)

| 요소 | 구현 |
|------|------|
| 리듬 | 갑작스러운 브레이크 |
| 하모니 | 불협화음 삽입 |
| 다이나믹 | 갑자기 +6dB 또는 무음 |
| 주파수 | 저역 드론 또는 고역 피드백 |

### 사라짐 (Disappearance)

| 요소 | 구현 |
|------|------|
| 볼륨 | 4-8마디 페이드 아웃 |
| EQ | 고역 먼저 페이드 |
| 리버브 | 점점 길어짐 (2초 → 10초) |
| 악기 | 하나씩 빠짐 |

## 규칙 강도 측정

### 측정 방법

| 규칙 | 측정 항목 | 점수화 |
|------|----------|--------|
| 시간 | 시작/끝 모호함 정도 | 1-5 |
| 움직임 | 미완결 비율 | % |
| 반복 | 패턴 vs 어긋남 비율 | X:Y |
| 개입 | 개입 순간 수 | 카운트 |
| 사라짐 | 페이드 시간 | 초 |

### 점수 기준

**시간 (1-5점)**
- 1점: 시작/끝 명확
- 3점: 약간 모호
- 5점: 완전히 중간에서

**움직임 (미완결 %)**
- 50% 이하: 부족
- 70-90%: 적정
- 100%: 완벽

**반복 (시스템:어긋남)**
- 100:0: 기계적
- 90:10: 적정 (권장)
- 70:30: 과도한 어긋남
