# TONE — 톤 슬라이더 (기준값)

> **이 문서가 필요한 순간:**
> - 콘텐츠 톤이 "결다운지" 확인할 때
> - 분위기 조절의 기준이 필요할 때
> - 비교 대상(Kage Bow)과의 차이를 확인할 때

---

## 톤 슬라이더 (기본값)

| 축 | 값 | 설명 | 근거 |
|----|-----|------|------|
| 밝음 ↔ 어둠 | **45/100** | 중간보다 살짝 어둡게 | 황혼/새벽 시간대 기준 |
| 감정 ↔ 쿨 | **75/100** | 쿨 쪽으로 강하게 | Kage Bow 수준 참조 |
| 전통 ↔ 미래 | **50/100** | 균형 | 시대 특정 불가능 목표 |
| 설명 ↔ 암시 | **90/100** | 거의 암시만 | MANIFESTO 원칙 |

---

## 시각화

```
밝음 |---------●--------------------| 어둠
                ↑ 45

감정 |-------------------------●----| 쿨
                              ↑ 75

전통 |--------------●---------------| 미래
                   ↑ 50

설명 |-----------------------------●| 암시
                                  ↑ 90
```

---

## 축별 상세

### 밝음 ↔ 어둠 (45/100)

- **완전 어둠은 아님** — 고딕/호러가 아니다
- **완전 밝음은 절대 아님** — 희망찬 분위기 금지
- 빛과 어둠 사이 어딘가
- **시각 참조**: 김홍도 그림의 배경톤, 박수근의 회색조, 안개 낀 새벽

### 감정 ↔ 쿨 (75/100)

- **강하게 쿨** — Kage Bow급 절제
- 감정을 설명하지 않음
- 슬픔도 분노도 직접 보여주지 않음
- **"슬프다"가 아니라 "멋있다"**

### 전통 ↔ 미래 (50/100)

- **정확히 균형**
- 전통만 = "국악" 느낌
- 미래만 = "사이버펑크" 느낌
- **둘이 섞여서 시대를 알 수 없게**

### 설명 ↔ 암시 (90/100)

- **거의 암시만**
- 직접 말하지 않음
- 팬이 해석하도록
- **설명하는 순간 결이 아님**

---

## Phase별 톤 변화

| Phase | 밝음↔어둠 | 감정↔쿨 | 전통↔미래 | 설명↔암시 |
|-------|---------|---------|---------|---------|
| Phase A | 45 | 75 | 50 | 90 |
| Phase B | 40 | 80 | 45 | 85 |
| Phase C | 35 | 70 | 55 | 80 |
| Phase 0 | 30 | 85 | 60 | 95 |

**해석:**
- **Phase A**: 기본값 — 균형 잡힌 기이함
- **Phase B**: 더 어둡고 쿨 — 경계가 흐려지는 분위기
- **Phase C**: 전통 방향으로 약간 이동 — 기록/흔적 강조
- **Phase 0**: 가장 어둡고 암시적 — 기원, 시작 이전

→ [[04_strategy/PHASE_MAP]]

## 오디오 톤 가이드

| 축 | 기본값 | 표현 |
|----|--------|------|
| 전통↔현대 | **50/100** | 국악 음계 + 일렉트로닉 질감 |
| 멜로디↔텍스처 | **70/100** | 멜로디보다 질감 중심 |
| 빠름↔느림 | **40/100** | 중간~느린 템포 |
| 깨끗↔거친 | **65/100** | 약간의 노이즈, 텍스처 |

**오디오 참조:**
- 장영규의 앰비언트 작업
- 전통 악기의 현대적 사용 (이날치 스타일이 아님, 더 절제)
- 사운드스케이프, 필드 레코딩 요소

---

## 톤 허용 범위

각 축에서 결의 정체성을 유지하는 범위:

| 축 | 기본값 | 허용 범위 | 벗어나면 |
|----|--------|----------|----------|
| 밝음↔어둠 | 45 | 30-55 | <30: 호러, >55: 희망적 |
| 감정↔쿨 | 75 | 65-90 | <65: 멜로, >90: 냉소적 |
| 전통↔미래 | 50 | 40-65 | <40: 복고, >65: SF |
| 설명↔암시 | 90 | 80-95 | <80: 설명적, >95: 무의미 |

> **허용 범위를 벗어나면 결이 아니다.**

---

## 존재별 톤 차이

각 존재가 등장할 때 톤 조정:

| 존재 | 어둠 | 쿨 | 전통 | 암시 |
|------|------|-----|------|------|
| 도깨비 | +5 | +5 | -5 | +5 |
| 해태 | +10 | +10 | +5 | +5 |
| 탈 | 0 | +5 | +10 | 0 |
| 인물 | -5 | -5 | 0 | -5 |

**해석:**
- **도깨비**: 더 어둡고 쿨하게, 덜 전통적
- **해태**: 가장 어둡고 쿨하게, 약간 더 전통적
- **탈**: 톤 유지, 전통 방향 강화
- **인물**: 약간 밝고 감정적 (상대적으로)

## Kage Bow와의 비교

| 요소 | Kage Bow | 결 |
|------|----------|-----|
| 중심 | 그림자의 형상 | 흐름의 방향 |
| 톤 | 쿨 / 스타일 | 쿨 / 스타일 |
| 감정 | 최소화 | 최소화 |
| 해석 | 팬에게 위임 | 팬에게 위임 |
| 문화권 | 일본 | 한국 |
| 상징 | 여우 가면, 우키요에 | 도깨비, 탈, 한지 |

> **결은 한국어로 만든 Kage Bow의 사촌격이다.**
> 하지만 복제가 아니라 **독자적 정체성**을 갖는다.

---

## 톤 체크 질문

콘텐츠 검증 시 자문:

1. **일본 요소가 없어도 성립하는가?**
   → 아니면 한국적 상징으로 대체

2. **"예쁘다"보다 "기이하다"가 먼저 느껴지는가?**
   → 아니면 낯설게

3. **'악마'가 아니라 '질서(시스템)'가 느껴지는가?**
   → 선악이 느껴지면 수정

4. **설명 없이도 의미가 전달되는가?**
   → 설명이 필요하면 암시로 전환

## 톤 측정 방법

### 정량적 측정

| 축 | 측정 방법 | 도구 |
|----|----------|------|
| 밝음↔어둠 | 평균 휘도 | 히스토그램 분석 |
| 전통↔미래 | 전통 요소 비율 | 수동 카운트 |

### 정성적 측정 (설문 기반)

| 축 | 질문 | 응답 척도 |
|----|------|----------|
| 감정↔쿨 | "감정이 느껴지나요?" | 1(매우) ~ 5(전혀) |
| 설명↔암시 | "설명이 필요한가요?" | 1(매우) ~ 5(전혀) |

### 자가 체크

```
□ 화면 밝기: 히스토그램 40-50% 영역 집중?
□ 색온도: 4500-5500K 사이?
□ 대비: 중간~높음?
□ 설명 텍스트 없음?
□ 감정 표현 최소?
```

### 변화 패턴

## 톤 이탈 복구 가이드

### 이탈 감지

| 축 | 이탈 징후 |
|----|----------|
| 어둠 | 화면 밝음, 색온도 높음 |
| 쿨 | 감정 표현, 멜로디 과다 |
| 전통 | 국악 느낌 또는 SF 느낌 |
| 암시 | 설명 텍스트, 내레이션 |

### 복구 방법

**밝음 이탈 복구:**
```
1. 컬러 그레이딩: 노출 -0.5 ~ -1
2. 색온도: 5000K 이하로
3. 비네팅 추가: 20-30%
4. 필요시 안개/그레인 오버레이
```

**감정 과잉 복구:**
```
1. 감정 장면 길이 50% 축소
2. 클로즈업 → 롱샷 대체
3. 사운드: 보컬 볼륨 -3dB
4. 편집: 컷 속도 늦추기
```

**전통 과잉 복구:**
```
1. 전통 악기 볼륨 -6dB
2. 앰비언트 레이어 추가
3. 이펙트: 리버브, 딜레이
4. 비주얼: 현대 텍스처 추가
```

**설명 과잉 복구:**
```
1. 텍스트 삭제
2. 내레이션 삭제
3. 시각적 암시로 대체
4. 열린 결말로 변경
```
