# TONE — 톤 슬라이더 (기준값)

> **이 문서가 필요한 순간:**
> - 콘텐츠 톤이 "결다운지" 확인할 때
> - 분위기 조절의 기준이 필요할 때
> - 비교 대상(Kage Bow)과의 차이를 확인할 때

---

## 톤 슬라이더

| 축 | 값 | 설명 |
|----|-----|------|
| 밝음 ↔ 어둠 | **45/100** | 중간보다 살짝 어둡게 |
| 감정 ↔ 쿨 | **75/100** | 쿨 쪽으로 강하게 |
| 전통 ↔ 미래 | **50/100** | 균형 |
| 설명 ↔ 암시 | **90/100** | 거의 암시만 |

---

## 시각화

```
밝음 |---------●--------------------| 어둠
                ↑ 45

감정 |-------------------------●----| 쿨
                              ↑ 75

전통 |--------------●---------------| 미래
                   ↑ 50

설명 |-----------------------------●| 암시
                                  ↑ 90
```

---

## 축별 상세

### 밝음 ↔ 어둠 (45/100)

- **완전 어둠은 아님** — 고딕/호러가 아니다
- **완전 밝음은 절대 아님** — 희망찬 분위기 금지
- 빛과 어둠 사이 어딘가
- **"신비로운 황혼"** 정도

### 감정 ↔ 쿨 (75/100)

- **강하게 쿨** — Kage Bow급 절제
- 감정을 설명하지 않음
- 슬픔도 분노도 직접 보여주지 않음
- **"슬프다"가 아니라 "멋있다"**

### 전통 ↔ 미래 (50/100)

- **정확히 균형**
- 전통만 = "국악" 느낌
- 미래만 = "사이버펑크" 느낌
- **둘이 섞여서 시대를 알 수 없게**

### 설명 ↔ 암시 (90/100)

- **거의 암시만**
- 직접 말하지 않음
- 팬이 해석하도록
- **설명하는 순간 결이 아님**

---

## Kage Bow와의 비교

| 요소 | Kage Bow | 결 |
|------|----------|-----|
| 중심 | 그림자의 형상 | 흐름의 방향 |
| 톤 | 쿨 / 스타일 | 쿨 / 스타일 |
| 감정 | 최소화 | 최소화 |
| 해석 | 팬에게 위임 | 팬에게 위임 |
| 문화권 | 일본 | 한국 |
| 상징 | 여우 가면, 우키요에 | 도깨비, 탈, 한지 |

> **결은 한국어로 만든 Kage Bow의 사촌격이다.**
> 하지만 복제가 아니라 **독자적 정체성**을 갖는다.

---

## 톤 체크 질문

콘텐츠 검증 시 자문:

1. **일본 요소가 없어도 성립하는가?**
   → 아니면 한국적 상징으로 대체

2. **"예쁘다"보다 "불안하다"가 먼저 느껴지는가?**
   → 아니면 더 어둡게

3. **'악마'가 아니라 '질서(시스템)'가 느껴지는가?**
   → 선악이 느껴지면 수정

4. **설명 없이도 의미가 전달되는가?**
   → 설명이 필요하면 암시로 전환
