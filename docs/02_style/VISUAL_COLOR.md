# VISUAL_COLOR — 색상 규칙

> **이 문서가 필요한 순간:**
> - 비주얼 제작 시 색상을 선택할 때
> - 팔레트가 "결다운지" 확인할 때
> - 금지 색상을 피해야 할 때

---

## 핵심 원칙

> **결의 색은 3가지면 충분하다.**
> **색을 늘리지 않고 반복으로 브랜드화한다.**

---

## 기본 팔레트

| 용도 | 색상 | Hex | RGB | 설명 |
|------|------|-----|-----|------|
| 주조 | 먹색 (Ink Black) | `#1a1a1a` | 26,26,26 | 완전한 검정 아님 |
| 보조 | 남색 (Indigo) | `#2c3e50` | 44,62,80 | 깊이감, 밤 |
| 배경 | 따뜻한 종이색 | `#f5f0e6` | 245,240,230 | 한지 느낌 |
| 중간톤 | 회갈색 | `#8b8178` | 139,129,120 | 전환부 |

---

## 악센트 색상

| 용도 | 색상 | Hex | RGB | 사용 조건 |
|------|------|-----|-----|----------|
| 강조 | 단청 빨강 | `#c23a3a` | 194,58,58 | 클라이맥스 20-30%에서만 |
| 신호 | 차가운 청록 | `#2d5a5a` | 45,90,90 | 경고/전환 순간 |

### 악센트 사용 규칙

**단청 빨강 (#c23a3a)**
- 전체 영상의 **20-30%** 에서만
- 등롱, 불꽃, 도장, 피
- 클라이맥스 구간에 집중
- "밝음"이 아닌 **"강렬함"**

**차가운 청록 (#2d5a5a)**
- 전환 직전에만
- 경고/신호
- 아주 짧게 (1-3초)
- 불안감 조성

## 금지 색상

| 금지 색상 | 이유 |
|----------|------|
| 순수 흰색 `#ffffff` ❌ | 너무 깨끗, 결의 질감 파괴 |
| 순수 검정 `#000000` ❌ | 너무 단순, 먹색 사용 |
| 채도 높은 네온 ❌ | 결의 톤 파괴 |
| 파스텔 톤 ❌ | 결의 톤 파괴 |
| 핑크/오렌지/노랑 ❌ | 분위기 파괴 |

---

## 색상 사용 비율

| 색상 유형 | 권장 비율 |
|----------|----------|
| 기본 팔레트 (먹/남/종이) | 70-80% |
| 중간톤 (회갈색) | 10-15% |
| 악센트 (빨강) | 10-20% |
| 신호색 (청록) | 1-5% |

---

## Phase별 색상 전략

| Phase | 주조 비율 | 악센트 비율 | 특징 |
|-------|----------|-----------|------|
| Phase A | 80% | 5-10% | 기본 팔레트, 빨강 최소 |
| Phase B | 75% | 15-20% | 빨강 악센트 증가, 긴장 |
| Phase C | 70% | 10-15% | 청록 신호색 등장, 기록 느낌 |
| Phase 0 | 85% | 5% | 가장 어둡고 절제, 종이색 최소 |

### Phase별 배경색 변화
| Phase | 배경 기본 | 보조색 |
|-------|----------|--------|
| Phase A | `#f5f0e6` (따뜻한 종이) | `#2c3e50` (남색) |
| Phase B | `#e8e0d4` (탁한 종이) | `#1e2b3a` (짙은 남색) |
| Phase C | `#d4cfc6` (회색 종이) | `#354a5f` (회색빛 남색) |
| Phase 0 | `#2a2520` (어두운 먹색) | `#0f1419` (거의 검정) |

## 후처리 가이드

### LUT 방향
- 따뜻한 그림자
- 차가운 하이라이트
- 낮은 채도
- 살짝 바랜 느낌

### 컬러 그레이딩
- 흰색 → 종이색으로 이동
- 검정 → 먹색으로 이동
- 채도 전체적으로 -20%
- 하이라이트에 살짝 청색

---

## 존재별 색상 연계

각 존재가 등장할 때 색상 변화:

| 존재 | 등장 시 색상 | 악센트 | 특징 |
|------|-----------|--------|------|
| 도깨비 | 단청 빨강 스파이크 | 5-10% | 등장 순간 악센트 급증 |
| 해태 | 남색 비중 증가 | 15→25% | 어둡고 무거워짐 |
| 탈 | 나무색 `#8b7355` | 유지 | 채도 변화 없음 |
| 인물 | 종이색 배경 | 감소 | 상대적으로 밝음 |

### 존재별 금지 색상
- **도깨비**: 밝은 녹색, 친근한 색 ❌
- **해태**: 빨강 악센트 ❌ (도깨비 영역)
- **탈**: 개별 탈마다 다른 색상 ❌
- **인물**: 채도 높은 의상색 ❌

---

## 색상 전환 가이드

장면 전환 시 색상 변화:

| 전환 유형 | 색상 변화 | 시간 |
|----------|----------|------|
| 평상 → 도깨비 등장 | 악센트 급증 | 0.5초 |
| 도깨비 퇴장 → 평상 | 악센트 급감 | 0.5초 |
| 평상 → 해태 발견 | 남색 증가 | 2-3초 점진적 |
| 인물 → 탈 전환 | 피부톤 → 나무색 | 1-2초 |
| Phase 전환 | 전체 팔레트 변화 | 페이드로 |

### 전환 금지
- 급격한 전체 팔레트 변화 ❌
- 무지개빛 그라데이션 ❌
- 플래시 전환 ❌

---

## 색상 조합 (컬러 하모니)

### 권장 조합

| 조합 유형 | 색상들 | 사용 |
|----------|--------|------|
| 유사색 | 먹색 + 남색 | 기본, 안정 |
| 보색 강조 | 남색 + 빨강 | 클라이맥스 |
| 단색조 | 먹색 변형들 | Phase 0 |
| 분리 보색 | 남색 + 빨강 + 청록 | Phase B 긴장 |

### 금지 조합
- 빨강 + 청록 동시 강조 ❌
- 종이색 + 빨강 직접 대비 ❌
- 3색 이상 동시 강조 ❌

---

## 시간대별 색상

| 시간대 | 주조 | 배경 | 악센트 | 사용 |
|--------|------|------|--------|------|
| 황혼 | 남색 80% | `#2c3e50` | 빨강 15% | 기본 |
| 심야 | 먹색 90% | `#1a1a1a` | 빨강 5% | 긴장 |
| 새벽전 | 남색 70% | `#354a5f` | 청록 10% | 전환 |
| 무시간 | 회색톤 | `#8b8178` | 없음 | Phase 0 |

### Phase별 시간대
- **Phase A**: 황혼 (애매한 시간)
- **Phase B**: 심야 → 새벽전 (전환)
- **Phase C**: 황혼 + 안개
- **Phase 0**: 무시간 (시간 불명)

## 색 온도 (Color Temperature)

### 구체적 수치

| 요소 | 색 온도 (K) | 느낌 |
|------|-----------|------|
| 등롱/횃불 | 2700-3000K | 따뜻함 |
| 달빛 | 5500-6500K | 차가움 |
| 모호한 빛 | 4000-4500K | 중립 |
| 전체 기본 | 4500K | 약간 따뜻 |

### 그림자 색 온도
- 그림자: 기본보다 +500K (차갑게)
- 하이라이트: 기본보다 -300K (따뜻하게)
- 결과: 따뜻한 그림자 느낌 (역전)

### Phase별 색 온도 기본값
| Phase | 기본 | 그림자 | 특징 |
|-------|------|--------|------|
| Phase A | 4500K | 5000K | 균형 |
| Phase B | 4000K | 4500K | 따뜻 (역설적 긴장) |
| Phase C | 5000K | 5500K | 차가움 |
| Phase 0 | 3500K | 4000K | 원시적 따뜻함 |
