# VISUAL_COLOR — 색상 규칙

> **이 문서가 필요한 순간:**
> - 비주얼 제작 시 색상을 선택할 때
> - 팔레트가 "결다운지" 확인할 때
> - 금지 색상을 피해야 할 때

---

## 핵심 원칙

> **결의 색은 3가지면 충분하다.**
> **색을 늘리지 않고 반복으로 브랜드화한다.**

---

## 기본 팔레트

| 용도 | 색상 | Hex | RGB | 설명 |
|------|------|-----|-----|------|
| 주조 | 먹색 (Ink Black) | `#1a1a1a` | 26,26,26 | 완전한 검정 아님 |
| 보조 | 남색 (Indigo) | `#2c3e50` | 44,62,80 | 깊이감, 밤 |
| 배경 | 따뜻한 종이색 | `#f5f0e6` | 245,240,230 | 한지 느낌 |
| 중간톤 | 회갈색 | `#8b8178` | 139,129,120 | 전환부 |

---

## 악센트 색상

| 용도 | 색상 | Hex | RGB | 사용 조건 |
|------|------|-----|-----|----------|
| 강조 | 단청 빨강 | `#c23a3a` | 194,58,58 | 클라이맥스 20-30%에서만 |
| 신호 | 차가운 청록 | `#2d5a5a` | 45,90,90 | 경고/전환 순간 |

### 악센트 사용 규칙

**단청 빨강 (#c23a3a)**
- 전체 영상의 **20-30%** 에서만
- 등롱, 불꽃, 도장, 피
- 클라이맥스 구간에 집중
- "밝음"이 아닌 **"강렬함"**

**차가운 청록 (#2d5a5a)**
- 전환 직전에만
- 경고/신호
- 아주 짧게 (1-3초)
- 불안감 조성

## 금지 색상

| 금지 색상 | 이유 |
|----------|------|
| 순수 흰색 `#ffffff` ❌ | 너무 깨끗, 결의 질감 파괴 |
| 순수 검정 `#000000` ❌ | 너무 단순, 먹색 사용 |
| 채도 높은 네온 ❌ | 결의 톤 파괴 |
| 파스텔 톤 ❌ | 결의 톤 파괴 |
| 핑크/오렌지/노랑 ❌ | 분위기 파괴 |

---

## 색상 사용 비율

| 색상 유형 | 권장 비율 |
|----------|----------|
| 기본 팔레트 (먹/남/종이) | 70-80% |
| 중간톤 (회갈색) | 10-15% |
| 악센트 (빨강) | 10-20% |
| 신호색 (청록) | 1-5% |

---

## Phase별 색상 전략

| Phase | 색상 특징 |
|-------|----------|
| Phase A | 기본 팔레트 위주, 악센트 최소 |
| Phase B | 빨강 악센트 증가 |
| Phase C | 청록 신호색 등장 |
| Phase 0 | 가장 어두움, 종이색 최소 |

## 후처리 가이드

### LUT 방향
- 따뜻한 그림자
- 차가운 하이라이트
- 낮은 채도
- 살짝 바랜 느낌

### 컬러 그레이딩
- 흰색 → 종이색으로 이동
- 검정 → 먹색으로 이동
- 채도 전체적으로 -20%
- 하이라이트에 살짝 청색

## 체크리스트

- [ ] 순수 흰색/검정 없는가?
- [ ] 네온/파스텔 없는가?
- [ ] 기본 팔레트가 70% 이상인가?
- [ ] 빨강 악센트가 30% 이하인가?
- [ ] 전체적으로 "결다운" 느낌인가?
