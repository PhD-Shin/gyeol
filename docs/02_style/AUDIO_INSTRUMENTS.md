# AUDIO_INSTRUMENTS — 악기 역할 정의

> **이 문서가 필요한 순간:**
> - 전통 악기를 어떻게 사용할지 결정할 때
> - 악기별 역할을 확인할 때
> - 금지 사용법을 확인할 때

---

## 핵심 원칙

> **K-판소리 판타지 트랩의 악기 철학**
> **전통 악기를 "전통처럼" 쓰면 실패한다.**
> **악기는 악기가 아니라 기능(Function)이다.**
> **몽환(판타지 팝) + 폭발(판소리 트랩) = 대비**

---

## 사용량 규칙

- **핵심 5악기**: 해금, 가야금, 장구, 태평소, 비트박스
- **현대 악기**: 신스 패드, 오케스트라, 808
- 전통 악기 = 전체 사운드의 **20-30%**
- 비트박스 = 전체 사운드의 **5-10%**
- **더 많이 쓰면 = "국악"이 된다**

---

## 0. 해금 (Haegeum) ★ 시그니처 악기

### 기능
> **결의 영혼 (Soul of 결)** — 시그니처 사운드

### 규칙

| 항목 | 규칙 |
|------|------|
| 역할 | **채널 시그니처**, 서사/긴장/감정 |
| 사용법 | 드론, 멜로딕 라인, 패드와 레이어 |
| 느낌 | 서사적, 몽환적, 긴장감 |
| 배치 | **Intro hit, Bridge drone, Outro fade** |

### 구간별 사용

| 구간 | 해금 역할 | 처리 |
|------|----------|------|
| **Intro** | 808과 함께 히트 | 짧고 강렬 |
| **Verse** | 배경 텍스처 | 패드와 레이어, 몽환적 |
| **Bridge** | 메인 드론 | 서사적, 신스 패드와 함께 |
| **Outro** | 페이드 아웃 | 여운, 다음 곡 연결 |

### 좋은 예
- 808 드롭과 동시에 해금 히트
- 신스 패드 위에 해금 멜로딕 드론
- Bridge에서 단독 서사적 라인

### 나쁜 예
- 전통 해금 산조 ❌
- 긴 솔로 연주 ❌
- 너무 전면에 배치 ❌

## 1. 가야금 (Gayageum)

### 기능
> **결의 선 (Line of 결)**

### 규칙

| 항목 | 규칙 |
|------|------|
| 멜로디 | ❌ 금지 |
| 사용법 | 반복 리프, 신스 패드처럼 깔기 |
| 느낌 | 차갑고 세련됨 |
| 배치 | Intro, Break, Outro |

### 좋은 예
- 반복되는 짧은 리프 (2-4노트)
- 앰비언트 패드처럼 깔림
- 다른 악기와 레이어

### 나쁜 예
- 전통 가야금 산조 ❌
- 멜로디 연주 ❌
- 솔로 연주 ❌

## 2. 장구 (Janggu) + 장단 (Jangdan)

### 기능
> **인간의 호흡 + 한국 리듬 DNA**

### 규칙

| 항목 | 규칙 |
|------|------|
| 전통 장단 | ⭕ **자진모리/중중모리 허용** (K-판소리 스타일) |
| 사용법 | 808 위에 레이어, Drop에서 장단 폭발 |
| 느낌 | 에너지, 한국적 그루브 |
| 배치 | **Verse: 중중모리 느낌 / Drop: 자진모리 폭발** |

### 장단 가이드

| 장단 | 특징 | 사용 구간 |
|------|------|----------|
| **자진모리** | 12/8 빠르게, 에너지 | **Drop** — 폭발 |
| **중중모리** | 중간 템포, 그루브 | **Verse** — 스토리텔링 |
| 굿거리 | 춤, 흥 | (선택적) |

### 좋은 예
- Drop에서 자진모리 장단 + 808 레이어
- Verse에서 중중모리 느낌의 그루브
- 비트와 약간 어긋나는 타이밍

### 나쁜 예
- 솔로 장구 (반주 없이) ❌
- 풍물/대취타 패턴 ❌
- 너무 전통적인 연주법 ❌

---

## 3. 태평소 / 피리 (Taepyeongso / Piri)

### 기능
> **경고 / 신호 (Warning / Signal)**

### 규칙

| 항목 | 규칙 |
|------|------|
| 긴 연주 | ❌ 금지 |
| 사용법 | 짧은 찢어지는 샘플 |
| 느낌 | 불안, 각성 |
| 배치 | 전환 직전, 클라이맥스 진입 |

### 좋은 예
- 1-2초 짧은 샘플
- 갑작스럽게 등장
- 전환 신호로만

### 나쁜 예
- 긴 태평소 연주 ❌
- 멜로디 연주 ❌
- 전통 대취타 ❌

## 4. 판타지 팝 요소 (Fantasy Pop Elements)

### 기능
> **접근성의 문 (Door of Accessibility)**

### 핵심 요소

| 요소 | 역할 | 사용 구간 |
|------|------|----------|
| **Dreamy Synth Pads** | 몽환적 분위기 | **Verse** |
| **Orchestral Swells** | 빌드업, 서사 | **Pre-Chorus** |
| **Ethereal Textures** | 공간감 | Verse, Bridge |
| **808 Sub Bass** | 펀치, 파워 | **Drop** |

### 구간별 배분

| 구간 | 전통 악기 | 판타지 팝 | 효과 |
|------|----------|----------|------|
| **Intro** | 80% (해금+808) | 20% | 정체성 선언 |
| **Verse** | 20% | **80%** (신스 패드) | 접근성 |
| **Pre-Chorus** | 30% | **70%** (오케스트라) | 기대감 |
| **Drop** | **70%** (장단+해금) | 30% | 폭발 |
| **Bridge** | 50% (해금) | 50% (패드) | 서사 |
| **Outro** | 60% | 40% | 여운 |

### 좋은 예
- Verse에서 몽환적 신스 패드 + 아니리 보컬
- Pre-Chorus에서 오케스트라 스웰 빌드업
- Drop 직전 모든 소리 빠지고 → 폭발

### 나쁜 예
- 전체가 몽환적이기만 함 ❌
- 대비 없이 균일한 에너지 ❌
- 오케스트라가 너무 클래식하게 ❌

## 5. 비트박스 (Beatbox) — Wing 스타일

### 기능
> **인간적 질감 (Human Texture)** — 전자음 속 유기적 요소

### 역할 정의

| 항목 | 규칙 |
|------|------|
| 역할 | **서브 레이어** (메인 비트 ❌) |
| 스타일 | **Wing 스타일** — 국악 소리 모방 |
| 사용량 | 전체 사운드의 **5-10%** |
| 목적 | 인간적 질감, 전통 요소 강화 |

### Wing 스타일 비트박스 요소

| 요소 | 설명 | 사용 |
|------|------|------|
| **꽹과리 소리** | 입으로 금속 타격음 | Drop 악센트 |
| **장구 채편** | 타악 비트 | 리듬 레이어 |
| **호흡 그루브** | 숨소리 리듬 | Verse 텍스처 |
| **추임새 비트** | "얼쑤" 리듬화 | 훅 강화 |

### 구간별 사용

| 구간 | 비트박스 역할 | 처리 |
|------|-------------|------|
| **Intro** | 808 킥과 동시에 꽹과리 소리 | 짧게 1회 |
| **Verse** | 배경 숨소리 그루브 | 거의 안 들리게 |
| **Pre-Chorus** | 빌드업 보조 | 점점 빨라지는 패턴 |
| **Drop** | 장구 채편 레이어 | 808과 함께 |
| **Bridge** | 무음 또는 솔로 | 선택적 |

### 좋은 예
- 808 드롭과 동시에 비트박스 꽹과리 한 번
- Verse에서 거의 들리지 않는 숨소리 그루브
- Drop에서 장구 소리를 비트박스로 보강

### 나쁜 예
- 비트박스가 메인 비트 ❌
- 서양 스타일 비트박스 ❌
- 과용 (10% 초과) ❌

### Suno 호환성

> ⚠️ **Suno로는 비트박스 직접 구현 불가**
> 별도 레이어로 믹싱하거나, "vocal percussion" 키워드 시도

```
vocal percussion texture,
mouth sounds subtle layer,
human breath groove
```

### 향후 계획

| Phase | 비트박스 사용 |
|-------|-------------|
| **Phase A** | 서브 레이어로 최소화 |
| **Phase B** | 점진적 증가 |
| **Phase C** | Wing 협업 검토 |

## 전통 악기 문헌적 배경 (제작진 레퍼런스)

> ⚠️ **주의**: 이 섹션은 제작진의 배경 이해용. 콘텐츠에 설명/인용 금지.

### 현악기 (絃樂器)

| 악기 | 문헌적 특성 | 음향적 특성 | 결에서의 역할 |
|------|-----------|-----------|-------------|
| 가야금 | 가야국 기원, 12현→25현 | 부드럽고 섬세한 음색 | 결의 선(Line) — 텍스처 |
| 거문고 | "백악지장(百樂之長)", 고구려 기원 | 깊고 무거운 저음, 3옥타브 | (제한적) 권위/경계 |
| 해금 | 서역 기원, 2현 | 날카롭고 명확한 고음, 비브라토 | 긴장/슬픔 레이어 |
| 아쟁 | 국악기 중 최저 음역 | 첼로와 유사, 깊은 울림 | 저음 드론 |

### 관악기 (管樂器)

| 악기 | 문헌적 특성 | 음향적 특성 | 결에서의 역할 |
|------|-----------|-----------|-------------|
| 대금 | 횡취악기, "청"의 떨림 | 깊고 그윽한 음, 청 진동 | 공간감/앰비언트 |
| 피리 | 서역계통, 5세기 고구려 도래 | 강렬하고 날카로움 | 경고/신호 (태평소와 함께) |
| 태평소 | 군악기 기원 | 가장 날카롭고 관통력 | 전환 신호 |

### 타악기 (打樂器)

| 악기 | 문헌적 특성 | 음향적 특성 | 결에서의 역할 |
|------|-----------|-----------|-------------|
| 장구 | 허리 잘록한 형태, 양면 | 채편(고음) + 북편(저음) | 인간의 호흡 |
| 꽹과리 | 풍물 상쇠 악기 | 금속성 고음, 날카로움 | 신호 (1-2회만) |
| 징 | 풍물 핵심, 원형 금속 | 깊은 공명, 지속 | 공간 확장 |
| 북 | 다양한 종류 | 저음 충격 | 단일 타격 |

### 존재별 악기 연계 (문헌적 근거)

| 존재 | 연계 악기 | 근거 |
|------|----------|------|
| 도깨비 | 장구, 꽹과리 | 노래와 춤을 좋아함 (전통 설화) |
| 해태 | 거문고, 아쟁 | 권위와 법의 상징 → 무겁고 낮은 악기 |
| 탈 | 태평소, 피리 | 탈춤 반주악기 |
| 인물 | 가야금, 해금 | 인간적 감정선 |

## 악기 음역대/튜닝

### 가야금
| 설정 | 값 |
|------|-----|
| 음역 | C3-C5 (중저음 위주) |
| 튜닝 | 표준 또는 약간 낮게 (A=432Hz 가능) |
| 리프 음수 | 2-4개 이하 |

### 장구
| 설정 | 값 |
|------|-----|
| 채편 (높은음) | 주로 사용 |
| 북편 (낮은음) | 강조점만 |
| 타이밍 | 비트 뒤 10-30ms (레이백) |

### 태평소/피리
| 설정 | 값 |
|------|-----|
| 음역 | G4-G5 (고음역) |
| 길이 | 1-2초 이하 |
| 피치 | 의도적 불안정 (미분음 허용) |

## 악기 샘플 처리

### 공통 이펙트 체인

```
원본 → EQ (저역 컷) → Compression → Reverb → Saturation
```

### 악기별 처리

| 악기 | 리버브 | 딜레이 | 새츄레이션 |
|------|--------|--------|----------|
| 가야금 | 크게 (2-3초) | 핑퐁 | 약하게 |
| 장구 | 적게 (0.5초) | 없음 | 중간 |
| 태평소 | 중간 (1초) | 없음 | 강하게 |

### EQ 가이드

| 악기 | 컷 주파수 | 부스트 |
|------|----------|--------|
| 가야금 | < 150Hz | 2-4kHz |
| 장구 | < 80Hz | 100-200Hz, 4kHz |
| 태평소 | < 300Hz | 1-2kHz |

## 믹싱 위치 (패닝)

### 스테레오 필드

```
L ←―――――――――――――――――――――――→ R
     장구           가야금
      (L30)   Center   (R30)
              태평소
              (C)
```

### 위치 규칙

| 악기 | 패닝 | 이유 |
|------|------|------|
| 가야금 | R 20-40% | 현대 악기와 분리 |
| 장구 | L 20-40% | 드럼과 구분 |
| 태평소 | Center | 신호, 중심 |

## 악기 진입/퇴장 패턴

### 진입 방식

| 악기 | 진입 방식 | 타이밍 |
|------|----------|--------|
| 가야금 | 페이드 인 (2-4초) | Intro 또는 Break |
| 장구 | 갑작스럽게 | Drop 진입 |
| 태평소 | 즉시 | 전환 직전 |

### 퇴장 방식

| 악기 | 퇴장 방식 | 타이밍 |
|------|----------|--------|
| 가야금 | 페이드 아웃 | Break 또는 Outro |
| 장구 | 갑작스럽게 멈춤 | Drop 종료 |
| 태평소 | 즉시 (잔향만) | 샘플 끝 |

### 존재와 악기 연결

| 존재 등장 시 | 강조 악기 | 방식 |
|------------|----------|------|
| 도깨비 | 장구/태평소 | 리듬 깨짐 |
| 해태 | 가야금 (정지) | 지속음 |
| 탈 | 장구 (반복) | 일정한 패턴 |
| 인물 | 모든 악기 균형 | 표준 믹스 |
