# VISUAL_TEXTURE — 질감 규칙

> **이 문서가 필요한 순간:**
> - 비주얼 제작 시 질감 처리할 때
> - 텍스처 오버레이를 적용할 때
> - "결다운 질감"을 확인할 때

---

## 핵심 원칙

> **디지털이지만 손으로 만든 것처럼.**
> **깨끗함은 결의 적이다.**

---

## 필수 질감

### 1. 한지 텍스처 (Hanji)
- **모든 배경의 기본 레이어**
- 불투명도: 10-30%
- Soft Light 또는 Overlay
- 미세한 섬유 결 느낌

### 2. 먹 번짐 (Ink Wash)
- 전환부, 페이드에 사용
- 경계를 모호하게
- 완전히 깨끗한 전환 금지

### 3. 필름 그레인 (Film Grain)
- 불투명도: **2-5%**
- 전체 영상에 적용
- 너무 강하면 안 됨 (노이즈 아님)

---

| 질감 | 사용 상황 | 설명 |
|------|----------|------|
| 천 (silk, linen) | 의상, 배경 | 부드러운 흐름 |
| 연기 | 전환, 분위기 | 경계 흐림 |
| 물결 | 반사, 움직임 | 변형 효과 |
| 먼지 입자 | 공간감 | 빛 줄기에 |
| 돌/나무 | 오브제, 배경 | 거친 표면 |
| 이끼 | 도깨비, 해태 | 시간의 흔적 |

### 존재별 고유 텍스처

| 존재 | 주 텍스처 | 보조 텍스처 |
|------|----------|-----------|
| 도깨비 | 돌, 이끼 | 연기, 먼지 |
| 해태 | 청동, 돌 | 안개 |
| 탈 | 나무결, 칠 | 먼지, 갈라짐 |
| 인물 | 천 (삼베, 모시) | 피부결 |

## 금지 질감 (일반 장면)

| 금지 | 이유 |
|------|------|
| 디지털 글리치 (과도한) ❌ | 일반 장면에선 금지 |
| 깨끗한 그라데이션 ❌ | 너무 디지털 |
| 플라스틱/메탈릭 광택 ❌ | 결의 질감 파괴 |
| 과도한 샤프니스 ❌ | 너무 선명 |
| CG 느낌 ❌ | 손 느낌 필요 |

## 사이버펑크/디스토피아 질감 (조건부)

> ⚠️ **디스토피아 장면에서만 사용**. 일반 장면에서는 아래 질감 금지.

### 허용되는 사이버펑크 질감

| 질감 | 사용 조건 | 관련 존재 |
|------|----------|----------|
| 녹슨 금속 | 장승, 작두, 기계 잔해 | 장승, 무당 |
| 회로 패턴 | 단청 변형, 건물 | 장승, 배경 |
| LED 발광 | 장승 눈, 네온 라인 | 장승 |
| 전선/케이블 | 건물, 장승 연결 | 장승, 배경 |
| 글리치 노이즈 | 시스템 오류 표현 | 불가사리 등장 |
| 용광로 빛 | 불가사리 입 안 | 불가사리 |

### 회로 = 단청 (핵심 변용)

> **단청 무늬를 PCB 회로처럼 재해석**

| 요소 | 전통 단청 | 사이버펑크 변용 |
|------|----------|---------------|
| 선 | 페인트 | 회로 트레이스 (발광 가능) |
| 패턴 | 기하학 반복 | 칩셋/회로도 느낌 |
| 색상 | 오방색 | 먹색 + 청록 발광 |
| 질감 | 나무 위 페인트 | 금속 위 에칭 |

### 녹슨 금속 질감 상세

| 상태 | 표면 | 색상 | 사용 |
|------|------|------|------|
| 경미 | 광택 약간 남음 | 갈색 얼룩 | 장승 (작동 중) |
| 중간 | 거칠어짐 | 녹슨 갈색 | 장승 (오래됨) |
| 심함 | 부식, 구멍 | 어두운 갈색 | 폐허 장승 |

### 글리치 질감 규칙

- 글리치는 **불가사리 등장 전후**에만
- 전체 화면 글리치 금지
- 부분적, 순간적 (0.5초 이하)
- RGB 분리, 스캔라인 정도만
- 과도한 데이터모싱 금지

### 존재별 사이버펑크 질감

| 존재 | 주 질감 | 보조 질감 |
|------|--------|----------|
| 장승 | 녹슨 금속, 나무 | LED 발광, 케이블 |
| 무당 | 반사 소재 (무복) | 회로 문양 (피부) |
| 불가사리 | 금속 파편 | 용광로 빛, 녹슨 액체 |
| 탈 (디스토피아) | 나무 + 금속 보강 | 갈라진 페인트 |

### Midjourney 사이버펑크 질감 키워드

## 단청 패턴 사용

### 원칙
> **단청은 전통 장식이 아니라 기계적/시스템적 구조로 재해석**

### 사용 방법

| 패턴 상태 | 의미 | 사용 |
|----------|------|------|
| 반복 패턴 | 시스템의 규칙성 | 배경, 질서 장면 |
| 깨진 패턴 | 결의 어긋남 | 개입 장면 |
| 흐릿한 패턴 | 배경 텍스처 | 전체 |
| 움직이는 패턴 | 기계처럼 맞물림 | 전환 |

### 금지
- 단청을 "예쁜 장식"으로 ❌
- 전통 건축물 그대로 ❌
- "한국 전통의 아름다움" 프레이밍 ❌

## 민화 레퍼런스 (제작진 배경)

> ⚠️ **주의**: 이 섹션은 제작진의 배경 이해용. 콘텐츠에 설명/인용 금지.

### 민화의 본질

> **민화는 "예쁜 그림"이 아니라 "서민의 풍자"**
> 결에서 민화는 시각적 문법의 레퍼런스, 내용 설명 금지

### 호작도 / 까치호랑이

| 요소 | 전통적 의미 | 결에서의 변용 |
|------|-----------|-------------|
| 호랑이 | 부패한 관리/양반 | 시스템, 권력 |
| 까치 | 서민, 약자 | 개입자, 어긋남 |
| 소나무 | 배경/장소 | 텍스처 요소 |
| 구성 | 호랑이를 비웃는 까치 | **사용 안 함** (설명적) |

> **결 적용**: 호랑이/까치의 시각적 문법만 참고.
> 풍자 구도 직접 인용 금지 (너무 명확함)

### 십장생도

| 십장생 | 결에서의 사용 |
|--------|-------------|
| 해(日) | 조명 레퍼런스 (직접 그리기 금지) |
| 구름 | ⭕ 연기/안개 텍스처 |
| 산 | ⭕ 배경 실루엣 |
| 물 | ⭕ 반사/물결 |
| 소나무 | ⭕ 배경 텍스처 |
| 거북 | ❌ 미사용 (너무 구체적) |
| 학 | ❌ 미사용 (너무 구체적) |
| 사슴 | ❌ 미사용 (너무 구체적) |
| 불로초 | ❌ 미사용 |
| 대나무 | △ 제한적 (배경) |

### 문자도

| 문자도 | 전통 의미 | 결 적용 |
|--------|----------|--------|
| 효제충신예의염치 | 유교 덕목 | 패턴으로만 (읽히지 않게) |

> **적용 방법**: 문자도 전체를 배경 텍스처로.
> 개별 글자가 읽히면 안 됨 (흐릿하게, 부분만)

### 민화 시각 문법 채택/배제

| 채택 | 배제 |
|------|------|
| 평면적 구성 | 원근법 |
| 윤곽선 강조 | 그림자 표현 |
| 반복 패턴 | 단일 주제 부각 |
| 비현실적 비례 | 해부학적 정확성 |
| 여백 활용 | 빽빽한 구성 |

### Midjourney 민화 키워드

```
minhwa style (not too literal),
korean folk painting aesthetics,
flat composition, bold outlines,
non-realistic proportions,
traditional korean visual grammar,
not cute, not decorative
```

## 텍스처 레이어 순서

```
1. 베이스 이미지
2. 한지 텍스처 (Soft Light, 15%)
3. 먹 번짐 (필요시)
4. 연기/입자 (필요시)
5. 필름 그레인 (Overlay, 3%)
6. LUT/컬러 그레이딩
```

## 후처리 가이드

### Photoshop/Premiere
1. 한지 텍스처 오버레이 (Soft Light)
2. Gaussian Blur 0.5px (샤프니스 줄이기)
3. Add Grain (2-5%)
4. Vignette (살짝)

### DaVinci Resolve
1. Texture node (한지)
2. Film Grain (Fine, 10-20%)
3. Glow (매우 약하게)
4. Lens Blur (가장자리)

## Phase별 질감 전략

| Phase | 한지 강도 | 먹 번짐 | 그레인 | 특징 |
|-------|----------|--------|--------|------|
| Phase A | 15% | 적음 | 3% | 깨끗한 편, 패턴 명확 |
| Phase B | 20% | 많음 | 4% | 경계 흐림, 번짐 증가 |
| Phase C | 25% | 중간 | 5% | 흔적, 삭음 |
| Phase 0 | 30% | 최소 | 2% | 태초, 원시적 |

### 존재 등장 시 질감 반응

| 존재 | 질감 변화 |
|------|----------|
| 도깨비 | 먹 번짐 30% 증가, 가장자리 어둠 확장 |
| 해태 | 질감 변화 없음 (정적) |
| 탈 | 단청 패턴 활성화 |
| 인물 | 그레인 미세 증가 |

## 텍스처 기술 스펙

### 해상도/사이즈

| 텍스처 | 최소 해상도 | 권장 해상도 |
|--------|-----------|-----------|
| 한지 | 2K (2048x2048) | 4K |
| 먹 번짐 | 1080p 이상 | 4K |
| 그레인 | 영상 해상도 동일 | - |

### 블렌드 모드 상세

| 텍스처 | 블렌드 모드 | 불투명도 | 대안 |
|--------|-----------|---------|------|
| 한지 | Soft Light | 10-20% | Overlay 5-15% |
| 먹 번짐 | Multiply | 30-60% | Darken |
| 그레인 | Overlay | 2-5% | Add (1-3%) |
| 연기 | Screen | 20-40% | Add |
| 먼지 | Add | 5-15% | Screen |

## 체크리스트

질감 완성 전:

- [ ] 한지 텍스처가 있는가?

- [ ] 너무 깨끗/선명하지 않은가?

- [ ] 플라스틱/메탈릭 광택이 없는가?
- [ ] 손으로 만든 느낌이 나는가?
