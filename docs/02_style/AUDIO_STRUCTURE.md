# AUDIO_STRUCTURE — 곡 구조

> **이 문서가 필요한 순간:**
> - 곡 구조를 설계할 때
> - 구간별 역할을 확인할 때
> - K-pop 구조를 적용할 때

---

## 핵심 원칙

> **K-pop은 "한국 음악"이 아니라
> 장르 혼합 + 강한 시각성 + 퍼포먼스 중심 구조다.**

---

## 🔥 초반 후킹 필수 규칙 (0-5초)

> **무명 채널의 생존 법칙:**
> 유튜브에서 시청자는 5초 안에 떠날지 결정한다.
> 첫 5초가 전부다.

### 후킹 타임라인

| 시간 | 필수 요소 | 설명 |
|------|----------|------|
| **0-2초** | **귀를 잡는 소리** | 예상 밖의 사운드, 가야금 꺾임, 글리치, 저음 펀치 |
| **2-5초** | **패턴 확립** | 리듬/멜로디 훅 시작, "이게 뭐지?" 유발 |
| **5-15초** | **세계관 진입** | 분위기 확립, 기존 Intro 역할 |

### 후킹 필수 기법 (최소 1개 선택)

| 기법 | 설명 | 예시 |
|------|------|------|
| **Sound Punch** | 첫 박에 강한 임팩트 | 808 킥 + 가야금 스트라이크 |
| **Unexpected Texture** | 예상 밖의 소리 | 한지 찢는 소리, 태평소 꺾임 |
| **Vocal Hook First** | 보컬 훅으로 시작 | 속삭임 또는 보컬 찹 |
| **Rhythm Lock** | 중독성 리듬 즉시 시작 | 장구 패턴 + 808 |
| **Drop Tease** | Drop 요소 미리 제시 | Drop의 핵심 사운드 1-2초 |

### 쇼츠/롱폼 대응

| 포맷 | 후킹 전략 |
|------|----------|
| **Shorts (60초)** | 0초부터 훅, Intro 생략, 바로 Drop 진입 가능 |
| **Long-form (3분+)** | 0-5초 후킹 후 15초까지 Intro 가능 |

### 금지

| 금지 | 이유 |
|------|------|
| 느린 앰비언트로 시작 ❌ | 5초 안에 이탈 |
| 페이드 인 ❌ | 임팩트 없음 |
| 설명적 인트로 ❌ | 유튜브는 기다려주지 않음 |

### Suno 프롬프트 추가 키워드

```
immediate hook, punchy intro, attention-grabbing first 5 seconds,
strong opening, instant impact, no fade in, front-loaded energy
```

---

## 기본 구조 템플릿

```
[Intro] - 전통 악기 솔로 (10-15초)
    ↓
[Verse 1] - 미니멀, 공간감
    ↓
[Pre-Chorus] - 빌드업
    ↓
[Drop 1] - 트랩/하이퍼팝, 장구 레이어
    ↓
[Break] - 무음에 가까운 공간 (5-10초)
    ↓
[Drop 2] - 보컬 찢기, 최고조
    ↓
[Outro] - 전통 악기만 남김
```

## 구간별 상세

### Intro (10-15초)
| 요소 | 내용 |
|------|------|
| 역할 | 세계관 진입 |
| 악기 | 가야금 솔로 또는 패드 |
| 분위기 | 고요, 긴장 |
| 보컬 | 없음 또는 속삭임 |

### Verse (30-45초)
| 요소 | 내용 |
|------|------|
| 역할 | 공간 확보, 서사 진행 |
| 악기 | 미니멀 비트, 베이스 |
| 분위기 | 절제, 공간감 |
| 보컬 | 있으면 속삭임/파편 |

### Pre-Chorus (15-20초)
| 요소 | 내용 |
|------|------|
| 역할 | 빌드업, 긴장 고조 |
| 악기 | 점점 레이어 추가 |
| 분위기 | 불안, 기대 |
| 보컬 | 반복 훅 |

### Drop (30-45초)
| 요소 | 내용 |
|------|------|
| 역할 | 클라이맥스, 해소 |
| 악기 | 808, 장구, 풀 비트 |
| 분위기 | 역동, 긴장 |
| 보컬 | 찢어지는 보컬 또는 없음 |

### Break (5-10초)
| 요소 | 내용 |
|------|------|
| 역할 | 숨, 전환 |
| 악기 | 거의 없음 |
| 분위기 | 고요, 숨쉬기 |
| 보컬 | 없거나 숨소리 |

### Outro (15-30초)
| 요소 | 내용 |
|------|------|
| 역할 | 여운, 퇴장 |
| 악기 | 가야금만 또는 앰비언트 |
| 분위기 | 고요, 열림 |
| 보컬 | 없거나 페이드 |

| 총 길이 | 권장 |
|--------|------|
| 2:00 | 쇼츠/티저용 |
| 2:30-3:00 | 표준 MV |
| 3:30-4:00 | 풀 버전 |

### 표준 3분 곡 예시
```
Intro: 0:00-0:15 (15초)
Verse 1: 0:15-0:45 (30초)
Pre-Chorus: 0:45-1:05 (20초)
Drop 1: 1:05-1:35 (30초)
Break: 1:35-1:45 (10초)
Verse 2: 1:45-2:00 (15초)
Drop 2: 2:00-2:40 (40초)
Outro: 2:40-3:00 (20초)
```

## Phase별 구조 변화

| Phase | 구조 특징 | Break 길이 | Drop 강도 |
|-------|----------|-----------|----------|
| Phase A | 표준 구조 | 10초 | 중간 |
| Phase B | Drop 2회 | 5초 | 강함 |
| Phase C | 느린 진행 | 15초 | 약함 |
| Phase 0 | 미니멀 구조 | 20초+ | 매우 약함 |

### Phase 0 특수 구조
```
[Ambient Intro] - 30초+
    ↓
[Minimal Texture] - 반복
    ↓
[Gentle Swell] - Drop 아닌 점진적 상승
    ↓
[Long Fade] - 30초+
```

## 존재 등장에 따른 구조 변화

존재가 등장하는 순간 구조적 변화:

| 존재 | 등장 구간 | 구조 변화 |
|------|----------|----------|
| 도깨비 | Drop 직전/직후 | 리듬 브레이크 삽입 |
| 해태 | Break 또는 Outro | 거의 무음으로 전환 |
| 탈 | Verse 또는 Pre-Chorus | 속삭임 레이어 추가 |
| 인물 | Verse | 숨소리/발자국 추가 |

### 존재 등장 타이밍 규칙
- **도깨비**: 리듬이 깨지는 순간에 등장
- **해태**: 고요한 순간에 발견됨
- **탈**: 반복 패턴 구간에 등장
- **인물**: 멜로디 있는 구간에 등장

## 테마별 구조 적용

각 테마가 강조되는 곡의 구조 조정:

| 테마 | 구조 특징 | Drop | Break |
|------|----------|------|-------|
| System vs Deviation | 정확한 반복 후 어긋남 | 퀀타이즈 변화 | 짧음 |
| Erasure | 점진적 감소 | 약함 | 길고 페이드 |
| Border | 모호한 전환 | 경계 없음 | 경계 없음 |
| Name | 불완전한 훅 | 보컬 중심 | 속삭임 |
| Record | 반복/에코 | 반복 모티프 | 에코 |
| Ritual | 최면적 루프 | 의례적 빌드업 | 의례적 고요 |

### 테마 조합 시 구조
- **2개 테마**: 하나는 Verse, 하나는 Drop에서 표현
- **3개 이상**: 금지 (과잉)

## 구간별 에너지 레벨

### 에너지 수치화 (0-100)

| 구간 | 에너지 | 긴장 | BPM 변화 |
|------|--------|------|----------|
| Intro | 10-20 | 30 | 기본 |
| Verse | 30-40 | 40 | 기본 |
| Pre-Chorus | 50-70 | 70 | +5-10 |
| Drop | 80-100 | 90 | 기본 또는 +10 |
| Break | 5-15 | 20 | 기본 또는 -10 |
| Outro | 20-30 | 30 | 기본 또는 -5 |

### 에너지 곡선 유형

| 유형 | 곡선 | 사용 |
|------|------|------|
| 표준 | 상승 → 정점 → 하강 | 대부분 |
| 더블 피크 | 상승 → 하강 → 상승 → 하강 | Phase B |
| 플랫 | 유지 → 미세 변화 | Phase 0 |
| 점진적 | 천천히 상승만 | Visualizer |

## 전환 (Transition) 기법

### 구간 간 전환

| 전환 | 기법 | 시간 |
|------|------|------|
| Intro → Verse | 드럼 진입 | 2-4박 |
| Verse → Pre-Chorus | 레이어 추가 | 4-8박 |
| Pre-Chorus → Drop | 라이저/필 | 1-2박 |
| Drop → Break | 즉시 컷 | 0박 (갑자기) |
| Break → Drop 2 | 빌드업 | 4-8박 |
| Drop → Outro | 페이드/필터 | 4-8박 |

### 전환 효과 라이브러리

| 효과 | 사용 | 길이 |
|------|------|------|
| 라이저 (Riser) | Pre-Chorus 끝 | 2-4박 |
| 리버스 심벌 | 섹션 진입 | 1-2박 |
| 필터 스윕 | Verse → Pre | 4-8박 |
| 스네어 롤 | Drop 직전 | 1-2박 |
| 무음 | Drop 직전 | 0.25-0.5박 |
| 태평소 샘플 | 전환 신호 | 0.5-1박 |

### Phase별 전환 강도

| Phase | 전환 강도 | 무음 사용 |
|-------|----------|----------|
| Phase A | 중간 | 적음 |
| Phase B | 강함 | 많음 |
| Phase C | 약함 | 거의 없음 |
| Phase 0 | 최소 | 없음 |

## 앨범 플로우 (곡 간 연결)

### 앨범 내 곡 순서 원칙

```
Track 1: 오프닝 - 세계관 진입 (에너지 50)
    ↓
Track 2-3: 전개 - 긴장 상승 (에너지 60-70)
    ↓
Track 4: 클라이맥스 - 최고조 (에너지 90)
    ↓
Track 5: 여운 - 감소 (에너지 40)
    ↓
Track 6: 마무리 - 열린 결말 (에너지 30)
```

### 곡 간 키 연결

| 연결 유형 | 설명 | 효과 |
|----------|------|------|
| 같은 키 | Am → Am | 연속성 |
| 관계조 | Am → Em | 자연스러운 전환 |
| 평행조 | Am → C | 밝아지는 느낌 |
| 반음 이동 | Am → Bbm | 긴장 |

### 앨범 플로우 예시 (6트랙)

| Track | 키 | BPM | 에너지 | 특징 |
|-------|-----|-----|--------|------|
| T1 | Am | 100 | 50 | 도입 |
| T2 | Em | 105 | 60 | 전개 |
| T3 | Dm | 110 | 70 | 상승 |
| T4 | Gm | 115 | 90 | 클라이맥스 |
| T5 | Dm | 95 | 50 | 하강 |
| T6 | Am | 90 | 35 | 마무리 |

### 곡 간 전환 (앨범용)

| 전환 | 방법 |
|------|------|
| 직접 연결 | 마지막 리버브가 다음 곡 진입 |
| 간격 연결 | 2-3초 무음 후 다음 곡 |
| 오버랩 | 아웃트로와 인트로 겹침 |
