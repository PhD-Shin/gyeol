# AUDIO_STRUCTURE — 곡 구조

> **이 문서가 필요한 순간:**
> - 곡 구조를 설계할 때
> - 구간별 역할을 확인할 때
> - K-pop 구조를 적용할 때

---

## 핵심 원칙

> **K-pop은 "한국 음악"이 아니라
> 장르 혼합 + 강한 시각성 + 퍼포먼스 중심 구조다.**

---

## 기본 구조 템플릿

```
[Intro] - 전통 악기 솔로 (10-15초)
    ↓
[Verse 1] - 미니멀, 공간감
    ↓
[Pre-Chorus] - 빌드업
    ↓
[Drop 1] - 트랩/하이퍼팝, 장구 레이어
    ↓
[Break] - 무음에 가까운 공간 (5-10초)
    ↓
[Drop 2] - 보컬 찢기, 최고조
    ↓
[Outro] - 전통 악기만 남김
```

---

## 구간별 상세

### Intro (10-15초)
| 요소 | 내용 |
|------|------|
| 역할 | 세계관 진입 |
| 악기 | 가야금 솔로 또는 패드 |
| 분위기 | 고요, 긴장 |
| 보컬 | 없음 또는 속삭임 |

### Verse (30-45초)
| 요소 | 내용 |
|------|------|
| 역할 | 공간 확보, 서사 진행 |
| 악기 | 미니멀 비트, 베이스 |
| 분위기 | 절제, 공간감 |
| 보컬 | 있으면 속삭임/파편 |

### Pre-Chorus (15-20초)
| 요소 | 내용 |
|------|------|
| 역할 | 빌드업, 긴장 고조 |
| 악기 | 점점 레이어 추가 |
| 분위기 | 불안, 기대 |
| 보컬 | 반복 훅 |

### Drop (30-45초)
| 요소 | 내용 |
|------|------|
| 역할 | 클라이맥스, 해소 |
| 악기 | 808, 장구, 풀 비트 |
| 분위기 | 역동, 긴장 |
| 보컬 | 찢어지는 보컬 또는 없음 |

### Break (5-10초)
| 요소 | 내용 |
|------|------|
| 역할 | 숨, 전환 |
| 악기 | 거의 없음 |
| 분위기 | 고요, 숨쉬기 |
| 보컬 | 없거나 숨소리 |

### Outro (15-30초)
| 요소 | 내용 |
|------|------|
| 역할 | 여운, 퇴장 |
| 악기 | 가야금만 또는 앰비언트 |
| 분위기 | 고요, 열림 |
| 보컬 | 없거나 페이드 |

---

## 이것은:

- **고유 K-pop** ⭕
- "국악 퓨전" ❌
- "전통 음악 현대화" ❌

## 시간 배분 가이드

| 총 길이 | 권장 |
|--------|------|
| 2:00 | 쇼츠/티저용 |
| 2:30-3:00 | 표준 MV |
| 3:30-4:00 | 풀 버전 |

### 표준 3분 곡 예시
```
Intro: 0:00-0:15 (15초)
Verse 1: 0:15-0:45 (30초)
Pre-Chorus: 0:45-1:05 (20초)
Drop 1: 1:05-1:35 (30초)
Break: 1:35-1:45 (10초)
Verse 2: 1:45-2:00 (15초)
Drop 2: 2:00-2:40 (40초)
Outro: 2:40-3:00 (20초)
```

## 영상 동기화

|------|-----------|
| Intro | 공간 샷, 느린 움직임 |
| Verse | 인물 등장, 롱 샷 |
| Pre-Chorus | 컷 빨라짐 |
| Drop | 빠른 컷, 클로즈업 |
| Break | 정적 샷, 여백 |
| Outro | 페이드, 여운 |
