# AUDIO_VOCAL — 보컬 규칙

> **이 문서가 필요한 순간:**
> - 보컬 스타일을 결정할 때
> - 가사 방향을 정할 때
> - 보컬 분량을 확인할 때

---

## 핵심 원칙

> **보컬은 캐릭터가 아니다.**
> **보컬은 내레이터가 아니다.**
> **보컬은 "세계 상태의 목소리"다.**

---

## 보컬의 정체성

- 특정 인물이 아님
- 이야기를 설명하지 않음
- **세계의 상태를 진동으로 전달**
- 감정을 직접 말하지 않음

---

## 보컬 스타일 비율

| 스타일 | 비율 | 설명 |
|--------|------|------|
| 속삭임 | 30% | 공간감, 친밀감 |
| 샘플/파편 | 25% | 단어 조각, 반복 |
| 클린 보컬 | 25% | 후렴, 훅 |
| 찢어지는 보컬 | 15% | 클라이맥스 |
| 코러스/레이어 | 5% | 배경 |

---

## 스타일별 상세

### 속삭임 (Whisper)
- ASMR에 가까운 친밀감
- 공간에 스며드는 느낌
- Intro, Break에서 효과적

### 샘플/파편 (Fragment)
- 단어 조각 반복
- 의미보다 리듬
- 신스처럼 사용

### 클린 보컬 (Clean)
- 명확한 발음
- 훅에서 사용
- 감정 과잉 금지

### 찢어지는 보컬 (Distorted)
- Drop 클라이맥스
- 가공된 목소리
- 인간/기계 경계

---

## 가사 원칙

### 우선순위
1. **발음과 호흡**이 먼저
2. **리듬**이 먼저
3. 의미는 그 다음

### 규칙
- 명확한 후렴 ❌
- 반복되는 훅 ⭕
- 설명적 가사 ❌
- 암시적 가사 ⭕

### 좋은 예
```
흔들린다 (반복)
지워진다 (반복)
```

### 나쁜 예
```
나는 슬퍼서 울고 있어
세상이 무너지는 느낌이야
```

---

## 가사 분량

- 전체 곡에서 가사 있는 부분: **40-50%**
- 나머지: 인스트루멘탈 또는 보컬 샘플
- 가사 없는 공간 = 여백

---

## 언어 전략

### 기본 원칙
- 영어 기본 (글로벌)
- 한국어는 **텍스처**로 사용

### 한국어 사용법
- 이해시키는 가사 ❌
- 주문/상징으로 사용 ⭕
- 짧은 단어/구절만
- 자막으로 해설 ❌

---

## 구간별 보컬 배치

| 구간 | 보컬 유형 | 분량 |
|------|----------|------|
| Intro | 없음 / 속삭임 | 0-10% |
| Verse | 속삭임 / 파편 | 30-50% |
| Pre-Chorus | 클린 / 반복 | 60-80% |
| Drop | 찢어짐 / 없음 | 0-30% |
| Break | 없음 / 숨소리 | 0% |
| Outro | 속삭임 / 없음 | 0-20% |

---

## Suno 프롬프트 보컬 키워드

```
minimal vocal, whispered vocals,
vocal chops, processed voice,
not narrative, not emotional,
voice as texture, layered vocals,
korean words as texture not lyrics
```
