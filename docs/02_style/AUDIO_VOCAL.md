# AUDIO_VOCAL — 보컬 규칙

> **이 문서가 필요한 순간:**
> - 보컬 스타일을 결정할 때
> - 가사 방향을 정할 때
> - 보컬 분량을 확인할 때

---

## 핵심 원칙

> **보컬은 캐릭터가 아니다.**
> **보컬은 내레이터가 아니다.**
> **보컬은 "세계 상태의 목소리"다.**

---

## 보컬의 정체성

- 특정 인물이 아님
- 이야기를 설명하지 않음
- **세계의 상태를 진동으로 전달**
- 감정을 직접 말하지 않음

---

## 보컬 스타일 비율

| 스타일 | 비율 | 설명 |
|--------|------|------|
| 속삭임 | 30% | 공간감, 친밀감 |
| 샘플/파편 | 25% | 단어 조각, 반복 |
| 클린 보컬 | 25% | 후렴, 훅 |
| 찢어지는 보컬 | 15% | 클라이맥스 |
| 코러스/레이어 | 5% | 배경 |

---

## 스타일별 상세

### 속삭임 (Whisper)
- ASMR에 가까운 친밀감
- 공간에 스며드는 느낌
- Intro, Break에서 효과적

### 샘플/파편 (Fragment)
- 단어 조각 반복
- 의미보다 리듬
- 신스처럼 사용

### 클린 보컬 (Clean)
- 명확한 발음
- 훅에서 사용
- 감정 과잉 금지

### 찢어지는 보컬 (Distorted)
- Drop 클라이맥스
- 가공된 목소리
- 인간/기계 경계

---

## 가사 원칙

### 우선순위
1. **발음과 호흡**이 먼저
2. **리듬**이 먼저
3. 의미는 그 다음

### 규칙
- 명확한 후렴 ❌
- 반복되는 훅 ⭕
- 설명적 가사 ❌
- 암시적 가사 ⭕

### 좋은 예
```
흔들린다 (반복)
지워진다 (반복)
```

### 나쁜 예
```
나는 슬퍼서 울고 있어
세상이 무너지는 느낌이야
```

---

## 가사 분량

- 전체 곡에서 가사 있는 부분: **40-50%**
- 나머지: 인스트루멘탈 또는 보컬 샘플
- 가사 없는 공간 = 여백

---

## 언어 전략

### 기본 원칙
- 영어 기본 (글로벌)
- 한국어는 **텍스처**로 사용

### 한국어 사용법
- 이해시키는 가사 ❌
- 주문/상징으로 사용 ⭕
- 짧은 단어/구절만
- 자막으로 해설 ❌

---

## 구간별 보컬 배치

| 구간 | 보컬 유형 | 분량 |
|------|----------|------|
| Intro | 없음 / 속삭임 | 0-10% |
| Verse | 속삭임 / 파편 | 30-50% |
| Pre-Chorus | 클린 / 반복 | 60-80% |
| Drop | 찢어짐 / 없음 | 0-30% |
| Break | 없음 / 숨소리 | 0% |
| Outro | 속삭임 / 없음 | 0-20% |

---

## Phase별 보컬 전략

| Phase | 보컬 분량 | 주 스타일 | 특징 |
|-------|----------|----------|------|
| Phase A | 45% | 속삭임 + 클린 | 균형 잡힌 |
| Phase B | 50% | 찢어짐 증가 | 긴장, 가공 |
| Phase C | 35% | 속삭임 + 파편 | 절제, 흔적 |
| Phase 0 | 20% | 속삭임만 | 거의 인스트 |

### Phase별 한국어 사용
| Phase | 한국어 비율 | 사용법 |
|-------|-----------|--------|
| Phase A | 10% | 텍스처, 짧은 단어 |
| Phase B | 15% | 훅, 반복구 |
| Phase C | 20% | 상징적 구절 |
| Phase 0 | 25% | 주문, 고어 느낌 |

## Suno 프롬프트 보컬 키워드

```
minimal vocal, whispered vocals,
vocal chops, processed voice,
not narrative, not emotional,
voice as texture, layered vocals,
korean words as texture not lyrics
```

---

## 보컬 이펙트 체인

### 기본 체인

```
마이크/원본 → EQ → Compression → De-esser → Reverb → Delay → Saturation
```

### 스타일별 이펙트

| 스타일 | 리버브 | 딜레이 | 피치 이펙트 |
|--------|--------|--------|-----------|
| 속삭임 | 크게 (3-4초) | 핑퐁 | 약간 디튠 |
| 샘플/파편 | 중간 (2초) | 짧은 슬랩 | 피치 시프트 |
| 클린 | 적게 (1초) | 없음/적음 | 없음 |
| 찢어짐 | 크게 (3초) | 강한 에코 | 옥타브 레이어 |

### 특수 이펙트

| 이펙트 | 사용 상황 | 설정 |
|--------|----------|------|
| 피치 시프트 | 존재 암시 | -12 또는 +12 반음 |
| 보코더 | 기계/시스템 | 캐리어: 신스 패드 |
| 그래뉼러 | 시간 왜곡 | 짧은 그레인 (20-50ms) |
| 리버스 | 전환/암시 | Pre-verb |

## 보컬 레이어링 규칙

### 레이어 구조

| 레이어 | 역할 | 패닝 | 볼륨 |
|--------|------|------|------|
| 메인 | 핵심 보컬 | Center | 0dB |
| 더블 | 두께 | L10/R10 | -6dB |
| 옥타브 | 공간감 | L30/R30 | -12dB |
| 위스퍼 | 친밀감 | Center | -9dB |

### Phase별 레이어

| Phase | 레이어 수 | 특징 |
|-------|----------|------|
| Phase A | 2-3 | 메인 + 더블 |
| Phase B | 3-4 | 모든 레이어 가능 |
| Phase C | 1-2 | 메인만 또는 위스퍼 |
| Phase 0 | 1 | 단일 레이어 |

## 음역대/키 가이드

### 권장 음역

| 보컬 타입 | 음역 | 이유 |
|----------|------|------|
| 여성 메인 | A3-E5 | 중간 음역 |
| 남성 메인 | E2-B4 | 낮은 음역 |
| 속삭임 | 음역 무관 | 피치 모호 |
| 찢어짐 | 고음역 | 텐션 |

### 키 권장

| 키 | 느낌 | Phase 적합 |
|-----|------|-----------|
| D minor | 어둡고 차가움 | A, B |
| A minor | 쿨하고 모호 | A, C |
| E minor | 열린 느낌 | 0 |
| G minor | 무거움 | B |
