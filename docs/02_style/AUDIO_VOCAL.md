# AUDIO_VOCAL — 보컬 규칙

> **이 문서가 필요한 순간:**
> - 보컬 스타일을 결정할 때
> - 가사 방향을 정할 때
> - 보컬 분량을 확인할 때

---

## 핵심 원칙

> **K-판소리 판타지 트랩의 보컬 철학**
> **창(唱)의 거침 + 아니리의 스토리텔링 + 추임새의 훅**
> **보컬은 "세계 상태의 목소리"다.**

---

## K-판소리 보컬 스타일

### 핵심 3요소

| 요소 | 설명 | 사용 구간 |
|------|------|----------|
| **창 (Chang)** | 거칠고 폭발적인 발성, 피치벤딩 | **Drop** |
| **아니리 (Aniri)** | 말하듯 스토리텔링 랩 | **Verse** |
| **추임새 (Chuimsae)** | "얼쑤", "좋다", "그렇지" 훅 | **Hook points** |

### 구간별 보컬 스타일

| 구간 | 보컬 스타일 | 특징 |
|------|-----------|------|
| **Intro** | 창 샤우트 + 추임새 | 폭발적, 훅 |
| **Verse** | 아니리 (몽환적) | 말하듯, Dreamy |
| **Pre-Chorus** | 빌딩 + 추임새 | 기대감 |
| **Drop** | 창 폭발 + 추임새 | 최대 에너지 |
| **Bridge** | 속삭임 / 공간 | 서사적 |
| **Outro** | 속삭임 페이드 | 여운 |

---

## 보컬의 정체성

- 특정 인물이 아님
- 이야기를 설명하지 않음
- **세계의 상태를 진동으로 전달**
- 감정을 직접 말하지 않음
- **창의 한(恨)을 현대적으로 표현**

---

## 보컬 스타일 비율 (K-판소리 판타지 트랩)

| 스타일 | 비율 | 사용 |
|--------|------|------|
| **창 (Chang)** | 30% | Drop, Intro 샤우트 |
| **아니리 (Aniri)** | 30% | Verse 스토리텔링 |
| 속삭임/몽환 | 20% | Verse 배경, Outro |
| **추임새 (Chuimsae)** | 10% | 훅 포인트 |
| 코러스/레이어 | 10% | 배경, 두께 |

---

## 스타일별 상세

### 속삭임 (Whisper)
- ASMR에 가까운 친밀감
- 공간에 스며드는 느낌
- Intro, Break에서 효과적

### 샘플/파편 (Fragment)
- 단어 조각 반복
- 의미보다 리듬
- 신스처럼 사용

### 클린 보컬 (Clean)
- 명확한 발음
- 훅에서 사용
- 감정 과잉 금지

### 찢어지는 보컬 (Distorted)
- Drop 클라이맥스
- 가공된 목소리
- 인간/기계 경계

---

## 추임새 (Chuimsae) 가이드

> **추임새 = K-판소리 훅**
> **현대 판소리의 재정립** — 전통 + 현대 + 하이브리드
> 짧고 강렬하게, 비트에 맞춰

### 1. 전통 추임새

| 추임새 | 발음 | 느낌 | 사용 |
|--------|------|------|------|
| **얼쑤** | eol-ssu | 폭발, 흥 | Drop 후, Intro |
| **좋다** | joh-da | 만족, 확인 | Pre-Chorus, Drop |
| **그렇지** | geu-reo-ji | 동의, 강조 | Drop 중간 |
| **아이고** | a-i-go | 한탄, 깊이 | Bridge |
| **으이** | eu-i | 긴장, 시작 | Intro |
| **옳지** | ol-ji | 확인 | Drop 후 |
| **어화** | eo-hwa | 흥 | Drop |

### 2. 현대 추임새 (K-Pansori Trap)

> **현대 판소리의 재정립** — 전통 추임새의 에너지를 현대적으로

| 추임새 | 발음 | 느낌 | 사용 |
|--------|------|------|------|
| **가자** | ga-ja | 출발, 전진 | Drop 끝, 전환점 |
| **간다** | gan-da | 돌진, 에너지 | Pre-Chorus, 빌드업 |
| **봐라** | bwa-ra | 주목 | Pre-Chorus |
| **해봐** | hae-bwa | 도전 | Build-up |
| **와** | wa | 감탄 | Drop 후 |
| **야** | ya | 호출, 시작 | Intro |

### 3. 하이브리드 추임새 (English-Korean)

> **코드 스위칭의 쾌감** — 언어 전환으로 임팩트 극대화

| 하이브리드 | 사용 | 효과 |
|-----------|------|------|
| **Let's go 가자** | Drop 끝 | 전진 + 글로벌 |
| **One more 한번** | 반복 전 | 기대감 |
| **Yeah 좋다** | Drop 후 | 만족 + 트랩 느낌 |
| **Ay 얼쑤** | Drop | 폭발 + 글로벌 |
| **Eolssu** (로마자) | Intro, Drop | 글로벌 청자용 |

### 추임새 배치 규칙

| 규칙 | 설명 |
|------|------|
| **비트에 맞춤** | 추임새는 반드시 비트/리듬에 착 |
| **혼합 사용** | 전통 + 현대 + 하이브리드 섞어서 |
| **트랙당 5-8회** | 과용 금지, 임팩트 유지 |
| **훅 포인트에만** | Verse에는 ❌ |

### 구간별 추임새 배치

| 구간 | 권장 추임새 | 스타일 |
|------|-----------|--------|
| **Intro** | 얼쑤, Eolssu, 야 | 폭발적 시작 |
| **Verse** | ❌ (사용 금지) | — |
| **Pre-Chorus** | 좋다, 간다, 봐라 | 빌드업 |
| **Drop** | 얼쑤, 그렇지, 가자, Yeah 좋다 | 최대 에너지 |
| **Bridge** | 아이고 (선택) | 서사적 |
| **Outro** | ❌ 또는 속삭임 | 여운 |

### 좋은 예 (v3 — K-pop 코드스위칭)

> **핵심**: 영어와 한국어가 **같은 의미 반복 ❌** → **연결/확장 ⭕**

```
[Drop]
Let it burn 재만 남아      ← 원인→결과 (burn→ashes)
얼쑤!
어둠 속에 ashes fall       ← 장소+결과
그렇지— 가자
```

### ❌ 피해야 할 패턴 (같은 의미 반복)
```
Can't stop 멈출 수 없어    ← 같은 의미!
Burn it all 다 태워버려    ← 같은 의미!
```

### 나쁜 예
```
어둠이 온다 얼쑤 (❌ Verse에 억지로)
눈을 감아 좋다 (❌ 문맥 안 맞음)
Let's go만 (❌ 한국어 없이)
```

## 가사 원칙

### 우선순위
1. **발음과 호흡**이 먼저
2. **리듬**이 먼저
3. 의미는 그 다음

### 규칙
- 명확한 후렴 ❌
- 반복되는 훅 ⭕
- 설명적 가사 ❌
- 암시적 가사 ⭕

### 좋은 예
```
흔들린다 (반복)
지워진다 (반복)
```

### 나쁜 예
```
나는 슬퍼서 울고 있어
세상이 무너지는 느낌이야
```

---

## 가사 분량

- 전체 곡에서 가사 있는 부분: **40-50%**
- 나머지: 인스트루멘탈 또는 보컬 샘플
- 가사 없는 공간 = 여백

---

## 언어 전략: 한영 혼용 (Konglish Mix)

> **글로벌 접근성 + 한국적 정체성**
> 영어로 잡고, 한국어로 각인시킨다.

### 비율 가이드

| 구간 | 영어 | 한국어 | 설명 |
|------|------|--------|------|
| **Hook/훅** | 60% | 40% | 영어 훅 + 한국어 펀치라인 |
| **Verse** | 70% | 30% | 영어 기반 + 한국어 악센트 |
| **Pre-Chorus** | 50% | 50% | 균형 혼용 |
| **Drop** | 30% | 70% | 한국어 파편이 더 효과적 |

### 혼용 패턴

| 패턴 | 예시 | 효과 |
|------|------|------|
| **English + Korean Word** | "Falling into 어둠" | 대비 |
| **Korean Hook + English Verse** | Verse: "I can't escape" → Hook: "흔들린다" | 대비 |
| **Code-Switching** | "Don't look back, 지워진다" | 리듬 변화 |
| **Repeated Korean** | "멈출 수 없어" (반복) | 최면적 훅 |

### 구간별 예시

**Hook (훅):**
```
Falling falling (English)
흔들린다 (Korean - 펀치라인)
```

**Verse:**
```
I tried to hold on (English)
But it slips away (English)
남김 (Korean accent)
```

**Drop:**
```
멈출 수 없어 (Korean dominant)
Let it break (English break)
```

### 한국어 활용법

| 사용법 | 설명 | 예시 |
|--------|------|------|
| **텍스처** | 의미보다 소리 | "흐름, 어둠, 지움" |
| **펀치라인** | 핵심 한 마디 | "흔들린다" (hook ending) |
| **반복 훅** | 단어 반복 | "멈출 수 없어" |
| **분위기** | 상황/감각 전달 | "눈을 떠", "숨이 막혀" |

> ⚠️ **금지**: 트랙 제목, 프로젝트명("결")은 가사에 사용 금지

### 금지 사항

| 금지 | 이유 |
|------|------|
| 한국어로 긴 문장 ❌ | 글로벌 접근성 저하 |
| 영어 문법에 한국어 억지 삽입 ❌ | 부자연스러움 |
| 한국어 설명 가사 ❌ | "설명 금지" 원칙 |
| 영어만 100% ❌ | 정체성 상실 |

### Suno 프롬프트 키워드

```
bilingual lyrics, korean-english mix, konglish,
korean words as hooks, english verses korean chorus,
code-switching vocals, korean texture words
```

| 구간 | 보컬 유형 | 분량 |
|------|----------|------|

| Verse | 속삭임 / 파편 | 30-50% |

| Drop | 찢어짐 / 없음 | 0-30% |
| Break | 없음 / 숨소리 | 0% |
| Outro | 속삭임 / 없음 | 0-20% |

## Phase별 보컬 전략

| Phase | 보컬 분량 | 주 스타일 | 특징 |
|-------|----------|----------|------|
| Phase A | 45% | 속삭임 + 클린 | 균형 잡힌 |
| Phase B | 50% | 찢어짐 증가 | 긴장, 가공 |
| Phase C | 35% | 속삭임 + 파편 | 절제, 흔적 |
| Phase 0 | 20% | 속삭임만 | 거의 인스트 |

### Phase별 한국어 사용
| Phase | 한국어 비율 | 사용법 |
|-------|-----------|--------|
| Phase A | 10% | 텍스처, 짧은 단어 |
| Phase B | 15% | 훅, 반복구 |
| Phase C | 20% | 상징적 구절 |
| Phase 0 | 25% | 주문, 고어 느낌 |


## 보컬 이펙트 체인

### 기본 체인

```
마이크/원본 → EQ → Compression → De-esser → Reverb → Delay → Saturation
```

### 스타일별 이펙트

| 스타일 | 리버브 | 딜레이 | 피치 이펙트 |
|--------|--------|--------|-----------|
| 속삭임 | 크게 (3-4초) | 핑퐁 | 약간 디튠 |
| 샘플/파편 | 중간 (2초) | 짧은 슬랩 | 피치 시프트 |
| 클린 | 적게 (1초) | 없음/적음 | 없음 |
| 찢어짐 | 크게 (3초) | 강한 에코 | 옥타브 레이어 |

### 특수 이펙트

| 이펙트 | 사용 상황 | 설정 |
|--------|----------|------|
| 피치 시프트 | 존재 암시 | -12 또는 +12 반음 |
| 보코더 | 기계/시스템 | 캐리어: 신스 패드 |
| 그래뉼러 | 시간 왜곡 | 짧은 그레인 (20-50ms) |
| 리버스 | 전환/암시 | Pre-verb |

| 레이어 | 역할 | 패닝 | 볼륨 |
|--------|------|------|------|

| Phase | 레이어 수 | 특징 |
|-------|----------|------|
| Phase A | 2-3 | 메인 + 더블 |
| Phase B | 3-4 | 모든 레이어 가능 |
| Phase C | 1-2 | 메인만 또는 위스퍼 |
| Phase 0 | 1 | 단일 레이어 |
