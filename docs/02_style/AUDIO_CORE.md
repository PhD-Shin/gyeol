# AUDIO_CORE — 오디오 핵심

> **이 문서가 필요한 순간:**
> - 음악 제작의 기본 방향을 확인할 때
> - 음악의 역할을 이해해야 할 때
> - 결의 음악 톤을 검증할 때

---

## 핵심 원칙

> **음악은 감정을 설명하지 않는다.**
> **음악은 움직임을 흔드는 진동이다.**

---

## 음악의 역할

- 감정을 설명하지 않는다
- 감정을 몰고 가는 주체가 아니다
- **움직임을 흔드는 진동**이다

---

## 핵심 공식

```
전통 소리 = 재료 (Material)
K-pop = 형식 (Form)
AI 시대 = 상태 (State, shown not said)
```

---

## 피해야 할 것

| 금지 | 이유 |
|------|------|
| 전통 국악 멜로디 패턴 ❌ | "국악" 느낌 금지 |
| 민속 분위기 ❌ | 전통 소개 금지 |
| 판소리식 감정 과잉 ❌ | 쿨하게 |
| 악기 "소개"처럼 ❌ | 악기는 텍스처 |
| 밝은 느낌 ❌ | 밝더라도 기이해야 |
| 감정 직접 설명 ❌ | 결은 설명 안 함 |

---

## 성공 공식

> **전통 악기 = 텍스처**
> **K-pop = 구조**
> **AI 시대 = 리듬과 반복**

---

## AI 시대 간접 표현

### 표현 방식

| 요소 | AI/시스템 | 인간 |
|------|----------|------|
| 비트 | 완벽한 반복 | 미세한 어긋남 |
| 퀀타이즈 | 100% 정확 | 90-95% |
| 악기 | 신스, 드럼머신 | 장구, 숨소리 |

### 암묵적 메시지
> AI는 완벽하지만,
> 우리는 결을 만든다.

**직접 말하지 않는다. 리듬으로 보여준다.**

---

## BPM & 키 가이드

### BPM 범위

| 분위기 | BPM | 사용 |
|--------|-----|------|
| 긴장 | 85-95 | Phase A 초반 |
| 기본 | 95-110 | 대부분 |
| 고조 | 110-125 | 클라이맥스 |

### 키 가이드
- **마이너 키** 기본 (70%)
- 메이저 키 = "밝음" 아닌 **"기이함"**
- 전조: 한 곡에 최대 1회

### 일관성
- 같은 Phase 내 트랙들 = 연관된 키
- BPM 변화 = ±10 범위

---

## Phase별 음악 전략

| Phase | BPM 범위 | 주요 키 | 분위기 |
|-------|---------|--------|--------|
| Phase A | 95-110 | Am, Em | 균형, 도입 |
| Phase B | 100-120 | Dm, Gm | 긴장, 고조 |
| Phase C | 85-100 | Cm, Fm | 무거움, 기록 |
| Phase 0 | 75-90 | Am, Bm | 고요, 기원 |

### Phase별 악기 비중
| Phase | 전통 악기 | 일렉트로닉 | 앰비언트 |
|-------|----------|-----------|---------|
| Phase A | 20% | 50% | 30% |
| Phase B | 25% | 55% | 20% |
| Phase C | 30% | 40% | 30% |
| Phase 0 | 35% | 25% | 40% |

---

## 톤 체크 질문

1. 전통 악기가 "소개"처럼 들리는가? → 수정
2. 감정이 직접 설명되는가? → 수정
3. 밝은 느낌인가? → 기이함으로 전환
4. 완벽한 반복 + 인간적 어긋남이 공존하는가? → 필수

---

## 존재별 사운드 시그니처

각 존재가 등장할 때 음악 특징:

| 존재 | 시그니처 사운드 | BPM 변화 | 특징 |
|------|---------------|---------|------|
| 도깨비 | 리듬 브레이크, 불협화음 | ±5-10 어긋남 | 퀀타이즈 깨짐 |
| 해태 | 저음 드론, 공명 | 유지 | 다른 소리 최소화 |
| 탈 | 속삭임 레이어, 무음 | 유지 | 여러 목소리 겹침 |
| 인물 | 숨소리, 발자국 | 미세 변화 | 인간적 요소 |

### 존재별 금지 사운드
- **도깨비**: 밝은 멜로디 ❌, 안정적 리듬 ❌
- **해태**: 빠른 리듬 ❌, 높은 음역 ❌
- **탈**: 개별 목소리 ❌, 대사 ❌
- **인물**: 내레이션 ❌, 독백 ❌

---

## 테마별 음악 표현

각 테마의 음악적 표현:

| 테마 | 리듬 | 멜로디 | 텍스처 |
|------|------|--------|--------|
| System vs Deviation | 정확 → 어긋남 | 반복 후 변주 | 기계 + 유기 |
| Erasure | 페이드, 감소 | 사라지는 멜로디 | 리버브 증가 |
| Border | 모호한 전환 | 경계 없는 진행 | 블렌딩 |
| Name | 속삭임, 파편 | 불완전한 훅 | 보컬 레이어 |
| Record | 에코, 반복 | 과거 모티프 | 노이즈, 빈티지 |
| Ritual | 반복 패턴 | 최면적 루프 | 장구, 의례적 |

## 믹싱/마스터링 기준

### 믹싱 목표

| 요소 | 목표 | 설정 |
|------|------|------|
| 스테레오 폭 | 넓음 | 리버브 + 패닝 |
| 깊이감 | 레이어드 | 전경/중경/후경 분리 |
| 선명도 | 약간 흐림 | 전체 롤오프 |
| 공간감 | 넓은 공간 | 긴 리버브 |

### 주파수 밸런스

| 주파수 | 레벨 | 역할 |
|--------|------|------|
| Sub (20-60Hz) | -3dB | 베이스 기반 |
| Low (60-250Hz) | 0dB | 따뜻함 |
| Mid (250Hz-2kHz) | +1dB | 보컬, 악기 |
| High (2k-8kHz) | -2dB | 약간 둔하게 |
| Air (8k+) | -4dB | 선명도 낮춤 |

### 마스터링 목표

| 요소 | 값 | 설명 |
|------|-----|------|
| Loudness | -14 LUFS | 스트리밍 표준 |
| True Peak | -1dB | 클리핑 방지 |
| Dynamic Range | 8-10 LU | 적당한 다이나믹 |

## 다이나믹 레인지 가이드

### 구간별 레벨

| 구간 | 상대 레벨 | 다이나믹 |
|------|----------|----------|
| Intro | -6dB | 넓음 |
| Verse | -3dB | 중간 |
| Pre-Chorus | -2dB | 좁아짐 |
| Drop | 0dB (기준) | 좁음 |
| Break | -12dB | 매우 넓음 |
| Outro | -6dB | 넓음 |

### 컴프레션 가이드

| 요소 | Ratio | Attack | Release |
|------|-------|--------|---------|
| 드럼 버스 | 4:1 | 10ms | 100ms |
| 베이스 | 3:1 | 30ms | 200ms |
| 보컬 | 2:1 | 20ms | 150ms |
| 마스터 | 1.5:1 | 30ms | 300ms |

|----------|----------|
