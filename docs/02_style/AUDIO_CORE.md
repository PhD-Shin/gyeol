# AUDIO_CORE — 오디오 핵심

> **이 문서가 필요한 순간:**
> - 음악 제작의 기본 방향을 확인할 때
> - 음악의 역할을 이해해야 할 때
> - 결의 음악 톤을 검증할 때

---

## 핵심 원칙

> **결 = K-판소리 판타지 트랩**
> **판타지 팝의 접근성 + 판소리 트랩의 정체성**
> **몽환으로 끌어들이고, 폭발로 각인시킨다.**
> **판소리로 대중음악을 접수한다.**

---

## 음악의 역할

- 감정을 설명하지 않는다
- 감정을 몰고 가는 주체가 아니다
- **움직임을 흔드는 진동**이다
- **판소리의 한(恨)을 현대적 사운드로 전달**한다

---

## 핵심 공식

```
판타지 팝 = 접근성 (Accessibility) — Verse
판소리/창 = 정체성 (Identity) — Drop
전통 악기 = 텍스처 (Texture) — 전체
트랩/808 = 형식 (Form) — 구조
오케스트라 = 서사 (Narrative) — Pre-Chorus, Bridge
```

### 구간별 스타일 배분

| 구간 | 스타일 | 역할 |
|------|--------|------|
| **Intro** | 판소리 트랩 | 훅, 정체성 선언 |
| **Verse** | 판타지 팝 | 접근성, 몽환 |
| **Pre-Chorus** | 오케스트라 빌드업 | 기대감 |
| **Drop** | 판소리 트랩 | 폭발, 각인 |
| **Bridge** | 서사적 | 해금 + 패드 |
| **Outro** | 몽환 | 여운 |

---

## K-판소리 판타지 트랩 정의

> **판타지 팝의 대중성 + 판소리의 한국성 + 트랩의 현대성**
> 국악 "소개"가 아니라, 국악으로 "접수"한다.
> 몽환으로 문을 열고, 폭발로 각인시킨다.

### 핵심 요소

| 요소 | 적용 구간 | 효과 |
|------|----------|------|
| **판타지 팝** | Verse | 몽환적 신스, 접근성 |
| **오케스트라** | Pre-Chorus | 빌드업, 서사 |
| **창 (Chang)** | Drop | 거칠고 폭발적 발성 |
| **아니리 (Aniri)** | Verse | 말하듯 스토리텔링 |
| **추임새 (Chuimsae)** | 훅 | "얼쑤", "좋다", "그렇지" |
| **장단 (Jangdan)** | Drop | 자진모리/중중모리 |
| **해금** | Bridge/전체 | 서사적 드론, 멜로디 |

### 왜 K-판소리 판타지 트랩인가

| 일반 K-pop | K-판소리 판타지 트랩 |
|-----------|---------------------|
| 아이돌 보컬 | **창 발성 + 몽환 보컬** |
| 전통 = 장식 | **판소리 = DNA** |
| 균일한 에너지 | **몽환 ↔ 폭발 대비** |
| "한국적 요소" | **"한국으로 세계 접수"** |

---

## 피해야 할 것

| 금지 | 이유 |
|------|------|
| 국악 "소개" 느낌 ❌ | 소개가 아니라 접수 |
| 민속 설명 ❌ | 전통 해설 금지 |
| 아이돌 깔끔한 보컬 ❌ | 창의 거침이 핵심 |
| 악기 전면 배치 ❌ | 악기는 텍스처 |
| 밝은 느낌 ❌ | 밝더라도 기이해야 |
| 감정 직접 설명 ❌ | 결은 설명 안 함 |

---

## 성공 공식

> **창/판소리 = 보컬 아이덴티티**
> **전통 악기 = 텍스처**
> **트랩/808 = 구조**
> **AI 시대 = 리듬과 반복**

---

## Kage Bow와의 음악적 차별화

> **J-Pop 구조에 전통 악기 ≠ 국악 구조에 일렉트로닉 질감**

### 구조의 차이

| 요소 | Kage Bow (일본) | 결 (한국) |
|------|----------------|----------|
| 기본 구조 | J-Pop (빌드업→드롭) | **장단 구조** (점층→폭발) |
| 리듬 | 정박 중심 | **엇박** — 3분박, 비대칭 |
| 악기 배합 | 샤미센+신스 | **국악+글리치** |
| 보컬 | 깔끔한 딕션 | **성음** — 거친 목소리 |

### 시김새 (Sigimsae)

> **음 사이의 꺾음, 떨림, 미끄러짐** — 국악의 핵심 표현

| 시김새 | 설명 | 적용 |
|--------|------|------|
| **추성** | 음을 끌어올림 | 멜로디 상승 전 |
| **퇴성** | 음을 끌어내림 | 프레이즈 끝 |
| **전성** | 음을 굴림 | 긴장 구간 |
| **요성** | 음을 떨음 (비브라토) | 감정 고조 |

**적용 방법:**
- 보컬: 음 끝에 시김새 삽입
- 가야금/해금: 시김새가 기본
- 신스: 피치벤드로 시김새 모방

### 성음 (Seongeum)

> **판소리 창법의 거친 목소리** — 깔끔한 J-Pop 보컬과 구분

| 성음 유형 | 특징 | 사용 |
|----------|------|------|
| **통성** | 맑고 울림 | 기본 |
| **괴음** | 거칠고 갈라짐 | 클라이맥스 |
| **수리성** | 슬픔, 애절 | 한 표현 |
| **발발성** | 떨림, 비틀림 | 흥 표현 |

**적용 방법:**
- AI 보컬에 디스토션 레이어
- 더블 트래킹으로 거친 질감
- 숨소리/호흡 강조

### 장단 (Jangdan)

> **국악의 리듬 구조** — 서양 4/4박자와 다른 비대칭

| 장단 | 구조 | 느낌 | 사용 |
|------|------|------|------|
| **굿거리** | 12/8 (3+3+3+3) | 춤, 흥 | 굿판 장면 |
| **중모리** | 12/4 | 서사, 진행 | 기본 진행 |
| **자진모리** | 12/8 빠르게 | 긴장, 고조 | 클라이맥스 |
| **엇모리** | 10/8 (3+2+3+2) | 엇박, 불안 | 도깨비 등장 |

**적용 방법:**
- 4/4 비트 위에 장단 타악 레이어
- 엇박 악센트로 장단 느낌
- 장구 패턴 샘플링

### 차별화 Suno 키워드

> **K-판소리 트랩을 Suno로 구현하는 키워드**

#### 기본 조합 (권장)
```
dark trap korean pansori fusion, atmospheric trap, industrial electronic
```

#### K-판소리 보컬 키워드
```
pansori vocal style, chang singing, traditional korean vocals,
raw emotional voice, pitch bending vibrato, rough korean vocals,
korean traditional singing technique, aniri spoken word
```

#### 전통 요소 키워드
```
korean jangdan rhythm, gayageum stab, janggu percussion,
sigimsae ornaments, chuimsae ad-libs,
asymmetric beat, gugak electronic fusion
```

#### 예시 프롬프트
```
dark trap korean pansori fusion, 110 bpm, pansori vocal style raw emotional,
chang singing technique, 808 sub bass, gayageum texture, janggu rhythm,
whispered aniri to shouted chang, pitch bending vibrato
```

#### 비교: J-Pop vs 결
| 구분 | J-Pop Anime | 결 |
|------|-------------|-----|
| 인식 태그 | anime soundtrack, J-rock | trap, korean fusion, atmospheric |
| 특징 | 오케스트라 + 록 | 전통악기 + 일렉트로닉 |
| Suno 인식률 | 높음 | 조합으로 구현 |

## AI 시대 간접 표현

### 표현 방식

| 요소 | AI/시스템 | 인간 |
|------|----------|------|
| 비트 | 완벽한 반복 | 미세한 어긋남 |
| 퀀타이즈 | 100% 정확 | 90-95% |
| 악기 | 신스, 드럼머신 | 장구, 숨소리 |

### 암묵적 메시지
> AI는 완벽하지만,
> 우리는 결을 만든다.

**직접 말하지 않는다. 리듬으로 보여준다.**

## BPM & 키 가이드

### BPM 범위

| 분위기 | BPM | 사용 |
|--------|-----|------|
| 긴장 | 85-95 | Phase A 초반 |
| 기본 | 95-110 | 대부분 |
| 고조 | 110-125 | 클라이맥스 |

### 키 가이드
- **마이너 키** 기본 (70%)
- 메이저 키 = "밝음" 아닌 **"기이함"**
- 전조: 한 곡에 최대 1회

### 일관성
- 같은 Phase 내 트랙들 = 연관된 키
- BPM 변화 = ±10 범위

---

## Phase별 음악 전략

| Phase | BPM 범위 | 주요 키 | 분위기 |
|-------|---------|--------|--------|
| Phase A | 95-110 | Am, Em | 균형, 도입 |
| Phase B | 100-120 | Dm, Gm | 긴장, 고조 |
| Phase C | 85-100 | Cm, Fm | 무거움, 기록 |
| Phase 0 | 75-90 | Am, Bm | 고요, 기원 |

### Phase별 악기 비중
| Phase | 전통 악기 | 일렉트로닉 | 앰비언트 |
|-------|----------|-----------|---------|
| Phase A | 20% | 50% | 30% |
| Phase B | 25% | 55% | 20% |
| Phase C | 30% | 40% | 30% |
| Phase 0 | 35% | 25% | 40% |

---

## 톤 체크 질문

1. 전통 악기가 "소개"처럼 들리는가? → 수정
2. 감정이 직접 설명되는가? → 수정
3. 밝은 느낌인가? → 기이함으로 전환
4. 완벽한 반복 + 인간적 어긋남이 공존하는가? → 필수

---

## 존재별 사운드 시그니처

각 존재가 등장할 때 음악 특징:

| 존재 | 시그니처 사운드 | BPM 변화 | 특징 |
|------|---------------|---------|------|
| 도깨비 | 리듬 브레이크, 불협화음 | ±5-10 어긋남 | 퀀타이즈 깨짐 |
| 해태 | 저음 드론, 공명 | 유지 | 다른 소리 최소화 |
| 탈 | 속삭임 레이어, 무음 | 유지 | 여러 목소리 겹침 |
| 인물 | 숨소리, 발자국 | 미세 변화 | 인간적 요소 |

### 존재별 금지 사운드
- **도깨비**: 밝은 멜로디 ❌, 안정적 리듬 ❌
- **해태**: 빠른 리듬 ❌, 높은 음역 ❌
- **탈**: 개별 목소리 ❌, 대사 ❌
- **인물**: 내레이션 ❌, 독백 ❌

## 테마별 음악 표현

각 테마의 음악적 표현:

| 테마 | 리듬 | 멜로디 | 텍스처 |
|------|------|--------|--------|
| System vs Deviation | 정확 → 어긋남 | 반복 후 변주 | 기계 + 유기 |
| Erasure | 페이드, 감소 | 사라지는 멜로디 | 리버브 증가 |
| Border | 모호한 전환 | 경계 없는 진행 | 블렌딩 |
| Name | 속삭임, 파편 | 불완전한 훅 | 보컬 레이어 |
| Record | 에코, 반복 | 과거 모티프 | 노이즈, 빈티지 |
| Ritual | 반복 패턴 | 최면적 루프 | 장구, 의례적 |

## 믹싱/마스터링 기준

### 믹싱 목표

| 요소 | 목표 | 설정 |
|------|------|------|
| 스테레오 폭 | 넓음 | 리버브 + 패닝 |
| 깊이감 | 레이어드 | 전경/중경/후경 분리 |
| 선명도 | 약간 흐림 | 전체 롤오프 |
| 공간감 | 넓은 공간 | 긴 리버브 |

### 주파수 밸런스

| 주파수 | 레벨 | 역할 |
|--------|------|------|
| Sub (20-60Hz) | -3dB | 베이스 기반 |
| Low (60-250Hz) | 0dB | 따뜻함 |
| Mid (250Hz-2kHz) | +1dB | 보컬, 악기 |
| High (2k-8kHz) | -2dB | 약간 둔하게 |
| Air (8k+) | -4dB | 선명도 낮춤 |

### 마스터링 목표

| 요소 | 값 | 설명 |
|------|-----|------|
| Loudness | -14 LUFS | 스트리밍 표준 |
| True Peak | -1dB | 클리핑 방지 |
| Dynamic Range | 8-10 LU | 적당한 다이나믹 |

## 다이나믹 레인지 가이드

### 구간별 레벨

| 구간 | 상대 레벨 | 다이나믹 |
|------|----------|----------|
| Intro | -6dB | 넓음 |
| Verse | -3dB | 중간 |
| Pre-Chorus | -2dB | 좁아짐 |
| Drop | 0dB (기준) | 좁음 |
| Break | -12dB | 매우 넓음 |
| Outro | -6dB | 넓음 |

### 컴프레션 가이드

| 요소 | Ratio | Attack | Release |
|------|-------|--------|---------|
| 드럼 버스 | 4:1 | 10ms | 100ms |
| 베이스 | 3:1 | 30ms | 200ms |
| 보컬 | 2:1 | 20ms | 150ms |
| 마스터 | 1.5:1 | 30ms | 300ms |

## 레퍼런스 트랙 (참조)

> 카피가 아닌 방향 참조. 느낌만 가져온다.

### 음악적 레퍼런스

| 아티스트/곡 | 참조 요소 | 결에 적용 |
|------------|----------|----------|
| Billie Eilish - bury a friend | 공간감, 속삭임 | Verse 분위기 |
| The Weeknd - Blinding Lights | 신스 톤, 구조 | Drop 에너지 |
| Radiohead - Everything In Its Right Place | 반복, 모호함 | 전체 톤 |
| FKA twigs - cellophane | 텍스처, 실험 | Break 분위기 |
| Joji - Slow Dancing in the Dark | 우울, 공간 | Outro 여운 |

### 프로덕션 레퍼런스

| 프로듀서 | 참조 요소 |
|----------|----------|
| FINNEAS | 미니멀, 공간감 |
| Flume | 텍스처, 글리치 |
| Oneohtrix Point Never | 실험, 불안 |
| Arca | 왜곡, 기이함 |

### 결 고유 요소
- 레퍼런스는 **출발점**이지 목표가 아님
- 전통 악기 레이어 = 결의 차별점
- 퀀타이즈 어긋남 = 결의 철학
