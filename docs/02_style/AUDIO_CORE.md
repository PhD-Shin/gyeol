# AUDIO_CORE — 오디오 핵심

> **이 문서가 필요한 순간:**
> - 음악 제작의 기본 방향을 확인할 때
> - 음악의 역할을 이해해야 할 때
> - 결의 음악 톤을 검증할 때

---

## 핵심 원칙

> **음악은 감정을 설명하지 않는다.**
> **음악은 움직임을 흔드는 진동이다.**

---

## 음악의 역할

- 감정을 설명하지 않는다
- 감정을 몰고 가는 주체가 아니다
- **움직임을 흔드는 진동**이다

---

## 핵심 공식

```
전통 소리 = 재료 (Material)
K-pop = 형식 (Form)
AI 시대 = 상태 (State, shown not said)
```

---

## 피해야 할 것

| 금지 | 이유 |
|------|------|
| 전통 국악 멜로디 패턴 ❌ | "국악" 느낌 금지 |
| 민속 분위기 ❌ | 전통 소개 금지 |
| 판소리식 감정 과잉 ❌ | 쿨하게 |
| 악기 "소개"처럼 ❌ | 악기는 텍스처 |
| 밝은 느낌 ❌ | 밝더라도 기이해야 |
| 감정 직접 설명 ❌ | 결은 설명 안 함 |

---

## 성공 공식

> **전통 악기 = 텍스처**
> **K-pop = 구조**
> **AI 시대 = 리듬과 반복**

---

## AI 시대 간접 표현

### 표현 방식

| 요소 | AI/시스템 | 인간 |
|------|----------|------|
| 비트 | 완벽한 반복 | 미세한 어긋남 |
| 퀀타이즈 | 100% 정확 | 90-95% |
| 악기 | 신스, 드럼머신 | 장구, 숨소리 |

### 암묵적 메시지
> AI는 완벽하지만,
> 우리는 결을 만든다.

**직접 말하지 않는다. 리듬으로 보여준다.**

---

## BPM & 키 가이드

### BPM 범위

| 분위기 | BPM | 사용 |
|--------|-----|------|
| 긴장 | 85-95 | Phase A 초반 |
| 기본 | 95-110 | 대부분 |
| 고조 | 110-125 | 클라이맥스 |

### 키 가이드
- **마이너 키** 기본 (70%)
- 메이저 키 = "밝음" 아닌 **"기이함"**
- 전조: 한 곡에 최대 1회

### 일관성
- 같은 Phase 내 트랙들 = 연관된 키
- BPM 변화 = ±10 범위

---

## 톤 체크 질문

1. 전통 악기가 "소개"처럼 들리는가? → 수정
2. 감정이 직접 설명되는가? → 수정
3. 밝은 느낌인가? → 기이함으로 전환
4. 완벽한 반복 + 인간적 어긋남이 공존하는가? → 필수

---

## 체크리스트

음악 완성 전:

- [ ] 전통 악기가 "소개"처럼 안 들리는가?
- [ ] 감정을 직접 설명하지 않는가?
- [ ] 완벽한 비트 + 인간적 어긋남 공존하는가?
- [ ] BPM이 지정 범위 내인가?
- [ ] 밝더라도 "기이함"인가?
- [ ] 반복 구조가 있는가?
