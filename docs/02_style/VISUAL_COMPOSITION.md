# VISUAL_COMPOSITION — 구도 규칙

> **이 문서가 필요한 순간:**
> - 화면 구도를 결정할 때
> - 여백의 비율을 정할 때
> - KageBow와 차별화된 시각 언어를 확인할 때

---

## 핵심 원칙

> **Kage Bow가 "꽉 찬 기이함"이라면,**
> **결은 "비어 있어서 기이하다."**

---

## 여백의 미학 (Negative Space)

### 여백 비율 규칙

| 장면 유형 | 여백 비율 | 피사체 비율 |
|----------|----------|-----------|
| 기본 장면 | **70%** | 30% |
| 인물 장면 | 60% | 40% |
| 군중 장면 | 50% | 50% |
| 클라이맥스 | 40% | 60% |
| 사라짐 장면 | **80%** | 20% |

### 여백 ≠ 빈 공간

여백은 비어 있지 않다:
- 한지 텍스처
- 먹 번짐
- 안개/연기
- 미세한 그레인

> **여백은 존재의 부재가 아니라 "잠재된 존재"**

### Kage Bow와의 차이

| 요소 | Kage Bow | 결 |
|------|----------|-----|
| 화면 밀도 | 80% 채움 | **70% 비움** |
| 여백 역할 | 장식, 프레임 | **힘, 긴장** |
| 디테일 | 정교함으로 승부 | **부재로 승부** |

---

## 곡선의 미학 (Korean Curve)

### 한국적 곡선

| 원천 | 곡선의 특징 | 적용 |
|------|-----------|------|
| 한옥 처마 | 끝이 살짝 올라감 | 건물 실루엣 |
| 달항아리 | 비대칭 곡선 | 오브제, 인물 실루엣 |
| 한복 치마 | 부드러운 흐름 | 의상, 움직임 |
| 소나무 | 구불구불 | 배경 라인 |

### 곡선 vs 직선

| 요소 | Kage Bow (일본) | 결 (한국) |
|------|----------------|----------|
| 기본 형태 | 직선, 기하학 | **곡선, 유기적** |
| 대칭 | 완벽한 대칭 | **비대칭** |
| 건축 | 도리이, 직선 | **처마 곡선** |
| 무기 | 직선적 칼 | **곡선 칼날** |

### 곡선 사용 규칙

```
프레임 가장자리: 곡선 (먹 번짐, 안개)
인물 실루엣: 곡선 (흐르는 옷)
건물 라인: 곡선 (처마, 기와)
탈 배치: 비대칭 곡선
```

---

## 비대칭의 미학 (Asymmetry)

### 비대칭 원칙

> **완벽하지 않아서 아름답다 — 달항아리의 철학**

| 요소 | 대칭 (피함) | 비대칭 (지향) |
|------|-----------|-------------|
| 인물 배치 | 중앙 | **황금비 또는 1:2** |
| 탈 군중 | 정렬 | **한 탈만 어긋남** |
| 여백 | 좌우 균등 | **한쪽으로 치우침** |
| 시선 | 정면 | **측면, 3/4** |

### 황금비 활용

```
┌─────────────────────────────────┐
│                     │           │
│      여백 (62%)     │  피사체   │
│                     │  (38%)    │
│                     │           │
└─────────────────────────────────┘
```

### 의도적 불균형

| 상황 | 불균형 방향 | 효과 |
|------|-----------|------|
| 떠남 | 피사체가 향하는 방향에 여백 | 갈 곳이 있음 |
| 고립 | 피사체 주변 전부 여백 | 외로움 |
| 압박 | 여백 없이 프레임 가장자리 | 갇힘 |
| 기대 | 비어있는 방향에서 무언가 올 것 | 긴장 |

## 구도 패턴

### 1. 고독 구도 (인물 단독)

```
┌─────────────────────────────────┐
│                                 │
│                                 │
│                         ●       │
│                       (인물)    │
│          여백 70%               │
└─────────────────────────────────┘
```

- 인물: 프레임 우측 하단
- 시선: 프레임 밖 또는 여백 방향
- 여백: 70% 이상

### 2. 대면 구도 (인물 둘)

```
┌─────────────────────────────────┐
│                                 │
│    ●                       ●    │
│  (인물1)      여백      (인물2) │
│                                 │
│                                 │
└─────────────────────────────────┘
```

- 인물: 프레임 양 끝 1/3 지점
- 사이: 여백 (긴장)
- **절대 가운데서 만나지 않음**

### 3. 군중 구도 (탈)

```
┌─────────────────────────────────┐
│          여백 40%               │
├─────────────────────────────────┤
│  ●  ●  ●  ●  ●  ◐  ●  ●  ●    │
│           (하나만 다름)         │
└─────────────────────────────────┘
```

- 탈: 하단 또는 측면에 배치
- 상단: 여백 (하늘, 압박)
- **반드시 하나가 어긋남**

### 4. 존재 구도 (도깨비/해태)

```
┌─────────────────────────────────┐
│                          ◉      │
│                       (존재)    │
│                                 │
│    ●                           │
│  (인물)        여백 60%         │
└─────────────────────────────────┘
```

- 존재: 프레임 가장자리/모서리
- 인물: 존재를 인식 못함
- 대각선 긴장

## Phase별 구도 전략

| Phase | 여백 | 비대칭 | 특징 |
|-------|------|--------|------|
| Phase A | 70% | 강함 | 고독, 여백의 힘 |
| Phase B | 60% | 중간 | 군중 등장, 밀도 증가 |
| Phase C | 50% | 약함 | 혼란, 경계 모호 |
| Phase 0 | 80% | 극강 | 태초, 거의 빈 화면 |

## 존재별 구도 규칙

### 도깨비

| 규칙 | 값 |
|------|-----|
| 위치 | 프레임 가장자리, 모서리 |
| 크기 | 프레임의 5-15% |
| 방향 | 인물과 대각선 |
| 초점 | 아웃포커스 또는 부분만 |

### 해태

| 규칙 | 값 |
|------|-----|
| 위치 | 상단, 높은 곳 |
| 크기 | 프레임의 15-30% |
| 방향 | 아래를 내려봄 |
| 초점 | 인포커스 (존재감) |

### 탈 군중

| 규칙 | 값 |
|------|-----|
| 위치 | 하단 또는 측면 |
| 배열 | 비대칭, 곡선 |
| 초점 | 전부 인포커스 (획일성) |
| 어긋남 | **반드시 하나** |

### 장승

| 규칙 | 값 |
|------|-----|
| 위치 | 프레임 양쪽 (문 역할) |
| 크기 | 프레임의 40-60% (거대함) |
| 배치 | **반드시 쌍으로** |
| 인물 | 장승 사이를 지나감 |

## Midjourney 구도 키워드

### 여백 강조

```
negative space dominant, 70% empty,
minimal composition, korean aesthetic,
asymmetrical balance, single figure small,
vast emptiness, breathing room,
not crowded, contemplative space
```

### 곡선 강조

```
korean curved lines, hanok eaves curve,
moon jar silhouette, organic shapes,
flowing lines, non-geometric,
traditional korean curves, soft edges
```

### 비대칭

```
asymmetrical composition, rule of thirds,
off-center subject, intentional imbalance,
golden ratio placement, weighted negative space
```

## 금지 구도

| 금지 | 이유 |
|------|------|
| 중앙 배치 ❌ | 너무 안정적, 긴장 없음 |
| 완벽한 대칭 ❌ | 결의 미학 아님 |
| 프레임 꽉 채움 ❌ | 여백의 힘 상실 |
| 모든 탈이 같은 방향 ❌ | 어긋남 필요 |
| 직선적 배열 ❌ | 곡선 지향 |
