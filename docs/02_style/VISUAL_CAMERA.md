# VISUAL_CAMERA — 카메라 규칙

> **이 문서가 필요한 순간:**
> - 카메라 움직임을 설정할 때
> - 샷 구성을 결정할 때
> - 구도 규칙을 확인할 때

---

## 핵심 원칙

> **카메라는 인물을 따라가지 않는다.**
> **인물이 프레임을 벗어나도 카메라는 멈춘다.**

---

## 움직임 원칙

- **최소 움직임**이 기본
- 카메라는 관찰자
- 인물에게 감정이입하지 않음
- 거리 유지

---

## 허용 움직임

| 움직임 | 사용 조건 | 속도 |
|--------|----------|------|
| 고정 (Static) | 기본값, 70% | - |
| 패닝 (Pan) | 공간 확장, 흐름 | 느리게 |
| 줌 (Zoom) | 느린 줌인만 | 매우 느리게 |
| 틸트 (Tilt) | 수직 공간 강조 | 느리게 |

---

## 금지 움직임

| 금지 | 이유 |
|------|------|
| 핸드헬드 흔들림 ❌ | 다큐/리얼리티 느낌 |
| 빠른 휩 팬 ❌ | 과한 역동성 |
| 드론 샷 (과도한) ❌ | 기술 과시 |
| 360도 회전 ❌ | 어지러움, 과잉 |
| 급격한 줌 ❌ | 충격 효과 금지 |
| 트래킹 (인물 추적) ❌ | 감정이입 유발 |

---

## 샷 거리 비율

| 샷 | 비율 | 용도 |
|----|------|------|
| 익스트림 롱 | 15% | 공간, 고립감 |
| 롱 샷 | 35% | 기본, 움직임 전체 |
| 미디엄 | 30% | 상호작용 |
| 클로즈업 | 15% | 디테일, 상징 |
| 익스트림 클로즈업 | 5% | 눈, 손, 물체 |

---

## 구도 가이드

### 기본 원칙
- **여백을 두려워하지 않는다**
- 인물은 프레임의 1/3 이하 가능
- 중앙 배치보다 **오프센터** 선호

### 구도 유형

| 구도 | 사용 상황 |
|------|----------|
| 삼분할 | 기본, 인물 배치 |
| 중앙 대칭 | 의례, 대치 장면 |
| 극단적 오프센터 | 불안, 긴장 |
| 프레임 내 프레임 | 감시, 관찰 |

---

## 여백 규칙

- 위쪽 여백: 넉넉하게 (하늘, 공간감)
- 시선 방향: 여백 확보
- 움직임 방향: 여백 확보
- **빈 공간이 많아도 됨**

---

## 존재별 카메라 반응

| 존재 | 카메라 반응 |
|------|-----------|
| 도깨비 등장 | 살짝 흔들림/어긋남 |
| 해태 등장 | 카메라 멈춤 |
| 탈 등장 | 살짝 줌 |
| 인물 | 따라가지 않음 |

---

## Phase별 카메라 전략

| Phase | 주 샷 | 움직임 | 특징 |
|-------|------|--------|------|
| Phase A | 롱 샷 40% | 고정 80% | 거리감, 관찰자 |
| Phase B | 미디엄 35% | 느린 팬 증가 | 긴장, 접근 |
| Phase C | 클로즈업 25% | 느린 줌 | 디테일, 흔적 |
| Phase 0 | 익스트림 롱 50% | 거의 고정 | 공간, 고립 |

---

## 앵글(각도) 규칙

### 허용 앵글

| 앵글 | 비율 | 의미 | 사용 |
|------|------|------|------|
| 아이레벨 | 60% | 중립, 관찰 | 기본 |
| 하이 앵글 | 25% | 내려다봄, 위계 | 해태 시점, 군중 |
| 로우 앵글 | 10% | 올려다봄, 위압 | 존재 강조 |
| 틸트 앵글 | 5% | 불안정 | 개입 장면만 |

### 앵글 금지

| 금지 | 이유 |
|------|------|
| 버드아이 뷰 (수직 하강) ❌ | 드론 촬영 느낌 |
| 극단적 더치 앵글 ❌ | 과한 불안 |
| POV 샷 ❌ | 감정이입 유발 |

---

## 초점(포커스) 규칙

### 심도 가이드

| 상황 | 심도 | 효과 |
|------|------|------|
| 롱 샷 | 딥 포커스 | 전체 선명 |
| 미디엄 | 중간 심도 | 배경 약간 흐림 |
| 클로즈업 | 얕은 심도 | 피사체만 선명 |
| 존재 등장 | 심도 변화 | 주의 유도 |

### 초점 전환 (포커스 풀)

| 허용 | 금지 |
|------|------|
| 느린 전환 (2-3초) | 빠른 랙 포커스 |
| 한 방향만 (A→B) | 왔다갔다 |
| 트랙당 2-3회 이하 | 과도한 사용 |

---

## 아스펙트 비율

### 권장 비율

| 비율 | 사용 | 느낌 |
|------|------|------|
| 2.39:1 (시네마) | Full MV | 영화적, 기본 |
| 16:9 | Visualizer | 표준 |
| 9:16 | Shorts | 세로, 임팩트 |
| 4:3 | 특별 장면 | 클래식, 회상 |

### Phase별 비율 변화

| Phase | 권장 비율 | 이유 |
|-------|----------|------|
| Phase A | 2.39:1 | 표준 시네마 |
| Phase B | 2.39:1 | 동일 |
| Phase C | 2.39:1 또는 4:3 혼용 | 기록 느낌 |
| Phase 0 | 4:3 | 과거, 기원 |

---

## 존재 조합별 카메라

| 조합 | 샷 | 앵글 | 특징 |
|------|-----|------|------|
| 도깨비 + 인물 | 미디엄 → 롱 | 아이레벨 | 인물 중심, 도깨비는 가장자리 |
| 해태 + 인물 | 롱 샷 | 하이 앵글 | 해태 관점 |
| 탈 + 인물 | 미디엄 | 아이레벨 | 경계 모호, 같은 프레임 |
| 도깨비 + 탈 | 롱 샷 | 아이레벨 | 군중 속 개입 |
| 해태 + 탈 | 익스트림 롱 | 하이 앵글 | 위계 강조 |

---

## Midjourney 프롬프트 카메라 키워드

```
wide shot composition, off-center framing,
negative space, rule of thirds,
static camera feel, distant observer,
long shot, extreme long shot,
minimal camera movement implied
```
