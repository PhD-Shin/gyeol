# 1. 사용자 직접 제작 필수 에셋

## 1.1 로고

### 메인 로고

| 항목 | 스펙 |
|------|------|
| **형식** | PNG (투명 배경) |
| **크기** | 1000×1000px (정방형) |
| **변형** | 가로형 (1000×300px) |
| **색상 버전** | 흰색, 먹색, 단청빨강 |
| **용도** | 워터마크, 인트로, 엔딩 |

### 로고 디자인 가이드

```
스타일:
- 수묵화 느낌의 붓 질감
- 한글 '결' 또는 로마자 'GYEOL'
- 단순하고 강렬한 형태
- 먹 번짐 효과 가능

금지:
- 화려한 그라데이션 ❌
- 3D 효과 ❌
- 귀엽거나 친근한 느낌 ❌
- 복잡한 장식 ❌
```

### 로고 Midjourney 참조 프롬프트

```
korean calligraphy logo, single character,
ink brush stroke, gyeol meaning grain,
minimalist, powerful, ancient modern fusion,
black ink on white, or white on black,
clean edges, professional logo design,
--ar 1:1 --style raw --s 100
```

|------|------|
| **형식** | PNG/JPG |

### 디자인 가이드

배경:

태그라인:
- "Korean Mythic Visual Album"
- "설명하지 않습니다. 방향만 남깁니다."
- 폰트: 명조 계열 또는 붓글씨

악센트:
- 단청 빨강 (#c23a3a) 최소 사용
- 5% 이하
```

## 1.4 폰트 선택

### 권장 폰트

| 용도 | 권장 폰트 | 대안 |
|------|----------|------|
| **제목 (한글)** | 나눔명조, 본명조 | 제주명조, 서울남산체 |
| **제목 (영문)** | Noto Serif, Playfair | Cormorant, Libre Baskerville |
| **본문** | Noto Sans KR | 본고딕, 스포카한산스 |
| **특수 (붓글씨)** | 궁서체, 휘경체 | 해움체, 미생체 |

### 폰트 사용 규칙

```
MV 내 텍스트:
- 최소화 (트랙 제목만)
- 위치: 하단 중앙 또는 하단 좌측
- 크기: 화면의 5% 이하
- 색상: 흰색 또는 먹색

썸네일 텍스트:
- 트랙 제목
- 크고 명확하게
- 명조 계열 권장
```

## 1.6 인트로/아웃트로 템플릿

### 인트로 (5초)

```
구성:
0-2초: 먹색 화면에서 페이드인
2-4초: 로고 + "GYEOL" 표시
4-5초: 페이드아웃

요소:
- 로고 (흰색, 중앙)
- 배경: 먹색 또는 한지 텍스처
- 사운드: 가야금 단음 또는 정적
```

### 아웃트로 (5-10초)

```
구성:
0-3초: 콘텐츠에서 페이드
3-7초: 로고 + 구독 CTA
7-10초: 다음 영상 추천

요소:
- 로고
- "구독" 텍스트 (선택)
- 다음 MV 썸네일 (선택)
```

# 2. AI 생성 에셋

## 2.1 음악 (Suno)

### 트랙당 필요

| 항목 | 수량 | 설명 |
|------|------|------|
| **음악 생성** | 10+ 버전 | 프롬프트로 다수 생성 |
| **베스트 선정** | 1-3개 | 가장 적합한 것 선택 |
| **최종** | 1개 | 편집 후 최종본 |

### 출력 스펙

| 항목 | 스펙 |
|------|------|
| **형식** | MP3/WAV |
| **길이** | 2:30-4:00 |
| **품질** | 320kbps+ |

### 후처리 작업

```
필요 소프트웨어:
- Audacity (무료) 또는
- GarageBand (Mac) 또는
- Adobe Audition

작업 내용:
1. 길이 조절 (인트로/아웃트로 추가)
2. 페이드 인/아웃
3. 노멀라이즈 (-14 LUFS 권장)
4. 필요시 이펙트 추가
```

# 3. 후처리 필요 에셋

## 3.1 MV 편집

### 필요 소프트웨어 (선택)

| 소프트웨어 | 가격 | 난이도 | 권장 |
|-----------|------|--------|------|
| **DaVinci Resolve** | 무료 | 중 | ⭐ 권장 |
| **CapCut** | 무료 | 하 | 초보자용 |
| **Adobe Premiere** | 유료 | 상 | 전문가용 |
| **Final Cut Pro** | 유료 | 중상 | Mac 전용 |

### MV 스펙

| 항목 | 스펙 |
|------|------|
| **형식** | MP4 (H.264) |
| **비율** | 16:9 |
| **해상도** | 1920×1080 (권장 4K) |
| **프레임** | 24fps 또는 30fps |
| **길이** | 2:30-4:30 |
| **비트레이트** | 10-20 Mbps |

### 편집 워크플로우

```
1. 프로젝트 설정
   - 1080p 24fps
   - 색공간: Rec.709

2. 타임라인 구성
   - 오디오 트랙 배치
   - 구간별 마커 설정
   - 음악 싱크 포인트 표시

3. 클립 배치
   - 구간별 클립 배치
   - 비트에 맞춰 컷
   - 트랜지션 추가 (디졸브, 페이드)

4. 컬러 그레이딩
   - 결 팔레트 적용
   - 일관된 톤 유지
   - 콘트라스트 조정

5. 마무리
   - 인트로/아웃트로 추가
   - 워터마크 배치
   - 오디오 레벨 체크
   - 렌더링
```

---

## 3.2 컬러 그레이딩

### 결 컬러 LUT 설정

```
기본 조정:
- 밝기: -10% ~ -20%
- 콘트라스트: +10%
- 채도: -20% ~ -30%
- 색온도: 약간 차갑게

색조 조정:
- 하이라이트: 약간 황토 틴트
- 미드톤: 중립
- 섀도우: 남색/먹색 쪽으로

악센트 처리:
- 빨강: 채도 유지 (단청 빨강)
- 나머지: 채도 저하
```

### 앨범별 컬러 변화

| 앨범 | 밝기 | 채도 | 특수 |
|------|------|------|------|
| A-1 | -15% | -25% | 악센트 5% |
| A-2 | -18% | -20% | 악센트 10% |
| B-1 | -20% | -25% | 안개 효과 |
| B-2 | -22% | -30% | 페이드 효과 |
| C-1 | -15% | -35% | 청록 추가 |
| C-2 | -18% | -30% | 열린 밝기 |
| 0 | -25% | -40% | 극단 어둠 |

---

## 3.3 오디오 믹싱

### 기본 설정

| 항목 | 값 |
|------|-----|
| **전체 볼륨** | -14 LUFS (YouTube 권장) |
| **피크** | -1 dB 이하 |
| **다이나믹 레인지** | 적당히 압축 |

### 이펙트 체인

```
1. EQ: 저역 롤오프 (30Hz 이하)
2. 컴프레서: 라이트 (2:1 ratio)
3. 리미터: -1dB ceiling
4. 스테레오 이미징: 적당히
```

# 4. 파생 콘텐츠

## 4.1 YouTube Shorts

| 항목 | 스펙 |
|------|------|
| **비율** | 9:16 (세로) |
| **해상도** | 1080×1920 |

### 추출 전략

```
MV에서 추출할 포인트:
1. 훅 장면 (가장 임팩트 있는 15초)
2. 존재 등장 장면 (도깨비/해태/탈)
3. 비주얼 하이라이트
4. Drop 구간

편집:
- 세로 크롭 (16:9 → 9:16)
- 자막 추가 (선택)
- 훅 강조 컷

| 타입 | 수량 | 내용 |

| 훅 | 1개 | 가장 강렬한 15초 |
| 존재 | 1-2개 | 도깨비/해태/탈 등장 |
| 분위기 | 1-2개 | 비주얼 하이라이트 |
| **총** | **3-5개** | |

## 4.3 썸네일

### 스펙

| 항목 | 스펙 |
|------|------|
| **크기** | 1280×720px |
| **형식** | PNG/JPG |
| **파일 크기** | 2MB 이하 |

### 디자인 가이드

```
구성:
- 키 비주얼 이미지 (MV에서)

금지:
- 화려한 효과 ❌
- 밝은 색상 과다 ❌
- 얼굴 클로즈업 (결 스타일 아님) ❌
```

```

[트랙 키워드],
dramatic lighting, ink wash style,
title space reserved, high contrast,
--ar 16:9 --style raw --s 100
```

# 6. 에셋 체크리스트

## 제작 시작 전 (1회)

- [ ] 로고 3종 (흰색, 먹색, 빨강)
- [ ] 로고 가로형 (워터마크용)
- [ ] 채널 배너 (2560×1440)
- [ ] 프로필 이미지 (800×800)
- [ ] 폰트 선택 및 설치
- [ ] 인트로 영상 (5초)
- [ ] 아웃트로 영상 (10초)
- [ ] 편집 소프트웨어 설정
- [ ] 컬러 LUT/프리셋 준비

## 트랙당 필요 에셋

### 제작 단계

- [ ] Suno 음악 (10+ 생성 → 1 최종)
