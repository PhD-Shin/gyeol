# QC_CHECKLIST — 품질 검사표

> **이 문서가 필요한 순간:**
> - 트랙 완성 후 품질 검사할 때
> - 공개 전 최종 확인할 때
> - 문제점 발견 시 체크할 때

---

## 사용법

1. 트랙 완성 후 이 체크리스트 순서대로 확인
2. 모든 항목 통과해야 공개 가능
3. 실패 항목은 수정 후 재검사
4. 최종 검사자 서명 후 완료

---

## 실패 시 조치

1. 실패 항목 목록 작성
2. 수정 우선순위 결정
3. 수정 후 해당 섹션만 재검사
4. 전체 통과 시 최종 승인

---

## 검사 우선순위/가중치

### 카테고리별 가중치

| 카테고리 | 가중치 | 최소 통과율 |
|----------|--------|-----------|
| 세계관 일관성 | 30% | 100% (전부 필수) |
| 오디오 | 25% | 85% |
| 비주얼 | 25% | 85% |
| 메타데이터 | 10% | 90% |
| 크로스 프로모션 | 5% | 70% |
| Phase별 특별 | 5% | 100% |

### 항목별 우선순위

| 우선순위 | 설명 | 실패 시 |
|----------|------|--------|
| P1 (필수) | 반드시 통과 | 즉시 수정 |
| P2 (중요) | 강력 권장 | 검토 후 결정 |
| P3 (권장) | 있으면 좋음 | 시간 있으면 수정 |

### P1 항목 (실패 시 공개 금지)
- 금지 규칙 전체 (1.1)
- 존재 규칙 전체 (1.3)
- **첫 3초 훅 (2.1)** — 폭발 필수
- **해금 존재감 (2.2)** — 시그니처 악기
- **가사에 제목/"결" 사용 금지 (2.5)**
- 클리핑/노이즈 (2.6)
- 팔레트 준수 (3.1)
- 해상도/프레임레이트 (4.3)

## 부분 승인 기준

### 점수 기반 승인

| 점수 | 판정 | 조치 |
|------|------|------|
| 95-100% | 통과 | 공개 가능 |
| 85-94% | 조건부 통과 | P2 항목 검토 후 공개 |
| 70-84% | 보류 | P2 수정 후 재검사 |
| 70% 미만 | 실패 | 전면 재작업 |

### P1 실패 시
- 점수 무관 즉시 실패
- 해당 항목 수정 필수
- 수정 후 전체 재검사

## 일반 오류 패턴과 해결책

### 세계관 오류

| 오류 | 빈도 | 해결책 |
|------|------|--------|
| 감정 직접 설명 | 높음 | 가사/보컬 재검토 |
| 희망 메시지 | 중간 | 아웃트로 확인 |
| 민속 설명 | 낮음 | 설명 제거 |

### 오디오 오류

| 오류 | 빈도 | 해결책 |
|------|------|--------|
| 전통 악기 과다 | 높음 | 레벨 다운, 레이어 제거 |
| 구조 불명확 | 중간 | 섹션 마커 재설정 |
| 보컬 과잉 | 중간 | 파트 축소 |

### 비주얼 오류

| 오류 | 빈도 | 해결책 |
|------|------|--------|
| 색상 불일치 | 높음 | LUT 재적용 |
| 광원 다중 | 중간 | 레이어 제거 |
| 얼굴 노출 과다 | 중간 | 컷 교체 |

### 공통 해결 워크플로우

1. 오류 항목 캡처
2. 원인 분석 (근본/표면)
3. 최소 수정 범위 결정
4. 수정 후 해당 섹션만 재검사
