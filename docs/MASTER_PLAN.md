## 세계관 존재 총괄

### 4대 존재

| 존재 | 기능 | 역할 | 결과의 관계 |
|------|------|------|-----------|
| **도깨비** | 교란자/변수 | 시스템을 흔드는 힘 | 결을 **흔든다** |
| **해태** | 경계/질서/관찰자 | 지켜보지만 개입 안 함 | 결을 **지켜본다** |
| **탈** | 페르소나/익명성 | 사회적 표면 | 결을 **가린다** |
| **인물** | 결을 따르는 존재 | 영웅도 비극도 아닌 자 | 결을 **따른다** |

### 존재별 등장 전략

```
Phase A (개입)
├── 도깨비: 암시 → 등장 (그림자 → 연기)
├── 해태: 없음
├── 탈: 배경 → 군중
└── 인물: 실루엣 중심

Phase B (경계)
├── 도깨비: 활발 (연기 → 불꽃)
├── 해태: 암시 → 등장 (조각상)
├── 탈: 군중, 인물과 경계 모호
└── 인물: 부분 노출 증가

Phase C (기록)
├── 도깨비: 감소 (불꽃 → 그림자)
├── 해태: 활발 (미동 → 시선)
├── 탈: 인물과 완전 경계 모호
└── 인물: 탈과 전환

Phase 0 (기원)
├── 도깨비: 첫 등장 (그림자)
├── 해태: 첫 등장 (조각상)
├── 탈: 첫 등장
└── 인물: 형체만
```

## 오디오 스타일 총괄

### 기본 공식

```
결 사운드 = Dark K-pop + 한국 전통악기 + 어둡고 쿨한 분위기
```

### 악기 체계

| 분류 | 악기 | 역할 |
|------|------|------|
| **전통 (20-30%)** | 가야금, 해금, 대금, 장구, 꽹과리 | 한국적 정체성 |
| **현대 베이스** | 808, Sub Bass | K-pop 리듬 기반 |
| **신스** | Distorted Synth, Ambient Pad | 어둡고 현대적 |
| **퍼커션** | Trap Hi-hats, Glitch | 긴장감 |

### Phase별 오디오 특성

| Phase | BPM 범위 | 키 | 특징 |
|-------|---------|-----|------|
| A | 95-115 | Am, Em, Dm | 균형, 미세한 어긋남 |
| B | 95-120 | Dm, Gm, Cm | 긴장, 경계 모호 |
| C | 80-100 | Cm, Fm, Am | 무거움, 느려짐 |
| 0 | 75-90 | Am, Bm | 미니멀, 태초 |

### 보컬 스타일

- **기본**: 가사 아닌 텍스처
- **한국어**: 짧은 구절 반복 (2-4음절)
- **처리**: 속삭임, 레이어링, 보코더
- **금지**: 서사적 가사, 감정 과잉

## 비주얼 스타일 총괄

### 기본 공식

```
결 비주얼 = 수묵화 + 한지 텍스처 + 단일 광원 + 쿨한 분위기
```

### 색상 체계

| 색상 | Hex | 비율 | 용도 |
|------|-----|------|------|
| 먹색 | `#1a1a2e` | 60-80% | 기본 배경 |
| 남색 | `#2c3e50` | 15-25% | 보조 배경 |
| 단청 빨강 | `#c23a3a` | 5-15% | 악센트 (도깨비) |
| 황토 | `#d4a574` | 3-8% | 따뜻한 포인트 |
| 청록 | `#5a7d7c` | 0-10% | Phase C |

### Phase별 색상 변화

| Phase | 먹색 | 악센트 | 특징 |
|-------|------|--------|------|
| A | 80% | 빨강 5-10% | 균형 |
| B | 70% | 빨강 15-20% | 긴장 |
| C | 65% | 청록 10-15% | 기록 |
| 0 | 85% | 최소 | 태초 |

### 텍스처

- **기본**: 한지 (hanji paper)
- **효과**: 먹 번짐 (ink wash)
- **금지**: 매끈한 CG, 3D 렌더링
