# PHASE_MAP — Phase 로드맵

> **이 문서가 필요한 순간:**
> - 트랙이 어느 Phase에 속하는지 결정할 때
> - 메시지 강도를 확인할 때
> - 전체 프로젝트 로드맵을 파악할 때

---

## 핵심 원칙

> **Phase 1에서 직구는 금물이다.**
> **상징으로 시작하고, 해석은 팬에게 위임한다.**

---

## Phase 개요

| Phase | 이름 | 핵심 | 메시지 강도 |
|-------|------|------|-----------|
| Phase A | 개입 (Intervention) | 시스템과 어긋남 | 상징적, 보편적 |
| Phase B | 경계 (Border) | 지워지는 선 | 은유적 |
| Phase C | 기록 (Record) | 남는 것과 지워지는 것 | 점진적 명시 |
| Phase 0 | 기원 (Origin) | 세계의 시작 | 가장 추상적 |

---

## Phase A — 개입 (Intervention)

### 핵심 질문
> **"시스템이 완벽할 때, 어긋남은 어디서 오는가?"**

### 허용 메시지 강도
- 상징적, 보편적
- 특정 역사/정치 언급 ❌
- 글로벌 누구나 공감 가능

### 시각적 특징
- 도깨비 등장 시작
- 완벽한 반복 → 미세한 어긋남
- 어둠 위주, 악센트 최소

### 테마 강조
- System vs Deviation
- Ritual
- Repetition

### 트랙 예시
- 흔들림 (HEUNDEULM)
- 틈 (TIL)
- 균열 (GYUNYEOL)

---

## Phase B — 경계 (Border)

### 핵심 질문
> **"선이 지워질 때, 안과 밖은 어떻게 되는가?"**

### 허용 메시지 강도
- 은유적
- "경계", "지움" 개념 등장
- 여전히 직접 언급 ❌

### 시각적 특징
- 먹 번짐 증가
- 경계 모호한 공간
- 빨강 악센트 증가

### 테마 강조
- Border
- Erasure
- Name

### 트랙 예시
- 경계 (GYEONGGYE)
- 지움 (JIUM)
- 이름 (IREUM)

---

## Phase C — 기록 (Record)

### 핵심 질문
> **"남는 것은 무엇이고, 지워지는 것은 무엇인가?"**

### 허용 메시지 강도
- 점진적 명시
- 커뮤니티 포스트로 "영감" 언급 가능
- 여전히 직구 ❌, 상징 중심

### 시각적 특징
- 지도, 도장, 흔적
- 해태 등장 증가
- 의례/완료 장면

### 테마 강조
- Record
- Name
- Disappearance

### 트랙 예시
- 기록 (GIROK)
- 도장 (DOJANG)
- 흔적 (HEUNJEOK)

---

## Phase 0 — 기원 (Origin)

### 핵심 질문
> **"모든 것이 시작되기 전, 무엇이 있었는가?"**

### 허용 메시지 강도
- 가장 추상적
- 시간 이전
- 선언적, 신화적

### 시각적 특징
- 극단적 어둠
- 단일 광원
- 존재들의 첫 등장

### 테마 강조
- Time
- Movement
- 모든 테마의 기원

### 특이사항
- **나중에 공개** (프리퀄)
- Season 1 이후 공개 권장

---

## Phase 전환 조건

| 전환 | 조건 |
|------|------|
| A → B | 코어 팬덤 형성, 구독자 안정 |
| B → C | 팬덤이 해석 시작, 커뮤니티 활성 |
| C → 0 | IP 확립, 세계관 확장 준비 |

---

## Phase별 금지 사항

### Phase A (엄격)
- 지명/국가명 ❌
- 역사 직구 ❌
- 정치 논쟁 ❌

### Phase B (완화)
- 직접 언급은 여전히 ❌
- "경계", "이름" 개념은 ⭕

### Phase C (점진적)
- 비하인드로 "영감" 언급 가능
- 본 콘텐츠는 여전히 상징 중심

---

## 운영 전략

### Phase A 운영
- 글로벌 타겟
- 댓글 모니터링 필수
- 논쟁 유발 시 즉시 대응

### Phase B-C 운영
- 커뮤니티 포스트로 점진적 공개
- 팬 해석 장려
- 공식 입장은 중립 유지
