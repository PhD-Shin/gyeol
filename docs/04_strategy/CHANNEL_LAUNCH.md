## 채널 포지셔닝

### 한 문장 정의
> **Korean Mythic Visual Album**

### 차별화 전략
일본 상징 MV 채널(Kage Bow 등)의 구조적 규율을 학습하되:
- 일본 도상/신화/미학 복제 ❌
- **한국 문화 재료 사용**
- **민속 설명 제거**
- **전통 → 추상적 기능 상징으로 전환**

### 결과 느낌
- 어둡지만 고딕 아님
- 무겁지만 감정적 아님
- 신화적이지만 역사적 아님
- 재료는 한국, 해석은 글로벌

## 런칭 실패 시나리오

### 실패 유형 및 대응

| 실패 | 증상 | 원인 | 대응 |
|------|------|------|------|
| 초기 노출 없음 | 첫 MV 조회 <500 | 티저 부족, 알고리즘 미타깃 | Shorts/Reels 집중 |
| 오해 발생 | "일본 컨텐츠" 댓글 | 차별화 실패 | 채널 설명 보강 |
| 톤 이탈 | "무섭다/이상하다" | 너무 공포적 | 다음 트랙 톤 조정 |
| 무관심 | 반응 자체 없음 | 훅 부족 | Shorts 훅 강화 |

### 실패 복구 프로토콜

```
1. 매주 데이터 분석 (월간 아님)
2. 실패 유형 즉시 판단
3. 다음 MV에서 보정
4. 3주 연속 저조 시 전략 재검토
```

### 최악의 시나리오

| 상황 | 기준 | 대응 |
|------|------|------|
| 완전 실패 | Album A-1 완료 후 구독자 <500 | Phase A 완료 후 평가, 계속 여부 결정 |
| 논란 | 정치/역사 논쟁 발생 | 해당 영상 비공개 + 무대응 |
| 저작권 문제 | 클레임 발생 | 즉시 비공개 + 수정 후 재업로드 |

## 위기 관리 계획

### 위기 유형

| 위기 | 심각도 | 대응 시한 |
|------|-------|---------|
| 정치/역사 논쟁 | Critical | 24시간 |
| 저작권 클레임 | High | 48시간 |
| 혐오 댓글 폭주 | Medium | 72시간 |
| 오해/루머 | Low | 1주 |

### 대응 원칙

```
1. 결은 설명하지 않는다 (기본)
2. 금지 규칙 위반 시만 조치
3. 논쟁에 참여하지 않음
4. 필요 시 영상 비공개
```

### 위기별 대응

| 위기 | 대응 |
|------|------|
| 정치 논쟁 | 무대응 + 댓글 숨김 |
| 저작권 | 비공개 + 수정 + 이의제기 검토 |
| 혐오 | 댓글 숨김 + 차단 |
| 오해 | 무대응 (해석은 팬에게) |

### 위기 에스컬레이션

| 단계 | 기준 | 액션 |
|------|------|------|
| 1 | 부정 댓글 10% 이상 | 모니터링 강화 |
| 2 | 부정 댓글 30% 이상 | 댓글 숨김 검토 |
| 3 | 외부 언급/기사화 | 영상 비공개 검토 |
| 4 | 법적 이슈 | 전문가 상담 |
