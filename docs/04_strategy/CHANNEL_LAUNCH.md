# CHANNEL_LAUNCH — 채널 런칭

> **이 문서가 필요한 순간:**
> - 유튜브 채널을 세팅할 때
> - 채널 네이밍/브랜딩 결정 시
> - 포지셔닝을 확인할 때

---

## 채널 포지셔닝

### 한 문장 정의
> **Korean Mythic Visual Album**

### 차별화 전략
일본 상징 MV 채널(Kage Bow 등)의 구조적 규율을 학습하되:
- 일본 도상/신화/미학 복제 ❌
- **한국 문화 재료 사용**
- **민속 설명 제거**
- **전통 → 추상적 기능 상징으로 전환**

### 결과 느낌
- 어둡지만 고딕 아님
- 무겁지만 감정적 아님
- 신화적이지만 역사적 아님
- 재료는 한국, 해석은 글로벌

---

## 채널 네이밍 방향

### 네이밍 축 3가지

| 축 | 키워드 | 느낌 |
|----|--------|------|
| Ink/Noir | 먹, 그림자, 어둠 | 아트필름 |
| Lantern/Night/Border | 등불, 밤, 경계 | 글로벌 직관 |
| Mask/Gears/Dokkaebi | 탈, 기계, 도깨비 | IP화 |

### 후보 예시
- GYEOL (결)
- Ink Ritual
- Border Mask
- Night Lantern
- Tal Archive

### 최종 선택: **GYEOL (결)**
- 한국어 고유 개념
- 발음 쉬움
- 브랜드화 가능
- 의미 확장성

## 비주얼 아이덴티티

### 채널 아트
- 단청 패턴 + 먹 질감
- 채널명 최소 크기
- 여백 충분히

### 프로필 이미지
- 도깨비/탈/해태 중 선택
- 실루엣
- 단순한 형태

### 썸네일 스타일
- 일관된 톤 (먹/남/종이)
- 텍스트 최소
- 인물 실루엣 중심

---

## 런칭 콘텐츠 전략

### Phase 1: 티저 (2-4주 전)
- 쇼츠 3-5개
- 훅 컷만
- "Coming soon" 느낌

### Phase 2: 런칭 (D-Day)
- 메인 MV 1개 (2-3분)
- 풀 퀄리티

### Phase 3: 확장 (런칭 후)
- 쇼츠 지속 업로드
- 비주얼라이저/루프
- 다음 MV 티저

---

## 업로드 전략

### 빈도
- 쇼츠: 주 2-3회
- 메인 MV: 월 1회
- 비주얼라이저: 격주

### 시간
- 글로벌 타겟: 금요일 저녁 (미국 시간)
- 한국 타겟: 금요일 오후 6시
