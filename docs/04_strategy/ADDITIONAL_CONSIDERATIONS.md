# ADDITIONAL_CONSIDERATIONS — 추가 고려사항

> **이 문서가 필요한 순간:**
> - 제작 외 전략적 결정이 필요할 때
> - 플랫폼 최적화가 필요할 때
> - 채널 성장 전략을 세울 때

---

## 🎯 1. 유튜브 알고리즘 최적화

### 썸네일 전략

| 요소 | 권장 | 이유 |
|------|------|------|
| **대비** | 높은 대비 | 작은 화면에서 눈에 띔 |
| **텍스트** | 최소화 (1-3단어) | 모바일 가독성 |
| **일관성** | 결 브랜드 색상 | 채널 인식 |
| **얼굴** | 없음 (결 스타일) | 세계관 유지 |

### 제목 패턴

```
[트랙명] — [영문 부제] | GYEOL

예시:
흔들림 — Into the Drift | GYEOL
지워짐 — Traces Remain | GYEOL
```

### 설명 템플릿

```
[한 줄 설명]

🎵 Stream: [링크]
📱 Follow: [링크]

---
결(GYEOL) — [프로젝트 한 줄 설명]

#GYEOL #결 #KoreanMusic #DarkPop #TraditionalFusion
```

### 해시태그 전략

| 필수 | 선택 |
|------|------|
| #GYEOL | #DarkMusic |
| #결 | #KoreanTraditional |
| #KPop | #AtmosphericPop |
| #MusicVideo | #ElectronicMusic |

## 📱 2. 크로스플랫폼 전략

### 플랫폼별 포맷

| 플랫폼 | 포맷 | 비율 | 길이 |
|--------|------|------|------|
| **YouTube Long** | 풀 MV | 16:9 | 2-4분 |
| **YouTube Shorts** | 하이라이트 | 9:16 | 15-60초 |
| **TikTok** | 훅 중심 | 9:16 | 15-60초 |
| **Instagram Reels** | 비주얼 중심 | 9:16 | 15-90초 |
| **Spotify Canvas** | 루프 | 9:16 | 3-8초 루프 |

### 콘텐츠 재활용 전략

```
1개 트랙 → 5개 콘텐츠

[Full MV (Long-form)]
    ↓
[Shorts 1: Hook 하이라이트 (15초)]
    ↓
[Shorts 2: Drop 클라이맥스 (30초)]
    ↓
[Shorts 3: Behind/Process (60초)]
    ↓
[Instagram: Best Visual Loop (15초)]
```

### Shorts 전략

| 요소 | 권장 |
|------|------|
| **시작** | 가장 강렬한 순간으로 시작 |
| **길이** | 15-30초 최적 |
| **루프** | 끝과 시작이 자연스럽게 연결 |
| **자막** | 영어 자막 필수 (음소거 시청 대비) |

## ⚖️ 3. 저작권 및 라이선스

### Suno AI 음악

## 🔧 9. 기술적 고려사항

### 영상 인코딩

| 설정 | 값 |
