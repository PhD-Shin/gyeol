# CONTENT_FUNNEL — 콘텐츠 퍼널

> **이 문서가 필요한 순간:**
> - 콘텐츠 형식을 결정할 때
> - 퍼널 구조를 이해할 때
> - 멀티 플랫폼 전략을 확인할 때

---

## 핵심 전략

```
┌─────────────────────────────────────────────────────┐
│  전곡 MV 전략 — 48트랙 전부 MV                        │
├─────────────────────────────────────────────────────┤
│  MV vs Visualizer 구분 없음                          │
│  모든 트랙 = 풀 퀄리티 MV (3-4분)                     │
│  "그래야 많이 본다"                                   │
└─────────────────────────────────────────────────────┘
```

---

## 플랫폼 구조

```
┌─────────────────────────────────────────────────────┐
│  3 플랫폼 동시 공략                                   │
├─────────────────────────────────────────────────────┤
│  YouTube 롱폼    3-4분 MV     핵심 콘텐츠            │
│  YouTube Shorts  15-60초     유입/훅               │
│  Instagram Reels 15-90초     확산/브랜딩            │
└─────────────────────────────────────────────────────┘
```

---

## 퍼널 개요

```
[Shorts/Reels] ←→ [Main MV]
      ↓               ↓
    훅/유입         핵심/전환
   (발견)          (팬덤화)
      ↓               ↓
  채널 유입  →  구독  →  다음 MV 대기
```

### 단순화된 2단계 퍼널

| 단계 | 형식 | 길이 | 역할 | 플랫폼 |
|------|------|------|------|--------|
| 1 | Shorts/Reels | 15-60초 | 훅, 유입, 확산 | YouTube Shorts, Instagram |
| 2 | Main MV | 3-4분 | 핵심 표현, 팬덤화 | YouTube |

> **Visualizer 폐지**: 모든 트랙이 풀 MV이므로 Long Visualizer 별도 제작 없음.
> 필요 시 MV 루프 버전으로 대체.

---

## 주간 출시 패턴

```
┌─────────────────────────────────────────────────────┐
│  1주 = 1 MV + 파생 숏폼                              │
├─────────────────────────────────────────────────────┤
│  월 │ MV 프리미어 공개 (YouTube)                     │
│  화 │ Shorts 1 — 훅 컷                              │
│  수 │ Shorts 2 — 존재 컷                            │
│  목 │ Reels 1-2 (Instagram)                        │
│  금 │ Shorts 3 — 분위기 컷                          │
│  토 │ 커뮤니티 포스트                                │
│  일 │ 다음 주 티저                                   │
└─────────────────────────────────────────────────────┘
```

### 주간 콘텐츠 수량

| 플랫폼 | 형식 | 주간 수량 |
|--------|------|----------|
| YouTube | MV | 1개 |
| YouTube | Shorts | 3개 |
| YouTube | 커뮤니티 | 1개 |
| Instagram | Reels | 2개 |
| Instagram | 스토리 | 티저 |
| **합계** | | **7+ 콘텐츠/주** |

## YouTube 전략

### MV (롱폼)

| 항목 | 사양 |
|------|------|
| 길이 | 3-4분 |
| 해상도 | 1080p 이상 |
| 포맷 | 프리미어 공개 |
| 빈도 | 주 1회 (월요일) |
| 목표 | 핵심 콘텐츠, 구독 전환 |

### 프리미어 설정

```
공개 24시간 전: 카운트다운 시작
공개 시: 라이브 채팅 활성화
공개 후 1시간: Shorts 1 업로드
공개 후 24시간: Reels 업로드
```

### Shorts

| 항목 | 사양 |
|------|------|
| 길이 | 15-60초 |
| 비율 | 9:16 세로 |
| 빈도 | 주 3회 |
| 목표 | 알고리즘 유입, 채널 발견 |

### Shorts 유형

| 유형 | 내용 | 타이밍 |
|------|------|--------|
| 훅 컷 | MV의 가장 인상적인 장면 | MV 다음날 |
| 존재 컷 | 도깨비/해태/탈 포커스 | MV +2일 |
| 분위기 컷 | 배경/조명/색감 중심 | MV +4일 |
| 티저 | 다음 주 MV 암시 | 일요일 |

## Instagram 전략

### Reels

| 항목 | 사양 |
|------|------|
| 길이 | 15-90초 |
| 비율 | 9:16 세로 |
| 빈도 | 주 2회 |
| 목표 | 확산, 새 오디언스 도달 |

### YouTube Shorts vs Instagram Reels 차이

| 요소 | Shorts | Reels |
|------|--------|-------|
| 편집 | 빠른 컷 | 약간 느림 |
| 음악 | 원곡 그대로 | 트렌딩 믹스 고려 |
| 캡션 | 최소 | 해시태그 풍부 |
| 스타일 | 충격적 훅 | 무드/에스테틱 |
| CTA | "풀 버전 프로필에서" | "링크 인 바이오" |

### Instagram 해시태그 전략

```

#Ambient #DarkAmbient #VisualMusic #MusicVideo
```

## 존재별 숏폼 훅

| 존재 | Shorts 훅 | Reels 훅 |
|------|----------|----------|
| 도깨비 | 프레임 가장자리 갑자기 등장, 마지막 2초에 스침 | 그림자 속 윤곽, 에스테틱 무드 |
| 해태 | 어둠 속 눈만 빛남, 3초간 정지 후 눈 뜸 | 슬로우모션 등장, 위엄 강조 |
| 탈 | 탈 여러 개 카메라 향해 회전, 루프 | 탈 클로즈업, 질감 강조 |
| 인물 | 뒤돌아보는 순간 정지, 얼굴 직전 끊김 | 실루엣 + 빛, 드라마틱 |

## MV에서 숏폼 추출 가이드

### 추출 원칙

```
1 MV → 5+ 숏폼 (Shorts 3 + Reels 2)
```

| 추출 유형 | 설명 | 예시 |
|----------|------|------|
| 훅 컷 | 가장 인상적인 순간 | 도깨비 첫 등장 |
| 존재 컷 | 특정 존재 포커스 | 해태 눈 클로즈업 |
| 분위기 컷 | 배경/조명 중심 | 안개 낀 산사 |
| 트랜지션 | 화면 전환 순간 | 씬 전환 모션 |
| 루프 | 반복 가능한 시퀀스 | 탈 회전 |

### MV 타임스탬프 기록

제작 시 숏폼 추출 포인트 미리 기록:

```
00:12-00:18 훅 컷 (도깨비 등장)
00:45-00:52 존재 컷 (해태 눈)
01:23-01:35 분위기 컷 (안개 산사)
02:10-02:22 트랜지션 (씬 전환)
02:45-02:55 루프 (탈 회전)
```

## Phase별 플랫폼 비중

| Phase | YouTube MV | Shorts | Reels | 특징 |
|-------|-----------|--------|-------|------|
| Phase A | 100% | 100% | 80% | 전 플랫폼 풀 가동 |
| Phase B | 100% | 100% | 100% | 최대 도달 |
| Phase C | 100% | 80% | 80% | MV 집중 |
| Phase 0 | 100% | 60% | 60% | 미스터리 유지 |

### Phase별 숏폼 전략

- **Phase A**: Shorts/Reels 대량 → 최대 유입
- **Phase B**: 크로스 플랫폼 → 팬덤 확장
- **Phase C**: MV 중심 → 서사 집중
- **Phase 0**: 숏폼 제한 → 희소성

## 퍼널 전환률 목표

### 단계별 전환

| 전환 | 목표율 | 측정 |
|------|--------|------|
| Shorts 시청 → 채널 방문 | 3-5% | Shorts 조회 대비 채널 뷰 |
| Reels 시청 → 프로필 방문 | 2-4% | Reels 조회 대비 프로필 뷰 |
| 채널/프로필 → MV 시청 | 20-30% | 방문 대비 MV 조회 |
| MV 시청 → 구독 | 2-5% | MV 조회 대비 구독 |
| 구독 → 다음 MV 시청 | 40-60% | 구독자 대비 신규 MV 조회 |

### 전환률 미달 대응

| 전환 | 미달 기준 | 대응 |
|------|----------|------|
| Shorts → 채널 | <2% | 채널명 노출 증가, 엔딩 CTA |
| Reels → 프로필 | <1.5% | 캡션에 CTA, 링크 명시 |
| 방문 → MV | <15% | 썸네일/제목 개선 |
| MV → 구독 | <1% | 엔드스크린 개선, 구독 요청 |
| 구독 → 다음 MV | <30% | 알림 설정 요청 강화 |

## 크로스 플랫폼 연동

### YouTube → Instagram

```
MV 공개 (월)
    ↓
Reels 업로드 (목): "Full MV on YouTube"
    ↓
스토리: MV 링크 스와이프
    ↓
바이오 링크: YouTube 채널
```

### Instagram → YouTube

```
Reels 바이럴
    ↓
캡션: "Full video → YouTube @GYEOL"
    ↓
바이오 링크 클릭
    ↓
YouTube MV 시청
```

### 크로스 프로모션 규칙

| 규칙 | 내용 |
|------|------|
| 모든 Shorts 엔딩 | "Full MV on channel" |
| 모든 Reels 캡션 | "🎬 Full video: Link in bio" |
| MV 설명 | Instagram 계정 태그 |
| 커뮤니티 포스트 | Reels 하이라이트 언급 |

## 긴급 콘텐츠 대응

### 긴급 상황 정의

| 상황 | 기준 | 대응 |
|------|------|------|
| 바이럴 | 24시간 조회 10x 평균 | 추가 Shorts/Reels 즉시 투입 |
| 논란 | 부정 댓글 30%+ | 영상 비공개 검토 |
| 저조 | 7일 조회 예상의 30% 미달 | 다음 콘텐츠 앞당김 |

### 긴급 콘텐츠 재고

항상 준비:
- 예비 Shorts 2개 (업로드 대기)
- 예비 Reels 1개 (Instagram 저장)
- 예비 커뮤니티 이미지 3개
- 다음 트랙 티저 (완성 후 대기)

### 논란 대응 프로토콜

```
1. 24시간 모니터링 (댓글/언급)
2. 금지 규칙 위반 여부 확인
3. 위반 시: 비공개 + 수정 후 재업로드
4. 미위반 시: 무대응 (결은 설명하지 않음)
```

## 콘텐츠 재활용 규칙

### MV → 파생 콘텐츠

| 원본 | 파생 | 재활용 비율 |
|------|------|-----------|
| MV 1개 | Shorts 3개 | 70% 원본 + 30% 재편집 |
| MV 1개 | Reels 2개 | 60% 원본 + 40% 재편집 |
| MV 1개 | 스토리/포스트 | 스틸컷 5-10장 |

### 재활용 금지

| 금지 | 이유 |
|------|------|
| 타 트랙 영상 혼합 ❌ | 세계관 혼란 |
| 다른 Phase 재료 사용 ❌ | Phase 분리 |
| MV 전체를 숏폼으로만 분할 ❌ | 본편 가치 저하 |

### 효율 목표

| 투자 | 산출 |
|------|------|
| MV 1개 제작 100% | Shorts 3개 (+15%) |
| MV 1개 제작 100% | Reels 2개 (+10%) |
| MV 1개 제작 100% | 스틸컷 5-10장 (+5%) |
| **총 추가 투자** | **+30%로 7+ 콘텐츠** |
