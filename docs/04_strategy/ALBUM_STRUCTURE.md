# ALBUM_STRUCTURE — 앨범 구조

> **이 문서가 필요한 순간:**
> - 앨범을 기획할 때
> - 트랙 순서/배치를 결정할 때
> - 앨범별 분위기를 확인할 때

---

## 전체 구조

```
┌─────────────────────────────────────────────────────┐
│  3 Phase × 2 앨범 × 8 트랙 = 48 트랙                  │
│  출시 주기: 매주 1 MV (전곡 MV)                        │
│  총 기간: 약 48주 (1년)                               │
└─────────────────────────────────────────────────────┘

Phase A (Season 1) — 16주
├── Album A-1: 흔들림 (HEUNDEULM) — 8 tracks, 8주
└── Album A-2: 균열 (GYUNYEOL) — 8 tracks, 8주

Phase B (Season 2) — 16주
├── Album B-1: 경계 (GYEONGGYE) — 8 tracks, 8주
└── Album B-2: 지움 (JIUM) — 8 tracks, 8주

Phase C (Season 3) — 16주
├── Album C-1: 기록 (GIROK) — 8 tracks, 8주
└── Album C-2: 흔적 (HEUNJEOK) — 8 tracks, 8주

[선택] Phase 0 — 반응 좋을 시
└── Album 0: 기원 (GIWON) — 6-8 tracks
```

## 앨범 간 연결

### Phase 내 연결

| 연결 | 방법 |
|------|------|
| A-1 → A-2 | 마지막 트랙 리듬 → 첫 트랙 연결 |
| B-1 → B-2 | 경계 → 지움 테마 연속 |
| C-1 → C-2 | 기록 → 흔적 서사 완결 |

### Phase 간 브릿지

| 브릿지 | 음악 | 시각 |
|--------|------|------|
| A → B | 리듬 끊김 → 안개 인트로 | 도깨비 시선 → 안개 |
| B → C | 저음 드론 → 기록 사운드 | 지워진 경계 → 남은 흔적 |
| C → 0 | 모든 소리 사라짐 | 모든 것 사라짐 → 어둠 |

### 숨겨진 연결 (팬 발견용)

각 앨범에 다른 앨범을 암시하는 **1개** 숨겨진 요소:
- 프레임 한 구석의 기호
- 오디오의 숨겨진 레이어
- 색상 조합

> **원칙**: 설명 없이 놓아두고, 팬이 발견하게 둔다.
