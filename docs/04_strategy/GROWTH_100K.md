## 🔥 Shorts 바이럴 전략

### 훅 유형 (초반 3초)

1. **미스터리 훅**
   - "이 존재의 정체는?"
   - "한국에서 금지된 의식"
   - 도깨비/탈 등장 → 설명 없이 끝

2. **사운드 훅**
   - 강렬한 비트 드롭 (Suno 초반 5초)
   - ASMR 텍스처 (종이, 먹물)
   - 전통 악기 + EDM 충돌

3. **비주얼 훅**
   - 탈 클로즈업 → 줌아웃
   - 먹물 퍼짐 모션
   - 색상 반전 (흑백 → 단색)

4. **트렌딩 훅**
   - 현재 트렌딩 사운드에 결 비주얼 입히기
   - "Korean mythology version of..."
   - 댓글 반응 유도형

### Shorts 카테고리 믹스

```
30% — 트랙 하이라이트 (MV에서 추출)
25% — 비하인드/메이킹 (세계관 힌트)
25% — 트렌딩 리믹스 (훅 사운드)
20% — 팬 참여 유도 (투표, 질문)
```

## 📈 알고리즘 최적화

### 제목 공식

```
[숫자/형용사] + [한국 신화 키워드] + [감정/행동 유도]

예시:
- "도깨비가 나타났을 때 절대 하면 안되는 것"
- "한국 신화 속 가장 무서운 존재 TOP 3"
- "이 탈을 쓰면 일어나는 일"
- "99%가 모르는 한국 전통 의식"
```

### 썸네일 공식

```
1. 강렬한 대비 (검정 + 단색 포인트)
2. 얼굴/눈 클로즈업 (탈, 도깨비)
3. 텍스트 3단어 이하
4. 미스터리 요소 (가려진 부분, 흐림)
```

### 해시태그 세트

```yaml
고정 태그:
  - #결 #GYEOL #KoreanMythology #한국신화
  - #Dokkaebi #도깨비 #Haetae #해태

트렌딩 믹스:
  - #DarkAesthetic #MythologyShorts #Folklore
  - #KoreanArt #TraditionalMeetsModern

사운드 태그:
  - #EDM #DarkElectronic #Cinematic
```
