## 컨셉 & 스토리

### 한 문장
> **완벽한 시스템 속에서, 무언가 흔들리기 시작한다.**

### 서사 흐름 (암시적)
1. 세계가 있다 — 규칙적으로 반복된다
2. 반복 속에서 미세한 어긋남이 발생한다
3. 어긋남의 원인은 보이지 않는다
4. 그림자가 스친다
5. 세계가 흔들린다

> **직접 설명하지 않는다. 시청자가 느낀다.**

```
□ 5트랙 개별 QC 통과
□ 앨범 연속 재생 테스트 통과
□ 세계관 금지 사항 전부 통과
□ 외부 피드백 반영 (선택)
→ 4개 중 3개 이상 ✓ = 완료
```

## 실패 시나리오 & 대안

### 오디오 실패 시

| 문제 | 징후 | 대안 |
|------|------|------|
| 스타일 불일치 | 밝음, 감정적 | 템플릿 변경, 10회 추가 생성 |
| 구조 불량 | K-pop 구조 아님 | 수동 편집으로 구간 재배치 |
| 전통악기 안 들림 | 가야금 없음 | 프롬프트에 "gayageum prominent" 강조 |
| 보컬 과잉 | 가사 너무 많음 | "minimal vocal, whispered only" 추가 |

### 비주얼 실패 시

| 문제 | 징후 | 대안 |
|------|------|------|
| 스타일 불일치 | 밝음, CG 느낌 | --sref로 스타일 고정 |
| 일관성 부족 | 트랙 간 톤 다름 | 시드 고정, 배치 생성 |
| 텍스처 없음 | 한지 느낌 없음 | 후보정으로 텍스처 오버레이 |
| 존재 규칙 위반 | 도깨비 과노출 | 해당 이미지 폐기 후 재생성 |

### 일정 지연 시

| 지연 정도 | 대응 |
|----------|------|
| 1주일 | Shorts 5개 → 3개로 축소 |
| 2주일 | Visualizer 1개 간소화 |
| 3주일 | Main MV 1개 나중으로 연기 |
| 4주일+ | 앨범 분할 출시 고려 |

## 피드백 수집 방법

### 채널

| 채널 | 수집 내용 | 빈도 |
|------|----------|------|
| YouTube 댓글 | 직접 반응 | 매일 확인 |
| YouTube Analytics | 시청 지속률, 이탈 지점 | 주 1회 분석 |
| 커뮤니티 포스트 | 투표, 질문 응답 | 2주 1회 |

### 분석 항목

| 항목 | 질문 |
|------|------|
| 이탈 지점 | 어디서 시청 중단하는가? |
| 재시청 구간 | 어디를 다시 보는가? |
| 검색 키워드 | 어떤 키워드로 유입되는가? |
| 다음 영상 | 어떤 영상으로 이동하는가? |

### 피드백 반영 기준

| 피드백 유형 | 반영 |
|------------|------|
| "잘 모르겠다" | 설명 추가 ❌, 다음 영상에서 힌트 |
| "너무 어둡다" | 무시 (의도적) |
| "음악 좋다" | 해당 스타일 유지 |
| "더 보고 싶다" | 출시 속도 유지/가속 |
| 기술적 문제 | 즉시 수정 |
