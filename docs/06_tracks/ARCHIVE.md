## 아카이브 방법

### 트랙 완료 시

1. **STATUS.md 업데이트**
   - 현재 트랙 상태를 🟢 완료로 변경
   - 다음 트랙을 "현재 진행 중"으로 이동

2. **트랙 문서 업데이트**
   - 진행률을 100%로
   - 모든 체크리스트 완료 표시
   - 최종 파일 경로 기록

3. **이 문서 업데이트**
   - 완료일 기록
   - MV 링크 추가 (업로드 후)
   - 조회수 주기적 업데이트

4. **파일 정리**
   - 최종 에셋만 남기고 중간 파일 정리
   - 백업 확인
