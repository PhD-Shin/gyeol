# ALL ALBUMS OVERVIEW — 전체 앨범 개요 & 트랙 프롬프트

> **이 문서가 필요한 순간:**
> - 전체 앨범 구조를 파악할 때
> - 특정 트랙 제작을 시작할 때
> - 프롬프트 템플릿이 필요할 때

---

## 전체 구조 한눈에

```
Phase A — 개입 (Intervention)
├── A-1: 흔들림 (시스템에서 첫 어긋남)
└── A-2: 균열 (개입 본격화)

Phase B — 경계 (Border)
├── B-1: 경계 (안과 밖의 모호)
└── B-2: 지움 (정체성 소멸)

Phase C — 기록 (Record)
├── C-1: 기록 (남는 것들)
└── C-2: 흔적 (사라진 후 여운)

[선택] Phase 0 — 기원 (Origin)
└── 0: 기원 (태초의 순간)
```
