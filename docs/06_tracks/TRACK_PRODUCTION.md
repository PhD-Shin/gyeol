## 6️⃣ QC 체크리스트

### 음악 QC
- [ ] 결 스타일과 맞는가?
- [ ] 전통 악기 들리는가?
- [ ] 감정 과잉 아닌가?
- [ ] 구조가 맞는가?

### 이미지 QC
- [ ] 색상 팔레트 준수?
- [ ] 텍스처 (먹/한지)?
- [ ] 조명 단일 광원?
- [ ] 일본/중국 요소 없음?

### 영상 QC
- [ ] 스타일 일관성?
- [ ] 음악 싱크?
- [ ] 컷 전환 자연스러움?

### 금지 규칙 확인
- [ ] 얼굴 명확히 보이지 않음?
- [ ] 설명적 텍스트 없음?
- [ ] 해피엔딩/구원 메시지 없음?
