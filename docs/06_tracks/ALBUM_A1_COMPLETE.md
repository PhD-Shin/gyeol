```
dark k-pop, trap elements, atmospheric,

808 bass, janggu over electronic drums,
minimal vocal, whispered korean fragments,
ink wash sonic aesthetic, mysterious, cold
```

```

## Track 1: 시작 (SIJAK)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **제목** | 시작 / SIJAK |
| **역할** | 세계관 진입, 앨범 오프닝 |
| **테마** | Time, System |
| **존재** | 없음 (세계만 보여줌) |
| **길이** | 2:30-3:00 |
| **BPM** | 95 |
| **키** | Am |

### 컨셉

> 시간이 흐르기 시작하는 순간. 어둠 속에서 세계가 드러난다.
> 설명 없이, 그냥 시작한다.

### 구조

```
[Intro] 0:00-0:30
  - 어둠에서 시작
  - 가야금 단일 음
  - 드론 배경

[Verse 1] 0:30-1:00
  - 비트 서서히 등장
  - 808 펄스 시작
  - 공간감 확장

[Pre-Chorus] 1:00-1:20
  - 장구 리듬 추가
  - 텐션 빌드

[Drop] 1:20-1:50
  - 풀 비트
  - 가야금 + 808
  - 세계 드러남

[Break] 1:50-2:00
  - 짧은 정적

[Outro] 2:00-2:30
  - 페이드
  - 여운
```

## Track 2: 반복 (BANBOK)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **제목** | 반복 / BANBOK |

| **존재** | 탈 (배경에 2-3개) |

```

  - 반복 강화

  - 모든 요소 정확히 맞물림

## Track 3: 틈 (TIL)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **제목** | 틈 / TIL |
| **역할** | 첫 어긋남 |
| **테마** | Deviation (첫 등장) |
| **존재** | 없음 |
| **길이** | 3:00 |
| **BPM** | 100 → 98 (미세 변화) |
| **키** | Am → Em 전환 |

### 컨셉

> 완벽한 반복에서 처음으로 무언가 어긋난다.
> 작은 틈. 미세한 균열. 아직 눈에 띄지 않지만, 무언가 시작되었다.

### 구조

```

  - 그러나 미세하게 다름

[Verse 1] 0:25-0:55
  - 첫 어긋남 발생
  - 비트 하나 밀림
  - 퀀타이즈 98%

[Pre-Chorus] 0:55-1:15
  - 어긋남 인식
  - 긴장 시작

[Drop] 1:15-1:50
  - 틈 확대
  - 불협화음 삽입

[Break] 1:50-2:10
  - 정적
  - 어긋남 인식

[Drop 2] 2:10-2:40
  - 틈이 자리잡음

[Outro] 2:40-3:00

```

## 제작 체크리스트

### 앨범 시작 전

- [ ] 컬러 팔레트 확인 (먹색 80%, 악센트 5%)
- [ ] BPM 범위 확인 (95-110)
- [ ] 존재 등장 순서 확인
- [ ] Shot Library 기본 이미지 준비

### 각 트랙 제작 시

- [ ] Suno 프롬프트 적용
- [ ] 10개 이상 생성 후 베스트 선정
- [ ] MJ 키 비주얼 15장
- [ ] 비디오 클립 20-30개
- [ ] QC 체크리스트 통과

### 앨범 완료 후

- [ ] 8트랙 일관성 확인
- [ ] 색상 톤 통일 확인
- [ ] 존재 등장 규칙 준수 확인
- [ ] 다음 앨범 연결 준비
