## 신규 확립된 시스템

### 1. 현대 추임새 시스템

| 타입 | 추임새 | 사용 |
|------|--------|------|
| 전통 | 얼쑤, 좋다, 그렇지 | Drop, Pre-Chorus |
| 현대 | 가자, 간다, 봐라 | Drop 끝, 빌드업 |
| 하이브리드 | Let's go 가자, Yeah 좋다 | 코드스위칭 포인트 |

### 2. 코드스위칭 전략

```
영어로 끌어들이고 → 한국어로 펀치
```

| 구간 | 영어 | 한국어 |
|------|------|--------|
| Verse | 70% | 30% |
| Drop | 40% | 60% |
| 추임새 | 하이브리드 | 하이브리드 |

### 3. A/B/C 테스트 시스템

| 버전 | 특징 | 용도 |
|------|------|------|
| Style A | 판소리 강조 | 정체성 테스트 |
| Style B | 판타지 팝 강조 | 접근성 테스트 |
| Style C | 하이브리드 | 균형 테스트 |
