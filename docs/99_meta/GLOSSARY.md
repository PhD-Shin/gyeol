# GLOSSARY — 용어 사전

> **이 문서가 필요한 순간:**
> - 용어의 정확한 의미를 확인할 때
> - 새 팀원이 프로젝트를 이해할 때
> - 문서 작성 시 용어 통일할 때

---

## ㄱ

### 가야금 (Gayageum)
한국 전통 현악기. 결에서는 Intro/Outro의 핵심 악기로 사용.
→ [[02_style/AUDIO_INSTRUMENTS]]

### 결 (Gyeol)
프로젝트명이자 핵심 개념. "나무결", "물결"처럼 시간이 지나며 쌓인 움직임의 흔적.
→ [[00_core/DEFINITION]]

### 경계 (Border)
메타 테마 중 하나. 안과 밖, 존재와 부재의 경계선.
→ [[01_world/THEMES]]

### 금지 규칙 (Prohibitions)
결 세계관에서 절대 하지 않는 것들. 역사 직구, 민속 설명 등.
→ [[01_world/PROHIBITIONS]]

### 기록 (Record)
메타 테마 중 하나. 남는 것과 지워지는 것.
→ [[01_world/THEMES]]

## ㄴ

### 인디고 (Indigo)
#2c3e50. 결의 보조색. 깊은 남색.
→ [[02_style/VISUAL_COLOR]]

---

## ㄷ

### 단청 (Dancheong)
한국 전통 건축의 장식 그림. 결에서는 배경 패턴으로 사용. 설명 없이.
→ [[02_style/VISUAL_TEXTURE]]

### 도깨비 (Dokkaebi)
결의 핵심 존재. 교란자/트릭스터 기능. 캐릭터가 아닌 힘.
→ [[03_characters/DOKKAEBI]]

### Drop
곡 구조의 클라이맥스 구간. 808 + 장구 레이어.
→ [[02_style/AUDIO_STRUCTURE]]

---

## ㅁ

### 먹 (Ink)
수묵화의 검은 잉크. 결의 핵심 텍스처.
→ [[02_style/VISUAL_TEXTURE]]

### 메타 테마 (Meta Themes)
결 세계관을 관통하는 6개 주제. Border, Name, Record, Erasure, Ritual, System vs Deviation.
→ [[01_world/THEMES]]

---

## ㅂ

### 반복 (Repetition)
작동 규칙 중 하나. 시스템은 완벽하게 반복, 인간은 미세하게 어긋남.
→ [[01_world/RULES]]

### 빨강 악센트 (Red Accent)
#c62828. 결의 유일한 따뜻한 색. 사용 비율 20-30%.
→ [[02_style/VISUAL_COLOR]]

---

## ㅅ

### 사라짐 (Disappearance)
작동 규칙 중 하나. 모든 것은 흔적만 남기고 사라진다.
→ [[01_world/RULES]]

### 속삭임 (Whisper)
보컬 스타일 중 하나. 30% 비율. 공간감과 친밀감.
→ [[02_style/AUDIO_VOCAL]]

### 시스템 vs 어긋남 (System vs Deviation)
메타 테마 중 하나. 완벽한 시스템과 그 안의 미세한 오류.
→ [[01_world/THEMES]]

### Shot Library
Midjourney로 생성한 이미지 라이브러리. 권장 40장.
→ [[02_style/VISUAL_PIPELINE]]

---

## ㅇ

### 의례 (Ritual)
메타 테마 중 하나. 반복되는 행위의 의미.
→ [[01_world/THEMES]]

### 이름 (Name)
메타 테마 중 하나. 정체성, 지워진 이름.
→ [[01_world/THEMES]]

### 인간 (Human)
결의 존재 중 하나. 얼굴 없는 개체. 자유의지 보유.
→ [[03_characters/HUMAN]]

## ㅈ

### 장구 (Janggu)
한국 전통 타악기. Drop에서 808과 레이어링.
→ [[02_style/AUDIO_INSTRUMENTS]]

### 지움 (Erasure)
메타 테마 중 하나. 존재의 삭제, 기록의 말소.
→ [[01_world/THEMES]]

---

## ㅊ

### 찢어지는 보컬 (Distorted Vocal)
보컬 스타일 중 하나. 15% 비율. Drop 클라이맥스에서만.
→ [[02_style/AUDIO_VOCAL]]

---

## ㅌ

### 탈 (Tal)
한국 전통 가면. 결에서는 페르소나/사회적 얼굴 기능.
→ [[03_characters/TAL]]

### 톤 (Tone)
결의 전체적인 분위기. 쿨하고 건조함. 감정 과잉 금지.
→ [[01_world/TONE]]

---

## ㅍ

### 퍼널 (Funnel)
콘텐츠 퍼널. Shorts → Main MV → Long Visualizer.
→ [[04_strategy/CONTENT_FUNNEL]]

### Phase
프로젝트 진행 단계. A(개입) → B(경계) → C(기록) → 0(기원).
→ [[04_strategy/PHASE_MAP]]

---

## ㅎ

### 한지 (Hanji)
한국 전통 종이. 결의 배경 텍스처.
→ [[02_style/VISUAL_TEXTURE]]

### 해태 (Haetae)
결의 핵심 존재. 관찰자/심판자 기능. 개입하지 않고 지켜봄.
→ [[03_characters/HAETAE]]

### 훅 (Hook)
반복되는 짧은 음악/가사 구절. 명확한 후렴 대신 사용.
→ [[02_style/AUDIO_VOCAL]]

## 영문

### 808
트랩 음악의 핵심 베이스/드럼 사운드. 결의 Drop 핵심.
→ [[02_style/AUDIO_INSTRUMENTS]]

### Break
곡 구조의 쉬는 구간. 5-10초. 거의 무음.
→ [[02_style/AUDIO_STRUCTURE]]

### K-pop Structure
결의 곡 구조 기반. 장르 혼합 + 강한 시각성 + 퍼포먼스 중심.
→ [[02_style/AUDIO_STRUCTURE]]

### Korean Mythic Visual Album
결의 한 문장 정의. 채널 포지셔닝.
→ [[04_strategy/CHANNEL_LAUNCH]]

### MCP (Model Context Protocol)
Claude가 외부 도구와 연결되는 프로토콜.

### Midjourney
AI 이미지 생성 도구. 결 비주얼의 80%+ 담당.
→ [[05_execution/MIDJOURNEY_PROMPTS]]

### QC (Quality Control)
품질 검사. 공개 전 필수.
→ [[05_execution/QC_CHECKLIST]]

### RAG (Retrieval Augmented Generation)
검색 기반 생성. 이 문서 구조의 설계 목표.

### Shot Library
→ Shot Library (한글 섹션)

### Suno
AI 음악 생성 도구. 결 오디오 제작에 사용.
→ [[05_execution/SUNO_PROMPTS]]
