# GLOSSARY — 용어 사전

> **이 문서가 필요한 순간:**
> - 용어의 정확한 의미를 확인할 때
> - 새 팀원이 프로젝트를 이해할 때
> - 문서 작성 시 용어 통일할 때

---

## ㄱ

### K-판소리 판타지 트랩 (K-Pansori Fantasy Trap)
결의 핵심 장르 정체성. 판소리(창/아니리/추임새) + 판타지 팝 + 트랩 베이스 융합.
→ [[02_style/AUDIO_CORE]]

### 가야금 (Gayageum)
한국 전통 현악기. 가야국 기원, 12현→25현 발전.
결에서는 Intro/Outro의 핵심 악기로 사용. "결의 선(Line)".
→ [[02_style/AUDIO_INSTRUMENTS]]

### 거문고 (Geomungo)
한국 전통 현악기. "백악지장(百樂之長)", 고구려 기원.
결에서는 권위/경계 표현에 제한적 사용.
→ [[02_style/AUDIO_INSTRUMENTS]]

### 결 (Gyeol)
프로젝트명이자 핵심 개념. "나무결", "물결"처럼 시간이 지나며 쌓인 움직임의 흔적.
→ [[00_core/DEFINITION]]

### 경계 (Border)
메타 테마 중 하나. 안과 밖, 존재와 부재의 경계선.
→ [[01_world/THEMES]]

### 금지 규칙 (Prohibitions)
결 세계관에서 절대 하지 않는 것들. 역사 직구, 민속 설명 등.
→ [[01_world/PROHIBITIONS]]

### 기록 (Record)
메타 테마 중 하나. 남는 것과 지워지는 것.
→ [[01_world/THEMES]]

## ㄴ

### 남색 (Indigo)
#2c3e50. 결의 보조색. 깊은 남색. 오방색 청(靑)의 변용.
→ [[02_style/VISUAL_COLOR]]

---

## ㄷ

### 단청 (Dancheong)
한국 전통 건축의 장식 그림. 결에서는 배경 패턴으로 사용. 설명 없이.
→ [[02_style/VISUAL_TEXTURE]]

### 도깨비 (Dokkaebi)
결의 핵심 존재. 교란자/트릭스터 기능. 캐릭터가 아닌 힘.
삼국유사 비형설화에서 최초 기록. "독각귀(獨脚鬼)" 기원.
→ [[03_characters/DOKKAEBI]]

### 독각귀 (獨脚鬼)
외다리 귀신. 도깨비의 문헌적 기원. 삼국유사 비형설화 등장.
→ [[03_characters/DOKKAEBI]]

### Drop
곡 구조의 클라이맥스 구간. 808 + 장구 레이어.
→ [[02_style/AUDIO_STRUCTURE]]

---

## ㅁ

### 먹 (Ink)
수묵화의 검은 잉크. 결의 핵심 텍스처.
→ [[02_style/VISUAL_TEXTURE]]

### 민화 (Minhwa)
한국 전통 서민 그림. "예쁜 그림"이 아닌 "서민의 풍자".
호작도, 십장생도 등. 결에서 시각적 문법 레퍼런스.
→ [[02_style/VISUAL_TEXTURE]]

### 메타 테마 (Meta Themes)
결 세계관을 관통하는 6개 주제. Border, Name, Record, Erasure, Ritual, System vs Deviation.
→ [[01_world/THEMES]]

---

## ㅂ

### 반복 (Repetition)
작동 규칙 중 하나. 시스템은 완벽하게 반복, 인간은 미세하게 어긋남.
→ [[01_world/RULES]]

### 비형설화 (鼻荊說話)
삼국유사에 기록된 도깨비 최초 문헌. 독각귀 등장.
→ [[03_characters/DOKKAEBI]]

### 봉산탈춤
황해도 지역 탈춤. 양반 계급 풍자. 결의 탈 레퍼런스.
→ [[03_characters/TAL]]

### 빨강 악센트 (Red Accent)
#c62828. 결의 유일한 따뜻한 색. 사용 비율 20-30%.
→ [[02_style/VISUAL_COLOR]]

---

## ㅅ

### 사라짐 (Disappearance)
작동 규칙 중 하나. 모든 것은 흔적만 남기고 사라진다.
→ [[01_world/RULES]]

### 사헌부 (司憲府)
조선시대 관리 감찰 기관. 흉배에 해태 문양 사용. 해태의 법치 상징 근거.
→ [[03_characters/HAETAE]]

### 삼국유사 (三國遺事)
도깨비 최초 문헌 기록 (비형설화). 독각귀 등장.
→ [[03_characters/DOKKAEBI]]

### 설문해자 (說文解字)
중국 한대 자전. "해치(獬豸) — 소와 비슷하고 뿔이 하나" 기록.
→ [[03_characters/HAETAE]]

### 속삭임 (Whisper)
보컬 스타일 중 하나. 30% 비율. 공간감과 친밀감.
→ [[02_style/AUDIO_VOCAL]]

### 시스템 vs 어긋남 (System vs Deviation)
메타 테마 중 하나. 완벽한 시스템과 그 안의 미세한 오류.
→ [[01_world/THEMES]]

### Shot Library
Midjourney로 생성한 이미지 라이브러리. 권장 40장.
→ [[02_style/VISUAL_PIPELINE]]

---

## ㅇ

### 아쟁 (Ajaeng)
한국 전통 현악기. 국악기 중 최저 음역. 결에서 저음 드론 담당.
→ [[02_style/AUDIO_INSTRUMENTS]]

### 오방색 (五方色)
동양의 5방위 색상 체계. 청/적/황/백/흑.
결에서는 흑-청-적-백 4색만 변용 사용.
→ [[02_style/VISUAL_COLOR]]

### 의례 (Ritual)
메타 테마 중 하나. 반복되는 행위의 의미.
→ [[01_world/THEMES]]

### 이름 (Name)
메타 테마 중 하나. 정체성, 지워진 이름.
→ [[01_world/THEMES]]

### 인간 (Human)
결의 존재 중 하나. 얼굴 없는 개체. 자유의지 보유.
→ [[03_characters/HUMAN]]

한국 전통 타악기. Drop에서 808과 레이어링.
→ [[02_style/AUDIO_INSTRUMENTS]]

### 지움 (Erasure)
메타 테마 중 하나. 존재의 삭제, 기록의 말소.
→ [[01_world/THEMES]]

## ㅊ

### 창 (Chang)
판소리의 노래 부분. K-판소리 트랩에서 Drop의 폭발적 발성.
→ [[02_style/AUDIO_VOCAL]]

### 찢어지는 보컬 (Distorted Vocal)
보컬 스타일 중 하나. 15% 비율. Drop 클라이맥스에서만.
→ [[02_style/AUDIO_VOCAL]]

### 추임새 (Chuimsae)
판소리의 추임새. K-판소리 트랩의 핵심 훅.
전통(얼쑤/좋다/그렇지) + 현대(가자/간다) + 하이브리드(Yeah 좋다).
→ [[02_style/AUDIO_VOCAL]]

## ㅌ

### 탈 (Tal)
한국 전통 가면. 결에서는 페르소나/사회적 얼굴 기능.
→ [[03_characters/TAL]]

### 톤 (Tone)
결의 전체적인 분위기. 쿨하고 건조함. 감정 과잉 금지.

## ㅎ

### 해금 (Haegeum)
한국 전통 찰현악기. K-판소리 판타지 트랩의 시그니처 악기. 유령의 멜로디.
→ [[02_style/AUDIO_INSTRUMENTS]]

### 한영 혼용 / 코드스위칭 (Code-Switching)
K-pop 스타일의 언어 혼용. 핵심 원칙: 같은 의미 반복 ❌ → 연결/확장 ⭕
예: "Let it burn 재만 남아" (burn→ashes = 원인→결과)
→ [[02_style/AUDIO_VOCAL]]

### 한지 (Hanji)
한국 전통 종이. 결의 배경 텍스처.
→ [[02_style/VISUAL_TEXTURE]]

### 하회탈 (Hahoe Tal)
경상도 하회마을 탈. 턱이 분리된 유일한 탈. 9종 원형 (양반/선비/부네/각시/중/초랭이/이매/백정/할미).
→ [[03_characters/TAL]]

### 해태 (Haetae)
결의 핵심 존재. 관찰자/심판자 기능. 개입하지 않고 지켜봄.
설문해자 "해치(獬豸)" 기원. 사헌부 상징. 경복궁 화기 제어.
→ [[03_characters/HAETAE]]

### 훅 (Hook)
반복되는 짧은 음악/가사 구절. 명확한 후렴 대신 사용.
→ [[02_style/AUDIO_VOCAL]]
