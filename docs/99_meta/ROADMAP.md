## 🎯 즉시 실행 (이번 주)

### Phase 1: A1-01 파일럿 제작

```
Day 1-2: 음악 제작
├── Suno로 A1-01 "시작" 생성 (3-5회 시도)
├── 최적 버전 선택
└── Gemini 분석 실행

Day 3-4: 이미지 제작
├── MJ 프롬프트 30장 실행
├── 핵심 훅 이미지 우선 (★★★)
└── 이미지 선별 및 보정

Day 5-6: 영상 제작
├── 타임라인 편집
├── 트랜지션 적용
└── 최종 렌더링

Day 7: QC + 업로드
├── /qc 스킬로 검증
├── 썸네일 제작
└── YouTube 업로드
```

### 실행 명령어

```bash
# 1. 분석 (음악 다운로드 후)
python3 scripts/gyeol-pipeline.py analyze A1-01 --music ~/Downloads/시작.mp3

# 2. 프롬프트 생성
python3 scripts/gyeol-pipeline.py prompts A1-01

# 3. QC
/qc A1-01_시작
```

## ⚡ 생산성 최적화 팁

### 병렬 처리
```
시간대별 작업 분배:
- 오전: Suno 생성 (CPU 대기)
- 오후: MJ 이미지 생성 (GPU 대기)
- 저녁: 영상 편집 (집중 필요)
- 밤: 렌더링 + 업로드 (자동)
```

### 배치 작업
```bash
# 여러 트랙 한번에 상태 확인
python3 scripts/gyeol-pipeline.py batch A1 --range 1-8

# 프롬프트 미리 생성
for i in 01 02 03 04; do
  python3 scripts/gyeol-pipeline.py prompts A1-$i
done
```

### 템플릿 재사용
- Suno: 성공한 프롬프트 저장 → 변형
- MJ: 스타일 시드 고정 → 일관성
- 편집: 프리셋 저장 → 적용
