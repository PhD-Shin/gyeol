# 2. 100K 성장 전략 (GROWTH_100K)

## 30일 로드맵

```
Week 1: 0 → 1K
├── 채널 세팅 + 첫 MV
├── Shorts 10개
└── 커뮤니티 포스트 3개

Week 2: 1K → 10K
├── 두 번째 MV
├── Shorts 14개
└── 협업 컨택 시작

Week 3: 10K → 50K
├── 세 번째 MV
├── Shorts 14개 (콜라보)
└── 팬 참여 유도

Week 4: 50K → 100K
├── 네 번째 MV
├── Shorts 14개
├── 바이럴 챌린지
└── 앨범 티저
```

---

## 콘텐츠 수량

| 유형 | 30일 | 빈도 |
|------|-----|------|
| MV | 4개 | 주 1회 |
| Shorts | 60개 | 일 2회 |
| 커뮤니티 | 15개 | 격일 |

---

## Shorts 바이럴 훅

**1. 미스터리 훅**
- "이 존재의 정체는?"
- 도깨비/탈 등장 → 설명 없이 끝

**2. 사운드 훅**
- 강렬한 비트 드롭 (초반 5초)
- 전통 악기 + EDM 충돌

**3. 비주얼 훅**
- 탈 클로즈업 → 줌아웃
- 먹물 퍼짐 모션

**4. 트렌딩 훅**
- 트렌딩 사운드 + 결 비주얼

---

## 제목 공식

```
[숫자/형용사] + [한국 신화 키워드] + [감정 유도]

예시:
- "도깨비가 나타났을 때 절대 하면 안되는 것"
- "이 탈을 쓰면 일어나는 일"
```

---

## 썸네일 공식

1. 강렬한 대비 (검정 + 단색 포인트)
2. 얼굴/눈 클로즈업 (탈, 도깨비)
3. 텍스트 3단어 이하
4. 미스터리 요소

---

## 커뮤니티 정책 (Phase A)

**허용:**
- 이모지 반응 (🖤, 👁️, 🌙)

# 3. 오디오 핵심 (AUDIO_CORE)

## 핵심 원칙

> **음악은 감정을 설명하지 않는다.**
> **음악은 움직임을 흔드는 진동이다.**

---

## 핵심 공식

```
전통 소리 = 재료
K-pop = 형식
AI 시대 = 상태 (shown not said)
```

---

## 성공 공식

> **전통 악기 = 텍스처**
> **K-pop = 구조**
> **AI 시대 = 리듬과 반복**

---

## 차별화 포인트 (vs Kage Bow)

| 요소 | Kage Bow | 결 |
|------|---------|-----|
| 구조 | J-Pop 빌드업 | 장단 (점층→폭발) |
| 리듬 | 정박 | 엇박 (3분박, 비대칭) |
| 악기 | 샤미센+신스 | 국악+글리치 |
| 보컬 | 깔끔 | 성음 (거친 목소리) |

---

## 시김새 (Sigimsae)

음 사이의 꺾음, 떨림, 미끄러짐

| 기법 | 설명 |
|-----|------|
| 추성 | 음을 끌어올림 |
| 퇴성 | 음을 끌어내림 |
| 전성 | 음을 굴림 |
| 요성 | 음을 떨음 |

→ 보컬, 가야금, 신스 피치벤드에 적용

---

## BPM & 키

| 분위기 | BPM |
|--------|-----|
| 긴장 | 85-95 |
| 기본 | 95-110 |
| 고조 | 110-125 |

**키**: 마이너 기본 (Am, Em, Dm)

---

## 존재별 사운드

| 존재 | 시그니처 |
|-----|---------|
| 도깨비 | 리듬 브레이크, 불협화음, 퀀타이즈 깨짐 |
| 해태 | 저음 드론, 다른 소리 최소화 |
| 탈 | 속삭임 레이어, 무음, 여러 목소리 |
| 인물 | 숨소리, 발자국 |

---

## AI 시대 표현

| 요소 | AI/시스템 | 인간 |
|-----|---------|-----|
| 비트 | 완벽한 반복 | 미세한 어긋남 |
| 퀀타이즈 | 100% | 90-95% |

→ **직접 말하지 않는다. 리듬으로 보여준다.**

---

## 톤 체크

```
□ 전통 악기가 "소개"처럼 안 들리는가?
□ 감정을 직접 설명하지 않는가?
□ 완벽 비트 + 인간적 어긋남 공존?
□ 밝더라도 "기이함"인가?
```

# 4. 제작 워크플로우 (음악 먼저)

## 핵심 원칙

> **가사 ≠ 비주얼**
> **음악 무드 = 비주얼**

가사와 직접 연관 없이, 음악의 분위기에 맞춰 비주얼을 만든다.

---

## 워크플로우

```
1. Suno 음악 생성 → 베스트 1개 확정
2. 음악 분석 (무드/에너지/전환점)
3. Claude에게 분석 결과 전달
4. 음악 기반 MJ 프롬프트 생성
5. MJ 이미지 생성
```

---

## 음악 분석 양식

```
전체 무드: _______ (긴장/고요/폭발/신비/공허)

에너지 곡선:
- Intro: 낮음 / 중간 / 높음
- Verse: 낮음 / 중간 / 높음
- Drop: 낮음 / 중간 / 높음
- Outro: 낮음 / 중간 / 높음

전환점:
- 0:__ -
- 1:__ -

시각화 키워드 (3-5개):
1.
2.
3.
```

---

## 음악 무드 → 비주얼

| 음악 무드 | 비주얼 키워드 |
|----------|--------------|
| 긴장 | `tense, unsettling, shadows` |
| 고요 | `still, meditative, fog` |
| 폭발 | `dramatic reveal, epic scale` |
| 신비 | `mysterious, barely visible` |
| 공허 | `lingering, fading, empty` |

# 요약: 4가지 기억할 것

```
1. 결은 설명하지 않는다
2. 전통은 텍스처, K-pop은 구조
3. Shorts 일 2개 + 미스터리 유지
4. 음악 먼저 → 음악 분위기 = 비주얼
```
