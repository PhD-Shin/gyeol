# 1) 참고한 일본 MV 분석 요약

## 1.1 작품의 핵심 미학

- 표면: **레트로 퓨처 + 전통(우키요에 질감) + 스팀/사이버 요소 혼합**
    
- 본질: “일본풍” 그 자체가 핵심이 아니라,
    
    - **전통 미학을 껍데기로 사용**하고
        
    - **현대 사회의 시스템(알고리즘·자본·노동·통제)을 ‘신화/공포’로 시각화**한 게 핵심
        

## 1.2 비주얼 룩 (Art Style)

- 텍스처: 종이 질감, 굵은 먹선, 빈티지 색감(바랜 느낌)
    
- 컬러 구조:
    
    - 기본 톤: **남색/먹색(밤·우울·전통)**
        
    - 강조 톤: **붉은색(등불·불꽃·위협·에너지)**
        

## 1.3 반복 상징(오브제)와 의미

- 여우 가면: 익명성/집단 최면/군중(개성 상실, 시스템 순응)
    
- 거대한 신체·눈: 감시/통제/전지적 시스템(알고리즘/지배 구조)
    
- 행렬과 군중: 맹목적 추종/저승길 암시/의식적 이동
    
- 톱니바퀴·기계: 사회 시스템의 부품화, 구조의 압력
    

## 1.4 장면 구조(3막)

- 초반(도입·세계관): 손동작·주술, 거대한 발, 비 내리는 거리 등으로 불길한 세계 설정
    
- 중반(혼란·축제): 등불 가득한 도시, 가면 군중의 춤/행진, 가사(현대·알고리즘)에 맞춘 기계 이미지 오버랩
    
- 후반(클라이맥스): 눈 클로즈업, 불타는 도시/붕괴, 붉은색 비중 증가, 다시 손동작으로 여운

# 2) “한국형 판타지”로 변환할 때의 핵심 원칙

## 2.1 한 문장 정의(이번 프로젝트의 본질)

> **전통 미학을 껍데기로 쓰고, 현대 시스템(알고리즘·자본·노동·통제)을 ‘신화적 공포’로 시각화한다.**

즉,

- 일본 요소는 교체 가능
    
- **‘전통 + 현대 억압 구조의 신화화’가 핵심**
    

## 2.2 절대 원칙 3가지

### 원칙 1) 1:1 요소 치환이 아니라 “의미 치환”

- 여우 가면을 하회탈로 바꾸는 수준에서 끝내면 실패 확률↑
    
- 반드시 “의미 단위”로 한국적 상징을 재설계해야 함
    

예시(의미→표현):

- 익명성/집단최면 → 하회탈/각시탈/흰 상복 군중
    
- 감시/통제 → 단청 문양의 눈/산신의 시선
    
- 시스템 압력 → 내려앉는 산/무너지는 기와/하늘의 그림자
    
- 행렬 → 장례 행렬의 만장/굿판 행렬
    

### 원칙 2) “신”은 캐릭터가 아니라 ‘시스템’

- 신 = 선/악이 아니라 **질서·구조·규칙**
    
- 얼굴/대사로 설명하지 말고, **눈·산·하늘·패턴·그림자** 같은 ‘형태 없는 압력’으로 표현
    

### 원칙 3) 한국 판타지는 “고요한 공포”가 강점

- 네온/빠른 액션/과잉 설명보다
    
- **느림·반복·정면 응시·침묵**이 무기로 작동

# 3) GPT 프로젝트 지침용 정리(시스템 프롬프트로 바로 쓰는 틀)

## 3.1 목적(Mission)

- 한국 전통 미학 기반
    
- 현대 시스템을 **신화적 공포**로 시각화한 MV 제작
    

## 3.2 비주얼 스타일 가이드

- 배경: 민화/수묵화/한지 질감/밤의 한옥/안개/달빛/산 실루엣
    
- 색: 청·남·먹(기본) + 붉은색(강조)
    
- 캐릭터: 탈(하회탈 등), 갓, 한복(변형 가능), 무표정/정면 응시
    
- 기계·시스템: 톱니바퀴를 그대로 쓰기보다 **기와·단청·패턴의 맞물림**으로 시스템성 표현
    

## 3.3 3막 스토리보드 프레임

- 초반: 부적/손동작, 먼 북소리, 하늘·산의 미세한 움직임(세계관)
    
- 중반: 탈 군중 행렬, 굿판 같은 반복 동작, 공간이 살아 움직임(광기)
    
- 후반: 산이 내려앉음/하늘의 시선/단청 눈, 모두 멈추고 카메라 응시(초월)
    

## 3.4 금지/검증 규칙

- 금지: 일본 고유명사/직접 일본 신화 언급/단순 동양 판타지 일반화
    
- 검증 질문:
    
    1. 일본 요소가 없어도 성립?
        
    2. 예쁘다보다 불안하다가 먼저?
        
    3. ‘악마’가 아니라 ‘질서(시스템)’가 느껴지나?
        
    4. 설명 없이도 의미가 전달되나?

## 4.3 당신 프로젝트와의 결정적 차이

- Demon Hunters: 스타일리시/히어로/명확한 악
    
- 당신 프로젝트: 고요한 공포/집단·행렬/신=시스템/상징적 의미 중심

# 5) “자기 스타일 vs 팬 무비” 전략 의사결정 정리

## 5.1 선택지 A: 처음부터 자기 스타일

- 장점: 장기 IP/작가성, 법적 리스크 낮음
    
- 단점: 초반 유입·알고리즘 진입이 매우 어려움(묻히기 쉬움)
    

## 5.2 선택지 B: 팬 무비(편승)

- 장점: 검색/추천/해시태그 유입이 쉬움(단기 트래픽 강함)
    
- 단점: “원작의 그림자”에 갇힐 위험, 2편 나오면 묻힐 수 있음
    

## 5.3 채팅에서 도출된 최적 해법: 2-Phase 전략

> **팬 무비로 ‘문(트래픽)’을 열고, 빠르게 ‘내 스타일’로 이탈 설계**

### Phase 1 (Hook)

- 겉모습: Demon Hunters 연상되는 “무대/퍼포먼스/실루엣” 문법 차용
    
- 안전 문구: fan-made tribute / inspired by / not official
    
- 목표: 알고리즘 진입, 해외 시청자 확보
    

### Phase 2 (Break)

- 중간부터 “이상해지는” 전환
    
- 히어로 서사/악마 보스/승리 엔딩은 피하고
    
- 무대가 의식 공간으로 변함, 함성이 북소리/굿 소리로 변형
    
- 캐릭터의 얼굴/정체성이 “탈/집단/행렬”로 흡수됨
    
- 결과: 팬 무비처럼 시작했지만 “작가성 있는 작품”으로 기억
