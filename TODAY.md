### Step 2: 음악 분석 (음악 완성 후)

> **원칙**: 가사와 직접 연관 없이, **음악 분위기**에 맞춰 비주얼을 만든다.

음악 들으면서 아래 채우기:

```
전체 무드: _____________ (긴장/고요/폭발/신비/공허)

에너지 곡선:
- Intro: 낮음 / 중간 / 높음
- Verse: 낮음 / 중간 / 높음
- Drop: 낮음 / 중간 / 높음
- Outro: 낮음 / 중간 / 높음

전환점 (분위기 확 바뀌는 지점):
- 0:__ -
- 0:__ -
- 1:__ -

음악에서 느껴지는 이미지 키워드 (3-5개):
1.
2.
3.
```
